# 方法一：位级运算&

## 解题思路和过程

在c语言中***除了常见逻辑运算符&&，||等外还有与其相似的位级运算符其二者的作用基本相同，但我们在进行逻辑是多用到的是逻辑运算符，因为逻辑运算符，在进行逻辑判断时如果前面就能判断出结果，后面就不会再判断了，而位级运算符，在二进制方面比较实用***，他在二进制判断时会对其中对应的0和1进行判断，这里我们用到这个性质，因为***只要是2的幂，那么他的二进制形式一定是1000000...多个零，而其-1后就是01111111...多个1，如果进行位级&判断，其结果就是0***

## 复杂度

- 时间复杂度:$O(1)$
- 空间复杂度:$O(1)$

## Code

```
class Solution {
public:
    bool isPowerOfTwo(int n) {
        return (n>=1)&&((n&(n-1))==0);
    }
};
```

# 方法二：暴力

## 解题思路和过程

一直对n/2,在除之前如果他是奇数，就直接返回false，直到最后除到结果为1，返回true

别忘了，如果n==0，或者n<0，直接返回false

## 复杂度

- 时间复杂度:$O(\log_2n)$

- 空间复杂度:$O(1)$

## Code

```
class Solution {
public:
    bool isPowerOfTwo(int n) {
        while(n>1){
            if(n%2==1)return false;
            n/=2;
        }
        if(n==1)return true;
        return false;
    }
};
```
