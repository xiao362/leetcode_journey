# 方法一

## 解题思路和过程

因为倘若使用交换完成任务，我们发现在矩阵的排数n和列数m不同时无法完成任务，所以我们依旧直接开一个ans数组，列数为n，排数为m，然后我们直接遍历原数组，然后将行数与列数交换，存到ans中即可

## 复杂度

- 时间复杂度:$O(n*m)$
- 空间复杂度:$O(n*m)$

## Code

```c++
class Solution {
public:
    vector<vector<int>> transpose(vector<vector<int>>& matrix) {
        int n=matrix.size();
        int m=matrix[0].size();
        vector<vector<int>> ans(m,vector<int>(n,0));
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                ans[j][i]=matrix[i][j];
            }
        }
        return ans;
    }
};
```
