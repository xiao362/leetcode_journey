# 方法一：双指针

## 解题思路

因为在交换时我们是行与行之间交换的，所以我们在交换的过程中我们只需要控制着行的坐标即可，在交换的行坐标确定时我们只需要再开一层循环（控制纵坐标）对一整行进行交换即可

同样因为左右指针有大小联系，我们也可以用单指针来解决

## 复杂度

- 时间复杂度:$O(k^2)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    vector<vector<int>> reverseSubmatrix(vector<vector<int>>& grid, int x, int y, int k) {
        int l=x;
        int r=x+k-1;
        int p=0;
        while(l<r){
            for(int i=y;i<y+k;++i){
                p=grid[l][i];
                grid[l][i]=grid[r][i];
                grid[r][i]=p; 
            }
            l++;
            r--;
        }
        return grid;
    }
};
```

另一种写法

```C++
class Solution {
public:
    vector<vector<int>> reverseSubmatrix(vector<vector<int>>& grid, int x, int y, int k) {
        int p=0;
        int left=0;
        for(int i=x;i<x+k/2;++i){
            left++;
            for(int j=y;j<y+k;++j){
                p=grid[i][j];
                grid[i][j]=grid[x+k-left][j];
                grid[x+k-left][j]=p;
            }
        }
        return grid;
    }
};
```
