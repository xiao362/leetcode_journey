# 方法一：

## 解题思路和过程

同样是模拟这个过程，开三重的循环，如果满足条件就让ans++

## 复杂度 

- 时间复杂度:O($$n^3$$)
- 空间复杂度:O($$1$$)

## Code

```
class Solution {
public:
    int countGoodTriplets(vector<int>& arr, int a, int b, int c) {
        int n=arr.size();
        int ans=0;
        for(int i=0;i<n-2;++i){
            for(int j=i+1;j<n-1;++j){
                for(int k=j+1;k<n;++k){
                    if(abs(arr[i]-arr[j])<=a&&abs(arr[j]-arr[k])<=b
                    &&abs(arr[i]-arr[k])<=c)ans++;
                }
            }
        }
        return ans;
    }
};
```
