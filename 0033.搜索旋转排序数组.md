# 方法一：二分

二分的具体写法可以有很多，要根据不同的情况变换结束条件以及区间的缩小情况

## 解题思路和过程

观察数组数据我们发现所谓的旋转排序数组不过是将原来升序的数组，截取一部分放在剩下一部分的后面，但起码这**两部分好是保持升序的**，那么**只要是有序可寻且缩短范围的体检是可以实现的我们就可以用二分查找**，且下面**题目要求我们设计一个时间复杂度为`logn`的算法**（基本常见的就是二分）

观察给出的实例数据，模拟查找的过程，我们大致会遇到4种情况：

1. **target位于左边数组**（元素较大）下面会产生两种情况（前提是：**`target>nums[n-1]`**）：
   - `nums[mid]<target&&nums[mid]>nums[n-1]`这都说明**mid的位置位于左边数组**,缩小左边区间即可
   - 除了上面情况以外，缩小右面区间（说明**mid在右面区间或者比target大**）
2. **target位于右边数组**（元素较小）也会有两种情况（**`target<=nums[n-1]`**）：
   - `nums[mid]<target||nums[mid]>nums[n-1]`说明**mid位于右边区间或者比target小**
   - **位于右面区间且比target大**

上面四种情况**实际上可以看成两种情况**：

1. **mid在target的左边**
2. **mid在target的右边**

但因为要实现代码再会划分成4种情况

> 注意别忘了找不到target要返回-1，可以用
>
> ***`nums[left]==target?left:-1`;***
>
> 三目运算符来返回

## 复杂度

- 时间复杂度:$O(\log n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int search(vector<int>& nums, int target) {
        int n=nums.size();
        int left=0;
        int right=n-1;
        while(left<=right){
            int mid=(left+right)>>1;
            if(target<=nums[n-1]){
                if(nums[mid]<target||nums[mid]>nums[n-1]){
                    left=mid+1;
                }else{
                    right=mid-1;
                }
            }else{
                if(nums[mid]<target&&nums[mid]>nums[n-1]){
                    left=mid+1;
                }else{
                    right=mid-1;
                }
            }
        }
        return nums[left]==target?left:-1;
    }
};
```
