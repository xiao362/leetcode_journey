# 方法一：埃氏筛+bfs

## 解题思路和过程

通过题目给的条件我们可以知道发生跳跃的条件是：相邻或者质数传送，相邻倒是好解决直接对坐标加1或减1即可，但质数传送则是，当现在的数是质数时才能触发，而且还要满足跳跃的位置是质数的倍数，所以在遍历过程中判断是否是质数肯定会用到，但是传统的判读质数的方法时间复杂度是O(n) ,如果叠加到传送过程的模拟中，毫无疑问时间会超，又因为这题的数据并不是很大，所以我们使用埃氏筛先将所有质数筛选出来存起来，后来判断就简单了

这里做一个巧妙的变换，即我们将所有数的质因数存起来，通过质因数的个数我们可以判断其是否是质数，还可以直接得出每个数的质因数(因为后面传送时还要用到每个数的质因数准确的说是质数的倍数)

这里***可以用lambda对全局变量初始化***：**千万不要忘记在后面加括号，不然这个lambda就没执行**，也可以显示调用这个lambda，不过那就与写函数没两样了

然后就是遍历原数组将对应的质因数变成坐标，这样才方便在跳跃，这里可以使用hashmap与vector结合来存储，每个数可以跳跃的位置**unordered_map<int,vector<int>> mp;**，这里可能稍微有点绕，我来举个例子，如6的质因数有2，3，那么mp.first存的是2，3后面vector存的是6这个数在nums中对应的坐标，也就是在**nums遍历到指数时可以直接通过mp找到它可以跳跃的坐标**

接下来就是bfs，这里可以使用两个队列，因为要保证一个队列中存的都是同一步，方便步数ans的变化

## 复杂度

- 时间复杂度：O(n+MXlog(MX))
- 空间复杂度：O(MAX·log log MAX) 为主导

## Code

```
const int MAX=1e6+1;
vector<int> factors[MAX];
bool prime=[](){
    for(int i=2;i<MAX;++i){
        if(factors[i].empty()){
            for(int j=i;j<MAX;j+=i){
                factors[j].push_back(i);
            }
        }
    }
    return 1;
}();

class Solution {
public:
    int minJumps(vector<int>& nums) {
        int n=nums.size();
        unordered_map<int,vector<int>> mp;
        for(int i=0;i<n;++i){
            for(int p:factors[nums[i]]){
                mp[p].push_back(i);
            }
        }
        int ans=0;
        vector<bool> vis(n,0);
        queue<int> q;
        q.push(0);
        vis[0]=1;
        while(1){
            queue<int> q2;
            while(!q.empty()){
                int i=q.front();
                q.pop();
                if(i==n-1)return ans;
                if(i-1>=0&&!vis[i-1]){
                    q2.push(i-1);
                    vis[i-1]=1;
                }
                if(i+1<=n-1&&!vis[i+1]){
                    q2.push(i+1);
                    vis[i+1]=1;
                }
                if(factors[nums[i]].size()==1){
                    int p=nums[i];
                    if(mp.count(p)){
                        for(int j:mp[p]){
                            if(!vis[j]){
                                vis[j]=1;
                                q2.push(j);
                            }
                        }
                       mp.erase(p);
                    } 
                    
                }
                
            }
            if(q2.empty())break;
            while(!q2.empty()){
                int a=q2.front();
                q.push(a);
                q2.pop();
            }
            
            ans++;
        }
        return ans;
       
    }
};
```
