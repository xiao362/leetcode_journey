# 方法一：双指针

## 解题思路

题目给的数组是非递减顺序排列的，因为数组是有符号数，所以我们在平方后数组并不一定还是相对有序的但是，如果把平方后的数组以原来数组的正负分为左右两部分，我们会发现，左边部分是非递增的，右边部分是非递减的，都是相对有序的，这时我们在左右两边设置两个指针，那他们往中间靠的时候就都是相对递减的了，这样我们就可以开一个ans数组，通过比较这两个指针位置值的大小来至右向左依次填入较大值，就能实现将平方数组以非递减的顺序排列了

## 实现细节

1. 要提前获得数组长度，并开好数组，不然无法实现至右向左依填入

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$，不算返回数组

## Code

```C++
class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        for(int &i:nums){
            i=i*i;
        }
        int n=nums.size();
        vector<int> ans(n,0);
        int left=0;
        int right=n-1;
        while(left<=right){
            if(nums[left]>nums[right]){
                ans[--n]=nums[left];
                left++;
            }else{
                ans[--n]=nums[right];
                right--;
            }
        }
        return ans;
    }
};
```

# 方法二：库函数sort

## 复杂度

- 时间复杂度:$O(n\log n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        for(int &i:nums)i=i*i;
        sort(nums.begin(),nums.end());
        return nums;
    }
};
```
