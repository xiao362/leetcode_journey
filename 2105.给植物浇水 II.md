# 方法一：双指针

## 解题思路

按照题意模拟这个过程，这个过程中我们我们用双指针分别从两端出发，用a，b分别表示二者剩余的水量，如果够浇水就直接减去，不够就补满，补水次数加一，然后在减去，但是在后面我们会遇到一种特殊情况就是在二者位置相同的时候，取二者较大的一个来浇水如果还不够就补满

## 实现细节

1. 因为这个指针是从两端同时滚动的，二者可以相互计算，故我们可以用单指针来写（开for循环），但是要注意他们要能取到中点，为此我们要用向上取整的方法来保证中点能取到`i < (n + 1) / 2`
2. 在取到中点并计算过后要直接break掉循环，不然会重复计算

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int minimumRefill(vector<int>& plants, int capacityA, int capacityB) {
        int n = plants.size();
        int ans = 0;
        int a = capacityA;
        int b = capacityB;
        for (int i = 0; i < (n + 1) / 2; ++i) {
            if (i == n - 1 - i) {
                if (max(a, b) < plants[i])
                    ans++;
                break;
            }
            if (a >= plants[i])
                a -= plants[i];
            else {
                a = capacityA - plants[i];
                ans++;
            }
            if (b >= plants[n - i - 1])
                b -= plants[n - i - 1];
            else {
                b = capacityB - plants[n - i - 1];
                ans++;
            }
        }
        return ans;
    }
};
```
