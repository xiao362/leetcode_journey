# 方法一：双指针+贪心

## 解题思路

按照题意前后缀只要字母相同就能删除，而不在意个数的多少，而题目要求我们求最短长度，所以我们要尽可能多的删除元素，为此我们把前缀中相同元素的字母当成一个元素来处理，后缀同样（尽可能多吃掉相同元素），若如果前后缀相同的的话就同时删除否者，直接返回结果就行（说明前后缀不同）

也就是我们可以用双指针控制首尾元素，把与首尾指针相连的并且与首尾指针相同的元素全部吃掉

## 实现细节

1. 在“吃掉”元素时，范围判断要用`left<=rifht`不然会漏掉一个解

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

写法一：（思路简单实现起来有点复杂）

```C++
class Solution {
public:
    int minimumLength(string s) {
        int n=s.size();
        int left=0;
        int right=n-1;
        char pre,nxt;
        int cnt=0;
        while(left<right){
            pre=s[left];
            left++;
            nxt=s[right];
            right--;
            cnt=2;
            while(s[left]==pre&&left<=right){
                cnt++;
                ++left;
            }
            while(s[right]==nxt&&left<=right){
                cnt++;
                --right;
            }
            if(pre==nxt)n-=cnt;
            else return n;
        }
        return n;
    }
};
```

写法二：思路和实现都不难

```C++
class Solution {
public:
    int minimumLength(string s) {
        int n=s.size();
        int left=0;
        int right=n-1;
        char ch='\0';
        while(left<right){
            ch=s[left];
            if(ch!=s[right])break;
            while(ch==s[left]&&left<=right)left++;
            while(ch==s[right]&&left<=right)--right;
        }
        return right-left+1;
    }
};
```
