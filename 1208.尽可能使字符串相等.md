# 方法一：滑动窗口

用双指针不断滚动满足条件的区间

## 解题思路和过程

为了使字符串相等我们需要不断的开销cost来改变字符，并且我们的**cost有上限，这也意味着不一定能全部改完所有不同的字符**，且题目要求我们返回的就是，最长的长度，这里要注意，要统计的最长长度其中**能统计到的一段字符串一定是连续的**，这里我们可以用滑动窗口的思想来**开一个窗口（连续区间），要保证窗口内的字符串都是相等的**，这个**区间的最大长度就是我们要返回的ans**

这里可以用双指针来确保字符串且cost小于等于maxCost，即每次res（用来记录窗口内的总cost）都可以加上当前的cost，然后当res>maxCost时不断更新left的值即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int equalSubstring(string s, string t, int maxCost) {
        int n=s.size();
        int ans=0;
        int res=0;
        int left=0;
        for(int i=0;i<n;++i){
            res+=abs(s[i]-t[i]);
            while(res>maxCost){
                res-=abs(s[left]-t[left++]);
            }
            ans=max(ans,i-left+1);
        }
        return ans;
    }
};
```

# 方法二：前缀和+二分

## 解题思路和过程

通过上面的方法的分析我们应该能更好地理解这个算法，我们上面的核心思想就是找一个区间，其中**所有的cost<=maxCost,上面我们用的双指针不断维护空间**，这里我们换种思路，**不维护**，而是每对应一个 i 就**找**一次合适的区间，为了更好的表示某一区间的cost我们可以用前缀和的差表示某一区间的cost，毫无疑问**前缀和绝对是有序的，为最快查找我们使用二分**，来**找出第一个使cost<maxCost的坐标-1（前面那一个）**，这时有**`i-left`就是区间的大小**（用前缀和的差时，**第一个数是用第一个前缀和减去第零个前缀和（0），也就是我们，在求前缀和时0的位置存0**）

## 复杂度

- 时间复杂度:$O(nlogn)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int equalSubstring(string s, string t, int maxCost) {
        int n=s.size();
        vector<int> diff(n+1,0);
        for(int i=1;i<=n;++i){
            diff[i]=abs(s[i-1]-t[i-1]);
            if(i>0)diff[i]+=diff[i-1];
        }
        int ans=0;
        for(int i=1;i<=n;++i){
            int left=0;
            int right=i;
            while(left<=right){
                int mid=(left+right)>>1;
                if(diff[i]-diff[mid]>maxCost){
                    left=mid+1;
                }else{
                    right=mid-1;
                }
            }
            ans=max(ans,i-left);
        }
        return ans;

    }
};
```
