# 方法一：贪心

这个数据范围暴力算法已不在适用

## 解题思路

在模拟暴力解法的过程中我们可以发现，我们可以换种思路，将问题转化另外两种看法：

1. 先完成陆地，后完成水上
2. 先完成水上，后完成陆地

以1为例，先完成陆地就意味着水上的结束时间就是完成时间，而陆地的结束时间自然是越早越好（我们要的结果是任务完成的最早时间），这时也会面对上面的两种情况，二者是否有交集，

1. 倘若有交集，完成时间=陆地的结束时间+水上花费时间
2. 如果没有交集，完成时间=水上的开始时间+水上的结束时间

而我们可以直接写一个函数来分别以陆地，水上作为先完成的活动，在二者中取最小值就是答案

## 实现细节

1. 不用写两遍分别以谁先开始的代码，可以写一个函数，使用时交换数据位置即可
2. 是否有交集产生的结束时间可以用`max(minval,waterStartTime[i])+waterDuration[i]`来计算

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int solve(vector<int>& landStartTime, vector<int>& landDuration, vector<int>& waterStartTime, vector<int>& waterDuration){
        int minval=INT_MAX;
        for(int i=0;i<landStartTime.size();++i){
            minval=min(minval,landStartTime[i]+landDuration[i]);
        }
        int minStart=INT_MAX;
        int ans=INT_MAX;
        for(int i=0;i<waterStartTime.size();++i){
            ans=min(ans,max(minval,waterStartTime[i])+waterDuration[i]);
        }
        return ans;
    }

    int earliestFinishTime(vector<int>& landStartTime, vector<int>& landDuration, vector<int>& waterStartTime, vector<int>& waterDuration) {
        int landStart=solve(landStartTime,landDuration,waterStartTime,waterDuration);
        int waterStart=solve(waterStartTime,waterDuration,landStartTime,landDuration);
        return min(landStart,waterStart);
    }
};
```
