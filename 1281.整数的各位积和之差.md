# 方法一

## 解题思路和过程

依题意我们只需要把num的每个数给分开，然后用两个数分别承接各位数之积（初始化为1）和各位数之和即可，最后将两数相减再返回

## 复杂度

- 时间复杂度:O($$d$$),num的位数
- 空间复杂度:O($$1$$)

## Code

```
class Solution {
public:
    int subtractProductAndSum(int n) {
        int num1=1;
        int num2=0;
        while(n){
            num1*=n%10;
            num2+=n%10;
            n/=10;
        }
        return num1-num2;
    }
};
```
