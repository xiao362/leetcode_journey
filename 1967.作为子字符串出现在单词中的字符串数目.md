# 方法一：遍历统计（contains函数）

## 解题思路

通过库函数contains判断，字符串是否是word的子串，如果是个数就加一，否则不变

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int numOfStrings(vector<string>& patterns, string word) {
        int ans=0;
        for(auto ch:patterns){
            if(word.contains(ch))ans++;
        }
        return ans;
    }
};
```
