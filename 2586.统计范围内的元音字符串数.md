# 方法一：暴力

## 解题思路和过程

因为他已经给了要判断的words的范围，我们直接遍历其间的每一个单词即可，写一个lambda函数判断是否是元音字母，倘若首尾都是元音字母，那就ans++，最后返回ans即可

## 复杂度

- 时间复杂度:$O(n)$，n是left到right的距离
- 空间复杂度:$O(1)$

## Code

```c++
class Solution {
public:
    int vowelStrings(vector<string>& words, int left, int right) {
        auto isyuan=[&](char ch)->bool{
            switch(ch){
                case 'a':
                case 'e':
                case 'i':
                case 'o':
                case 'u':
                    return true;
                    break;
                default: 
                    return false;     
            }
        };
        int ans=0;
        for(int i=left;i<=right;++i){
            int sz=words[i].size();
            if(isyuan(words[i][0])&&isyuan(words[i][sz-1])) ans++;
        }
        return ans;
    }
};
```
