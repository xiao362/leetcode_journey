# 方法一：双指针

## 解题思路

用左右两个指针控制两个可以交换字母的位置，使两个指针一直处于可以交换的状态（确保是字母），然后交换二者后，左指针右移，右指针左移即可，直到两个指针相遇即可停止

## 实现细节

1. 在更新两个指针位置以及交换是否停止，都要有`left<right`这个条件

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$(1)$

## Code

```C++
class Solution {
public:
    string reverseOnlyLetters(string s) {
        int n=s.size();
        int left=0;
        int right=n-1;
        while(left<right){
            while(!isalpha(s[left])&&left<right)++left;
            while(!isalpha(s[right])&&left<right)--right;
            swap(s[left],s[right]);
            ++left;
            --right;
        }
        return s;
    }
};
```
