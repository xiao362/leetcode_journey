#方法一：二分查找分割线法(划分数组)

##解题思路和过程

在数组已经有序的情况下求中位数，倘若只有一个数组，那只需要分两种情况讨论：
1. 这段数的个数(长度设为n)为奇数，直接找到中间那个数即可，坐标为(n+1)/2
2. 长度为偶数,找到中间的两个数求平均值

但考虑到这个题目是**两个数组**，同时题目还要求，时间复杂度为**O(log(n+m))**级别的，**那么基本易想的暴力方法：将两个数组合并**就行不通了，所以只能将两个数组同时进行查找，那么这时就要将两个数组分为**主动和被动**来思考了，不然就太麻烦了

> 将两个数组分为左半部分和右半部分，两个数组的左半部分个数和，同单个数组的坐标无异，用此来找中间值

这时在主动数组中设置mid（用i来表示）中值在数组上查找
j在被动数组上查找**（n+m+1)/2,两个数组个数和的中值**
- i=(left+right)/2
- j=(n+m+1)/2-i
- **为了防止数组越界，将nums1设为较短的数组（被动数组范围大，可为主动数组兜底）**

> 在查找时只需要满足 
- nums1[i-1]<=nums2[j]
- nums2[j-1]<=nums1[j]
**同样为了防止数组越界要在i==0,i==n,j==0,j==m时将边界值进行处理：

```
	int nums1maxvalleft = (i==0) ?INT_MIN:nums1[i-1];
    int nums1minvalright = (i==n) ? INT_MAX:nums1[i];
    int nums2maxvalleft = (j==0) ? INT_MIN:nums2[j-1];
    int nums2minvalright = (j==m) ? INT_MAX:nums2[j];

```

然后就是二分查找边界的变动，只需要注意，变动的方向的条件即可

##复杂度

- 时间复杂度：O(log(min(n,m))
- 空间复杂度：O(1)

##Code

```
class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        int n=nums1.size();
        int m=nums2.size();

        if(n>m){
            return findMedianSortedArrays(nums2,nums1);
        }

        int l=0,r=n;

        while(l<=r){

            int i=(l+r)/2;
            int j=(m+n+1)/2-i;

            int nums1maxvalleft = (i==0) ?INT_MIN:nums1[i-1];
            int nums1minvalright = (i==n) ? INT_MAX:nums1[i];
            int nums2maxvalleft = (j==0) ? INT_MIN:nums2[j-1];
            int nums2minvalright = (j==m) ? INT_MAX:nums2[j];

            if(nums1maxvalleft <= nums2minvalright && 
            nums2maxvalleft <= nums1minvalright){
                if((n+m)%2){
                    return (double)max(nums1maxvalleft,nums2maxvalleft);
                }else{
                    return (double)(max(nums1maxvalleft,nums2maxvalleft)
                    +min(nums1minvalright,nums2minvalright))/2;
                }
            }else{
                if(nums1maxvalleft>nums2minvalright){
                    r=i-1;
                }else if(nums2maxvalleft>nums1minvalright){
                    l=i+1;
                }
            }
        }

        return 0;

    }
};
```

#方法二：二分

##解题思路和过程

和前面的思路略有差别，中心流程为：**确认假若两个数组合并后的中间值的大小k-->然后以k-1为坐标比较两个数组对应数值的大小，小的及证明其及其前面的数都是比中位数小的，然后就可以排除这些数**

> 这一点的实现方法可以使用两个头指针分别定位两个数组的‘头’（因为数组是有序的）
- 同时不要忘记用k减去所排除数的个数即：**k-=index1-head+1**;
- 千万不要忘记，**边界的处理，防止数组越界**如**min(head1+k/2-1,n-1)**，这是对应的index的值

然后就是二分查找mid的变化了，不同于一般的二分，这里的二分值的变化主要体现在**k值的变化**，通过不断压缩k的大小来实现，每次最多可以压缩掉k值的一半，故该算法的时间复杂度略逊于二分查找分割法，依我的拙见，该算法的思路和实现要略比方法一麻烦

##复杂度

- 时间复杂度：O(log(n+m))
- 空间复杂度：O(1)

##Code

```class Solution {
public:

    double find(vector<int>& nums1, vector<int>& nums2,int k){
        int n=nums1.size();
        int m=nums2.size(); 
        int head1=0;
        int head2=0;

        while(1){
            if(head1==n){
                return nums2[head2+k-1];
            }
            if(head2==m){
                return nums1[head1+k-1];
            }
            if(k==1){
                return min(nums1[head1],nums2[head2]);
            }

            int index1=min(head1+k/2-1,n-1);
            int index2=min(head2+k/2-1,m-1);
            if(nums1[index1]<=nums2[index2]){
                k-=index1-head1+1;
                head1=index1+1;
            }else{
                k-=index2-head2+1;
                head2=index2+1;
            }
        }
    }

    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        int n=nums1.size()+nums2.size();
        if(n%2){
            return find(nums1,nums2,n/2+1);
        }else{
            return (find(nums1,nums2,n/2)+find(nums1,nums2,n/2+1))/2;
        }
    }
};```
