# 方法一：暴力模拟

## 解题思路和过程

通过对所给的样例数据观察，我们发现轮转时用的x就等于第一个`nums[i]<nums[i-1]`的idx，因为这个数组是非递减的，我们可以先通过一遍遍历得到idx，然后再遍历一遍，当`nums[(idx+i)%n]>nums[(idx+i+1)%n`时返回false即可（也就是使数组不能经轮转后得到），否则返回true

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    bool check(vector<int>& nums) {
        int n=nums.size();
        int idx=0;
        for(int i=1;i<n;++i){
            if(nums[i]<nums[i-1]){
                idx=i;
            }
        }
        for(int i=0;i<n-1;++i){
            if(nums[(idx+i)%n]>nums[(idx+i+1)%n])return false;
        }
        return true;
    }
};
```

# 方法二：数据规律

## 解题思路和过程

对样例数据进行观察我们可以发现，nums原数组倘若能经过轮转后变为有序数组，那么他会产生逆序元素的次数就一定小于等于1，利用这一点，我们用sorted统计元素逆序的次数，并且为了建立头和尾的联系我们用`(i+1)%n`来表示下一个坐标,最后返回sorted<=1的结果即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    bool check(vector<int>& nums) {
        int n=nums.size();
        int sorted=0;
        for(int i=0;i<n;++i){
            if(nums[(i+1)%n]<nums[i])sorted++;
        }
        return sorted<=1;
    }
};
```
