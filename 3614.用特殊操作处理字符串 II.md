# 方法一：逆向思维

## 解题思路

观察数据：

- $1 <= s.length <= 10^5$
- `s` 只包含小写英文字母和特殊字符 `'*'`、`'#'` 和 `'%'`。
- $0 <= k <= 10^{15}$
- 处理 `s` 后得到的 `result` 的长度不超过 $10^{15}$。

倘若我们我们按照他给的字符串，另开一个字符串进行模拟，这样时间复杂度是$O(n^2)$（对整个字符串进行操作，这几乎已经是最优解）毫无疑问会超时，所以我们需要另外寻找其他办法，注意到他要求我们找的是一个字符，那我们是否可以通过坐标进行操作呢，已知我们要面对的四种操作：

- 如果字符是 **小写** 英文字母，则将其添加到 `result` 中。
- 字符 `'*'` 会 **删除** `result` 中的最后一个字符（如果存在）。
- 字符 `'#'` 会 **复制** 当前的 `result` 并**追加**到其自身后面。
- 字符 `'%'` 会 **反转** 当前的 `result`。

这时如果我们要找k坐标对应的字符，首先它的长度一定要达到k，又因为处理 `s` 后得到的 `result` 的长度不超过 $10^{15}$，那我们通过模拟操作对长度的变化，先判断是否存在这个字符，然后再通过逆向操作对长度的反向变化，找出k在逆向操作时对应的位置，例如，最后一次操作是复制，而k的坐标刚好在这次操作后，满足，那么k的坐标对应的字符不就能在复制前字符串中找到了吗，比如操作前长度为len

`len-1<k`，操作后长度为`2*len`，且`2*len-1>=k`，那么k对应的字符不就与没操作前`s[k-len]`相等吗，类似的如果`k<=len-1`时，反转操作使得我们要找的坐标变成了`len-1-k`（字符串反转了）

这样我们就能通过逆向操作不断改变我们要找的k的坐标最终找到我们要的字符串了，但是有一个重要的问题就是什么时候确定我们找到了呢

答案是当`k==len-1`并且该位置对应的操作是添加字母的时候，因为字符串中的每一个字符都是这样添加过来的，并且我们是逆向操作，在这个操作中len要改变长度的，所以直接返回就可以了，删除反转复制等的炒作我们已经考虑过了

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    char processStr(string s, long long k) {
        long long n=s.size();
        long long len=0;
        for(int i=0;i<n;++i){
            if(s[i]=='*'&&len){
                len--;
            }else if(s[i]=='#'){
                len*=2;
            }else if(islower(s[i])){
                len++;
            }
        }
        if(len-1<k)return '.';
        for(int i=n-1;i>=0;--i){
            if(s[i]=='#'){
                len/=2;
                if(len-1<k){
                    k-=len;
                }
            }else if(islower(s[i])){
                if(len-1==k)return s[i];
                len--;
            }else if(s[i]=='%'){
                k=len-1-k;
            }else{
                len++;
            }
        }
        return 0;
    }
};
```

另一种写法，通过一个数组来记录长度

时间复杂度不变，空间复杂度是:$O(n)$

```C++
class Solution {
public:
    char processStr(string s, long long k) {
        long long n=s.size();
        vector<long long> l(n);
        long long len=0;
        for(int i=0;i<n;++i){
            if(s[i]=='*'&&len){
                len--;
            }else if(s[i]=='#'){
                len*=2;
            }else if(islower(s[i])){
                len++;
            }
            l[i]=len;
        }
        if(len-1<k)return '.';
        for(int i=n-1;i>=0;--i){
            len=l[i];
            if(s[i]=='#'){
                len/=2;
                if(len-1<k){
                    k-=len;
                }
            }else if(islower(s[i])){
                if(len-1==k)return s[i];
            }else if(s[i]=='%'){
                k=len-1-k;
            }
        }
        return 0;
    }
};
```
