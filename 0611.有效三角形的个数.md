# 方法一：双指针

## 解题思路

要统计有效三角形的个数，即在数组中选出三个不同坐标的三个值，使这三个值能够围成一个三角形，也就是：

1. 满足两边之和大于第三边

   > 注意：第三边是三者中最大的边

2. 或者满足两边之差小于第三边

   > 这里第三边是最小的边

本质上这两个条件是一个意思，但是却引出了不同的写法

从数组中选出三个数，如果用暴力的方法就是用三个指针，时间复杂度也就是$O(n^3)$，会超时，我们要优化掉一层，可以使用固定一个数，另外两个数用双指针（时间复杂度$O(n)$）就能满足要求

## 复杂度

- 时间复杂度:$O(n^2)$
- 空间复杂度:$O(1)$

## 写法一:两边之和大于第三边

把第三遍看为固定值，在剩余数中找到另外两边，使得满足条件，我们可以把查找另外两边的右指针设置到第三边的左边，这样就能时刻满足第三边是三者中的最大边

## 实现细节

1. 因为数组有序，所以左右边界的和大于固定值的时候，右边界加上区间内任意一个值都满足条件，所以每次答案更新为`ans+=right-left`
2. 不满足条件左边界右移，使二者的和变大，知道满足条件或者二者相遇

## Code

```C++
class Solution {
public:
    int triangleNumber(vector<int>& nums) {
        sort(nums.begin(),nums.end());
        int n=nums.size();
        int ans=0;
        for(int i=n-1;i>1;--i){
            int left=0;
            int right=i-1;
            while(left<right){
                if(nums[left]+nums[right]>nums[i]){
                    ans+=right-left;
                    right--;
                }else{
                    left++;
                }
            }
        }
        return ans;
    }
};
```

## 写法二：滑动窗口/同向双指针（两边之差小于第三边）

我们设置最小边为第三边（另外两个指针放在他的右边），然后用两个指针保证右边指针与左边指针的差值都小于固定值（第三边），因为数组是有序的，倘若边界满足条件，那么边界内所有的元素都会满足条件（左边界变大，差值变小）

所以个数变化为`ans+=right-left`

## 复杂度

- 时间复杂度:$O(n^2)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int triangleNumber(vector<int>& nums) {
        sort(nums.begin(),nums.end());
        int n=nums.size();
        int ans=0;
        for(int i=0;i<n-2;++i){
            int left=i+1;
            for(int right=i+2;right<n;++right){
                while(nums[right]-nums[left]>=nums[i]&&left<right){
                    left++;
                }
                ans+=right-left;
            }
        }
        return ans;
    }
};
```
