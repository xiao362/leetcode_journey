# 方法一：滑动窗口

## 解题思路和过程

为了知道一个字符字串的最大长度，我们可以设置一个”窗口”，既是一个不定长的区间，保证区间内的每个字符最多出现两次，可以用两个指针来维护区间长度，i 作为主动指针，不断在前面加入数据，left在后面维护区间符合要求，用差值来求长度，ans不断更新最大值，为了方便统计每个元素的个数，故用hashmap来存储数据以及查询数据

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$，数组的最大长度

## Code

```C++
class Solution {
public:
    int maximumLengthSubstring(string s) {
        int n=s.size();
        int ans=0;
        int left=0;
        unordered_map<char,int> mp;
        for(int i=0;i<n;++i){
            mp[s[i]]++;
            while(mp[s[i]]>2){
                mp[s[left++]]--;
            }
            ans=max(ans,i-left+1);
        }
        return ans;
    }
};
```
