# 方法一：滑动窗口

## 解题思路

题目要求我们求出一个数组的子数组内的所有元素的乘积小于k的连续子数组的个数，首先子数组即代表一个来连续的区间，我们可以开一个窗口不断维护区间边界，使得区间内的元素乘积都小于k

观察数据示例1：

> 输入：nums = [10,5,2,6], k = 100
> 输出：8
> 解释：8 个乘积小于 100 的子数组分别为：[10]、[5]、[2]、[6]、[10,5]、[5,2]、[2,6]、[5,2,6]。
> 需要注意的是 [10,5,2] 并不是乘积小于 100 的子数组。

因为我们是在求乘积小于k的子数组的个数，像数组`[5,2,6]`满足条件的话，那他的所有子数组一定都满足条件，而我们每次也只需要加上，每次数组变长后新增的子数组个数即可，就像数组`[5]`，增加为`[5,2]`，比原来增加了`[2],[5,2]`两个子数组每次增加`right-lrft+1`个数

## 实现细节

1. `k<=1`时每个数都不满足条件直接返回0即可
2. 每次都要加上满足条件的子数组的个数`ans+=i-left+1`;

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int numSubarrayProductLessThanK(vector<int>& nums, int k) {
        if(k<=1)return 0;
        int n=nums.size();
        int multiply=1;
        int left=0;
        int ans=0;
        for(int i=0;i<n;++i){
            multiply*=nums[i];
            while(multiply>=k){
                multiply/=nums[left++];
            }
            ans+=i-left+1;
        }
        return ans;
    }   
};
```
