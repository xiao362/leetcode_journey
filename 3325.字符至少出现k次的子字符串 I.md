# 方法一：滑动窗口

## 解题思路

题目要求我们统计一个字符串中**至少有一个** 字符 **至少出现** `k` 次的子字符串总数

因为子字符串个数较多，所以传统的暴力算法肯定会爆时间，且题目要求我们是子字符串的个数，即代表他是连续的，那我们就可以用滑动窗口来解决，我们让窗口处于恰好不满足条件的状态，即`[left,right]`是不满足条件的，但`[left-1,right]`，`[left-1,right]`，`[left-3,right]`......`[0,right]`都是满足条件的，个数也就是left个

## 实现细节

1. 用`unordered_map<int,int> mp`来统计其中个元素出现的次数，（也可以用vector必经只有26个字母）
2. 用while以`mp[s[i]]>=k`来不断维护区间的左边界
3. 用`ans+=left`来计数

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int numberOfSubstrings(string s, int k) {
        int ans=0;
        int n=s.size();
        int left=0;
        unordered_map<int,int> mp;
        for(int i=0;i<n;++i){
            mp[s[i]]++;
            while(mp[s[i]]>=k){
                mp[s[left++]]--;
            }
            ans+=left;
        }
        return ans; 
    }
};
```
