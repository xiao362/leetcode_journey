# 方法一：快排sort

## 解题思路和过程

所谓的“好”数组，不过是在原来数组排序后，数组是从1到n-1逐渐递增的，且除最后一位外没有重复的数，故我们可以对数组进行排序，排序后分两种情况讨论：

1. i !=n-1时，只需要使此时的nums[ i ]= i+1即可
2. i ==n-1时，也只用使nums[ i ] == n-1即可

如果不能满足上面两种情况的任意一种，直接return false即可，如果程序能遍历完全程，那就证明其是“好”数组，直接return true即可

## 复杂度

- 时间复杂度:O($$n*logn$$)
- 空间复杂度:O($$log n$$)，算上初始时给的数组就是O:($$n$$),空间复杂度取决于 `std::sort` 的实现，通常需要 **O(log n)** 的递归栈空间（内省排序），若不计输入数组则额外空间为 O(log n)。

## Code

```
class Solution {
public:
    bool isGood(vector<int>& nums) {
        int n=nums.size();
        sort(nums.begin(),nums.end());
        for(int i=0;i<n;++i){
            if(i!=n-1){
                if(nums[i]!=i+1)return false;
            }else{
                if(nums[i]!=n-1)return false;
            }
            
        }
        return  true;
    }
};
```

# 方法二：统计频率

## 解题思路和过程

思路基本与上面方法一致，只是将排序改成统计出现频率，我们可以直接开一个hashmap统计数字的出现次数，方便查询，然后再重头遍历，当 i 没到最后时次数都是1，最后一个数的出现次数为2，有点需要注意，就是长度为1的数组可以直接返回false，不然可能进不去循环

## 复杂度

- 时间复杂度:O($$n$$)
- 空间复杂度:O($$n$$)

## Code

```
class Solution {
public:
    bool isGood(vector<int>& nums) {
        int n=nums.size();
        if(n==1)return false;
        unordered_map<int,int> mp;
        for(int it:nums){
            mp[it]++;
        }
        for(int i=1;i<=n-1;++i){
            if(i!=n-1){
                if(mp[i]!=1)return false;
            }else{
                if(mp[n-1]!=2)return false;
            }
        }
        return true;
    }
};
```
