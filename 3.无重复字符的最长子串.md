#方法一：暴力哈希(本质还是滑动窗口，只是懒省事直接用HashMap)

##解题思路和过程
与滑动窗口同样思路，我们只需要保证一段区间内没有重复的值即可，利用HashMap，查询快，不重复的特点，不断的取最大值
- 在遇到一个数时只需判断，这个数是否在这个区间里，如果不在计数值t就++，不断地取t的最大值
- 如果在这个区间里，那就清空HashMap，从重复的这一位的下一位继续循环，直到走完

##复杂度
- 时间复杂度：O(n^2)（最坏）
- 空间复杂度：O(min(n, ∣Σ∣))

##Code
```
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char ,int> mp;
        int n=s.size();
        if(n==1)return 1;
        int maxval=0;
        int t=0;
        int i=0;
        while(i<n){
            if(mp.count(s[i])){
                maxval=max(t,maxval);
                i=mp[s[i]]+1;
                mp.clear();
                t=0;
                continue;
            }else{
                t++;
                mp[s[i]]=i;//emplace(s[i],i)也可以；
                i++; 
            }
        }
        maxval=max(maxval,t);
        return maxval;
    }
};
```

##滑动窗口优化
其中hashmap改用hashset，空间小了点；
同时用左右index明确位置，较标准的滑动窗口模板
- 右指针可以用-1来区分区间大小
##复杂度
时间复杂度：O(n)
##Code
```
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_set<char> mp;
        int n=s.size();
        int r=0;
        int ans=0;
        for(int i=0;i<n;++i){
            if(i!=0){
                mp.erase(s[i-1]);
            }

            while(r<n&&!mp.count(s[r])){
                mp.insert(s[r]);
                ++r;
            }
            ans=max(ans,r-i);
        }

        return ans;

    }
};
```
