# 方法一：暴力

## 解题思路

题目要求我们求多个区间的最大值以及最小值，且同一个区间可以被反复利用，那我们只需要求最大值以及最小值然后用二者的差值*k即可

## 实现细节

1. 注意用long long

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    long long maxTotalValue(vector<int>& nums, int k) {
        int minval=INT_MAX;
        int maxval=INT_MIN;
        long long ans=0;
        for(int i=0;i<nums.size();++i){
            if(nums[i]<minval)minval=nums[i];
            if(nums[i]>maxval)maxval=nums[i];
        }
        ans=k*((long long)maxval-minval);
        return ans;
    }
};
```
