# 方法一：滑动窗口

## 解题思路和过程

题目要求我们只能采两种水果，且要求每棵树上只能采一个，并且在遇到除了要采的水果外的第三种水果时要停止采摘（**这就保证了，所摘水果的连续性，也就是摘的水果在坐标上是连续的，那也就代表唯一一段连续的区间**），这段连续的区间就引出我们**滑动窗口**思想，我们可以**用双指针来确保区间内最多只有两种水果**，并以此来计算里面水果的个数，通过**不断的更新来取得最大值ans**，代码实现思路如下

右指针不停遍历fruits数组，left控制窗口的左边界，当区间内水果的种类数大于2时不断更新left直到种类小于等于2，又因为**我们主要用到的是区间内水果的种类数和个数**，至于**顺序有fruits来保证**，那我们可以**直接开hashmap来统计个数**

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$，mp中最多存三个数，是常数级别

## Code

```C++
class Solution {
public:
    int totalFruit(vector<int>& fruits) {
        int n=fruits.size();
        int ans=0;
        int left=0;
        unordered_map<int,int> mp;
        int cnt=0;
        for(int i=0;i<n;++i){
            if(mp[fruits[i]]==0)cnt++;
            mp[fruits[i]]=i;
            while(cnt>2){
                mp[fruits[left]]--;
                if(mp[fruits[left++]]==0)cnt--;
            }
            ans=max(ans,i-left+1);
        }
        return ans;
    }
};
```

> 也可以用mp.size()来代替cnt，不过注意要搭配erase()来使用不然可能使数据偏小
