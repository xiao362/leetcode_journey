# 方法一：滑动窗口

## 解题思路和过程

这是比较标准的不定长滑动窗口问题，（只要窗口内的0的个数大于k就缩小区间）我们用left维持窗口的左边界，cnt记录0的个数，用right-left+1，计算区间长，ans更新最大区间长

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int longestOnes(vector<int>& nums, int k) {
        int n=nums.size();
        int ans=0;
        int left=0;
        int cnt=0;
        for(int i=0;i<n;++i){
            if(nums[i]==0)cnt++;
            while(cnt>k){
                if(nums[left++]==0)cnt--;
            }
            ans=max(ans,i-left+1);
        }
        return ans;
    }
};
```
