# 方法一：

## 解题思路和过程

可以用while开两层循环，外层控制当一次求和后的结果是否为个位数，为个位数是停止，内层循环进行对不同位数的求和

## 复杂度 

- 时间复杂度:O($$log num$$)
- 空间复杂度:O($$1$$)

## Code

```
class Solution {
public:
    int addDigits(int num) {
        while(num/10){
            int ans=0;
            while(num){
                ans+=num%10;
                num/=10;
            }
            num=ans;
        }
        return num;
    }
};
```

# 方法二：数根

## 解题思路和过程

利用数学知识：

任何一个十进制数都可以表示为它的各位数字之和，再加上 9 的倍数：

例如 `38 = 3 × 10 + 8 = 3 × (9 + 1) + 8 = 3 × 9 + (3 + 8)`

推广到一般情况：

text

```
num = a_k × 10^k + ... + a_1 × 10 + a_0
    = a_k × (9...9 + 1) + ... + a_1 × (9 + 1) + a_0
    = (9 的倍数) + (a_k + ... + a_1 + a_0)
```



因此，`num` 和它的各位数字之和对模 9 同余。反复应用这个性质，就能得到最终的数根。



一个非负整数的**数根**，就是将它所有位数上的数字相加，如果结果大于 9，就继续把结果的各位数相加，直到得到一个一位数为止。
这个最终的一位数，就是该数的数根。

例如：`38` → `3 + 8 = 11` → `1 + 1 = 2`，所以 38 的数根是 2



## 复杂度

- 时间复杂度:O($$1$$)
- 空间复杂度:O($$1$$)

## Code 

```
class Solution {
public:
    int addDigits(int num) {
        return num==0 ? 0 : num%9==0 ? 9:num%9;
    }
};
```
