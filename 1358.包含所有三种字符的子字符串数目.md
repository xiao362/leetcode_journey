# 方法一：滑动窗口

## 解题思路

题目要求我们统计字符串s中同时含有abc的字串的个数，关于统计一个区间是否同时含有abc的问题我们可以用滑动窗口的思想用一个vector来计数：

反向思考：

当满足条件时我们会遇到一种情况，假若窗口满足同时拥有abc那么这时我们无论再加任何字母他都是符合条件的的，那这时满足条件的字串个数，就是n-i

正向思考：

我们让窗口处于恰好不满足条件的状态，即`[left,right]`是不满足条件的，但`[left-1,right]`，`[left-1,right]`，`[left-3,right]`......`[0,right]`都是满足条件的，个数也就是left个

二者都只需在吗，满足条件时更新左边界left即可

## 实现细节

1. 反向思路时，满足条件的窗口可以看作是从left到n，即也就意味着每次更新left都要算一次ans+=i-n
2. 正向思路时，满足条件的窗口看做从0到 i 只需要在更新 i 时计算ans+=left即可
3. 关于二者是否会漏解，我想是不会了，二者都相当于遍历了各种情况，只是有些一定满足的满足的没有直接遍历，而使用公式代替遍历罢了

## 复杂度

- 时间复杂度:$O(n|\Sigma|)$,$|\Sigma|=3$是字符集的个数
- 空间复杂度:$O(|\Sigma|)$

## Code

反向

```C++
class Solution {
public:
    int numberOfSubstrings(string s) {
        int ans=0;
        int n=s.size();
        int left=0;
        int checked=0;
        vector<int> check(3,0);
        for(int i=0;i<n;++i){
            int p=(s[i]&31)-1;
            check[p]++;
            if(check[p]==1)checked++;
            while(checked==3){
                int q=(s[left]&31)-1;
                check[q]--;
                if(check[q]==0)checked--;
                ans+=n-i;
                left++;
            }
        }
        return ans;

    }
};
```

正向

```C++
class Solution {
public:
    int numberOfSubstrings(string s) {
        int n=s.size();
        int ans=0;
        int left=0;
        int cnt[3]{};
        for(int i=0;i<n;++i){
            cnt[s[i]-'a']++;
            while(cnt[0]&&cnt[1]&&cnt[2]){
                cnt[s[left++]-'a']--;
            }
            ans+=left;
        }
        return ans;
    }
};
```
