# 方法一：贪心

## 解题思路

因为每买两个糖果就会送一个糖果，为了让花费最小，并且买完所有糖果，我们可以让每次送的糖果的花费尽可能的大，这样就能保证我们有最小的花费，并且题目要求送的糖果要比花钱买的便宜，为此我们可以将cost排序，让后用钱买最贵的两个，让他送剩下糖果中最贵的，这样我们就能保证每次送的都是最贵的，这也是贪心的思想

## 实现细节

1. 用cnt记录我们买的的个数，如果到第三个我们就跳过，并把cnt重新置零
2. 遍历顺序要从大到小

## 复杂度

- 时间复杂度:$O(nlogn)$，主要是排序的时间消耗
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int minimumCost(vector<int>& cost) {
        int ans=0;
        int n=cost.size();
        int cnt=0;
        sort(cost.begin(),cost.end());
        for(int i=n-1;i>=0;--i){
            cnt++;
            if(cnt==3){
                cnt=0;
                continue;
            }
            ans+=cost[i];
        }
        return ans;
    }
};
```
