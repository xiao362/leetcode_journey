# 方法一：滑动窗口

**关于滑动窗口，一般我们的滑动窗口都是，使窗口内恰好不满足所给的条件，这样在窗口的左边界改变之前既不影响我们统计数，左边界的改变还给下一个要进来的数留足了位子，免得再分出条件来判断是否满足条件**

## 解题思路

这道题我们研究的对象是一个子数组，那我们也想采用滑动窗口来解决，但是我们发现如果我们想使窗口恰好满足条件，常规办法（直接以相等为条件）并不能保证不漏解，因为我们在这**个过程中会遇到0，+0大小不改变但是确实形成了一个新数组**

为此我们转换思路，**我们不统计恰好等于，而是统计大于等于**，我们**统计出大于等于goal的在统计出大于等于goal+1的二者的差就是恰好等于goal的**，**统计大于等于比较容易实现**，因为倘若一个数组满足条件，那无论在加入任何一个新元素（包括0）都会满足条件

我们让**窗口处于恰好不满足条件的状态**，即`[left,right]`是不满足条件的，但`[left-1,right]`，`[left-1,right]`，`[left-3,right]`......`[0,right]`都是满足条件的，**个数也就是left个**，这样我们就能得到大于等于某个值的个数了

## 实现细节

1. 可以把**滑动窗口写成一个函数solve，这样就不用写两遍了**
2. 对于**`goal==0`要单独处理，因为带入到函数中，会没有止境的一直循环下去（别忘了减去`goal+1`的）**
3. solve中返回值ans的个数增加公式是`ans+=left`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int solve(vector<int>& nums, int goal){
        int n=nums.size();
        if(goal==0)return n*(n+1)/2;
        int left=0;
        int ans=0;
        long long sum=0;
        for(int i=0;i<n;++i){
            sum+=nums[i];
            while(sum>=goal){
                sum-=nums[left];
                left++;
            }
            ans+=left;
        }
        return ans;
    }
    int numSubarraysWithSum(vector<int>& nums, int goal) {
        int n=nums.size();
        if(goal==0)return n*(n+1)/2-solve(nums,1);
        return solve(nums,goal)-solve(nums,goal+1);
    }
};
```

# 方法二：前缀和+hashmap

## 解题思路

同样注意到我们的研究对象是一段区间，且要计算的也是一段区间的和，这里我们要**有前缀和来计算区间和**，然后我们只需统计某段区间的和恰好等于goal的个数即可注意到区间和是两个前缀和的差，即**sum[i]-sum[x]代表x+1到 i 之间的和让他等于goal就是条件**，那我们是否可以反过来，**让sum[i]-goal，通过sum[x]的个数来判断区间和等于goal的个数**

## 实现细节

1. 要**先更新mp在更新sum，因为前缀和是从0开始的**
2. **用hashmap来统计每个前缀和的个数（不一定一直为一，遇到0个数就会增加），即不会漏解**

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int numSubarraysWithSum(vector<int>& nums, int goal) {
        int n=nums.size();
        int ans=0;
        int sum=0;
        unordered_map<int,int> mp;
        for(int i=0;i<n;++i){
            mp[sum]++;
            sum+=nums[i];
            ans+=mp[sum-goal];
        }
        return ans;
    }
};
```
