# 方法一：

## 解题思路和过程

最好想的办法就是直接开一个二维循环，不断遍历nums数组，只要nums[i]!=nums[j]就让ans++；

## 复杂度 

- 时间复杂度:O($$n^2$$)
- 空间复杂度:O($$1$$)

## Code

```
class Solution {
public:
    int numIdenticalPairs(vector<int>& nums) {
        int ans=0;
        int n=nums.size();
        for(int i=0;i<n-1;++i){
            for(int j=i+1;j<n;++j){
                if(nums[i]==nums[j])ans++;
            }
        }
        return ans;
    }
};
```
