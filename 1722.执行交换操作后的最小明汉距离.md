#方法一：并查集+哈希表

##解题思路及过程

依题可知，我们可以将数组中的元素（在他所给的特定指向的元素）进行交换，如果是有顺序并且有限定次数的交换，那就只需统计和按要求交换即可，但既然题目已经明确指出，可以**不按顺序，且不限制次数**的交换，那我们就可以得到一些性质了：

-  在那些**直接或者间接有交换关系的元素**他们之间是可以**随意交换位置的**

- 如[1,3],[3,5]之间，如果1要到达5的位置，只需要先交换1，3后交换3，5，反之亦可，这就类似与**冒泡排序**，每轮循环都有一个元素到达指定位置（**过程也是通过相互交换实现的**）

> 在统计他们之间直接或者间接可以交换的元素的关系时我们可以用**并查集**，将可以交换的元素的关系，用连通图来表示；并查集的基本代码如下：

```
		void ini(vector<int>& check,int m){
            for(int i=0;i<m;++i){
                check[i]=i;
            }
        }
        int find(vector<int>& check,int x){
            if(check[x]==x)return x;
            else return check[x]=find(check,check[x]);
        }uff

        void merge(vector<int>& check,int x,int y){
            int l=find(check,x);
            int r=find(check,y);

            check[l]=r;
        }
```

而题目所谓的最小明汉距离，不过是：**在交换过后，交换过数组与目标数组不同元素的个数的最小值，也就是经过交换不能与target数组对应的元素相同**罢了

在解决了交换的问题之后，还有**统计的问题需要解决**
- 我们可以将在同一个连通图的source坐标对相应的元素放在一起(同一个元素不能出现在多个连通图中，根据这个性质，我们可以用哈希表)
- 开一个二维的哈希表，两个键分别储存**根(root)和图中的元素**(根可以理解为不同连通图的序号)二者都不会重复，其中他们的值赋为1（用来判断是否已经归位），归位后--即可
- 然后用遍历数组target找出那些不在对应数组中的元素即可

> 注意，根的大小只与数组的索引有关，不会重复

##复杂度

- 时间复杂度: O(n)
- 空间复杂度: O(n^2)

##Code

```
class Solution {
public:

    void ini(vector<int>& check,int m){
            for(int i=0;i<m;++i){
                check[i]=i;
            }
        }

        int find(vector<int>& check,int x){
            if(check[x]==x)return x;
            else return check[x]=find(check,check[x]);
        }

        void merge(vector<int>& check,int x,int y){
            int l=find(check,x);
            int r=find(check,y);

            check[l]=r;
        }

    int minimumHammingDistance(vector<int>& source, vector<int>& target, vector<vector<int>>& allowedSwaps) {
        int n=source.size();
        vector<int> check(n);

        unordered_map<int,unordered_map<int,int>> mp;
        ini(check,n);

        for(auto it:allowedSwaps){
            merge(check,it[0],it[1]);
        }

        for(int i=0;i<n;++i){
            int root=find(check,i);
            mp[root][source[i]]++;
        }
        int ans=0;
        for(int i=0;i<n;++i){
            int root=find(check,i);
            if(mp[root][target[i]]>0){
                mp[root][target[i]]--;
            }else{
                ++ans;
            }
        }
        return ans;
    }
};
```

##优化

**对并查集的优化：(按秩合并)
只需要将合并merge函数修改即可**


> 路径压缩已经很强大了，一般情况下加入秩序合并可能会使运行时间变长
加入按秩合并后运行时间反而变长是一个真实存在的现象，而且在实际编码中并不罕见。

```
 vector<int> rank;
 void merge(int x, int y) {
        int rootX = find(x);
        int rootY = find(y);
        if (rootX == rootY) return;

        // 按秩合并
        if (rank[rootX] < rank[rootY]) {
            parent[rootX] = rootY;
        } else if (rank[rootX] > rank[rootY]) {
            parent[rootY] = rootX;
        } else {
            parent[rootY] = rootX;
            rank[rootX]++;
        }
    }
```

#方法二：bfs+hashmap

##Code

```
    class Solution {
    public:
        int minimumHammingDistance(vector<int>& source, vector<int>& target, vector<vector<int>>& allowedSwaps) {
            int n=source.size();
            vector<vector<int>> arr(n);
            vector<bool> vis(n,0);
            unordered_map<int,int> mp;

            for(auto it : allowedSwaps){
                arr[it[0]].push_back(it[1]);
                arr[it[1]].push_back(it[0]);
            }
            int ans=0;
            for(int i=0;i<n;++i){
                mp.clear();
                if(!vis[i]){
                    vector<int> co;
                    queue<int> q;

                    q.push(i);
                    vis[i]=1;
                    while(!q.empty()){
                        int t=q.front();
                        q.pop();
                        co.push_back(t);
                        for(auto j:arr[t]){
                            if(!vis[j]){
                                q.push(j);
                                vis[j]=1;
                            }
                        }
                    }
                    for(auto idx:co){
                        ++mp[source[idx]];
                    }

                    for(auto idx:co){
                        if(mp[target[idx]]){
                            mp[target[idx]]--;
                        }else{
                            ans++;
                        }
                    }
                }
            }
            return ans;

        }
    };
```
