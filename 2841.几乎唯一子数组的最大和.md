# 方法一：滑动窗口

## 解题思路和过程

依照题意，让我们统计几乎唯一子数组的最大和，首先了解到如果 `nums` 的一个子数组有至少 `m` 个互不相同的元素，我们称它是 **几乎唯一** 子数组。其题目要求的长度为`k`的几乎为子数组，那么这个问题就能通过定长滑动窗口来解决，我们设返回值变量为ans，初始为ans，要想更新ans的值，要满足下面的条件：

1. 长度达到`k`
2. 是几乎唯一子数组，也就是数组中至少有m个互不相同的元素，说到互不相同，我们可以采用hashmap（res）来统计互不相同的元数个数，不能暴力的遍历统计，不然只是统计这个的时间复杂度就来到了$O(k^2)$，k是定长区间长度，再加上遍历nums数组的时间复杂度，毫无疑问会爆时间
3. 数组的和最大

其中关键的一步就是不同元素个数的统计，我们即不同元素个数为cnt，初始化为0，在定长区间每次加入数时，判断如果是定长区间没有过的数cnt++（`res[nums[i]]==1`），以及在出定长区间时，如果是唯一的数出去了（`res[nums[i]]==0`）cnt--，在更新cnt同时不要忘记更新定长区间的和`sum`,注意sum和ans要定义为long long类型，不然会爆数据

 ## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(k)$，`k` 是hashmap空间花销，也是最大的不相同元素的个数

## Code

```C++
class Solution {
public:
    long long maxSum(vector<int>& nums, int m, int k) {
        long long ans=0;
        int n=nums.size();
        unordered_map<int,int> res;
        int cnt=0;
        long long sum=0;
        for(int i=0;i<n;++i){
            res[nums[i]]++;
            sum+=nums[i];
            if(res[nums[i]]==1)cnt++;
            if(i<k-1)continue;
            if(cnt>=m)ans=max(ans,sum);
            res[nums[i-k+1]]--;
            sum-=nums[i-k+1];
            if(res[nums[i-k+1]]==0)cnt--;
        }
        return ans;
    }
};
```
