# 方法一：双指针

同向双指针

## 解题思路

最简单的暴力思路就是，找到所有元素等于`key`的坐标`i`，然后把这些坐标中 所有`|i - j| <= k` 的坐标`j`全部存入返回数组中

这时我们会面对，如何保证不重复存储同一坐标，就是问题的关键，我们可以再添加一个指针，控制我们已经添加过的元素的边界，这个边界只能向右移动而不能向左移动，这样就能保证我们不存储重复元素了

## 实现细节

1. 左边界通过条件`left=max(left,right-k)`来更新
2. 每次条件元素时必须保证`left<=min(n-1,right+k)`不越界

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    vector<int> findKDistantIndices(vector<int>& nums, int key, int k) {
        int n=nums.size();
        vector<int> ans;
        int left=0;
        int right=0;
        while(right<n){
            if(nums[right]==key){
                left=max(left,right-k);
                while(left<=min(n-1,right+k)){
                    ans.push_back(left);
                    left++;
                }
            }
            right++;
        }
        return ans;
    }
};
```
