# 方法一：模拟

## 解题思路

按照他给的情况进行模拟就行，遇到相应的元素进行相应的操作

> 进行删除最后一个元素是要确保字符串不为空，不然会报错

## 实现细节

1. 删除用erase
2. 添加用append，或者直接+=
3. 反转用reverse

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string processStr(string s) {
        string ans="";
        for(auto ch:s){
            if(ans.size()==0&&!islower(ch))continue;
            if(ch=='#'){
                ans+=ans;
            }else if(ch=='*'){
                ans.erase(ans.size()-1);
            }else if(ch=='%'){
                reverse(ans.begin(),ans.end());
            }else{
                ans+=ch;
            }
        }
        return ans;
    }
};
```

另一种写法：

```C++
class Solution {
public:
    string processStr(string s) {
        string ans = "";
        for (auto ch : s) {
            if (ch == '#') {
                ans += ans;
            } else if (ch == '*') {
                if (ans.size() != 0)
                    ans.pop_back();
            } else if (ch == '%') {
                ranges::reverse(ans);
            } else {
                ans += ch;
            }
        }
        return ans;
    }
};
```
