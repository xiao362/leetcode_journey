# 方法一：枚举元素个数

## 解题思路

要想形成`n`个气球，那么气球字符串的每一个字符都应该大于等于`n`而`l`和`o`这两个字符应该大于等于`2n`，也就是我们可以反方向的，同个最小的字符可以形成的气球个数来判断，能形成多少个气球，以此来判断气球个数

我们可以用数组，或者用hashmap来存储元素个数

## 复杂度

- 时间复杂度:$O(n)$，n是字符串长度
- 空间复杂度:$O(|\Sigma|)$，$|\Sigma|=出现的字符种数$

## Code

```C++
class Solution {
public:
    int maxNumberOfBalloons(string text) {
        unordered_map<int,int> mp;
        for(char ch:text){
            mp[ch]++;
        }
        int a=min(mp['b'],mp['a']);
        int b=min(mp['l']/2,mp['o']/2);
        int ans=min(a,b);
        return min(ans,mp['n']);
    }
};
```
