# 方法一：两次扫描，传播约束

## 解题思路

在满足他给出的条件下，我们要达到最高的建筑，我们的每一步都要达到最高的高度，为此我们用贪心的思路（就是尽可能地每一步都使楼层的高度增加`1`），尽可能使每一步都达到最大，为此我们可以把上面的条件分为三方面来解决：

1. 如果这栋楼存在高度限制，那么它的高度就不能高过这个限制
2. 正向的我们楼层高度差都小于等于`1`，正向遍历每一步楼层的高度加`1`
3. 反向的我们楼层高度差都小于等于`1`，反向遍历

但是因为$2 <= n <= 10^9$,这也就意味着我们不能开一个长度为n的数组来记录每一层的最高楼层高度，为此我们要进一步优化算法，因为每个相邻的楼层之间层数+`1`，而我们把视角集中在受限制的楼层之间，我们可以只通过坐标的差就可以计算出高度受限制得楼层的最高高度（通过上面分析的方法我们能去除任何位置的最高高度，但是我们存不下，这也就意味着我们不能逐一遍历每个位置（包括不受限制的位置）），在我们已经知道两楼层最高高度的基础上，我们是否可以等到这两层楼之间最高楼层的高度呢，答案是可以的

我们会面对三种情况：

1. 两层楼位置的差的绝对值小于高度差的绝对值，这种情况是不合理的因为前面的位置加上高度差的绝对值小于后面位置的高度，这样是不满足上面的三种情况的，也就是不成立的
2. 二者相等，那最该高度就是二者中较高的高度
3. 前者大于后者，那么我们就可以把位置差分为两部分：
   - 一部分用来弥补高度
   - 另一部分用来形成最高高度，（因为形成后还要回到二者中较高的高度，那么就是对称的，去一半形成最高高度即可）

综上，二者的**最高高度可以通过公式$max(a,b)+\lfloor  d-|b-a|  \rfloor$,`a`,`b`分别表示两层楼的高度，`d`表示二者的差**

这样就优化完成了

## 实现细节

1. 事先以位置对二维数组排序时，不要进行值传递不然会多次复制会浪费大量的时间，进而引起超时
2. 限制1和2可以写在一起，在二者中取较小值即可
3. 在找最高的高度时，不要忘记头（第一栋楼的高度一定是0）和尾（最后一个受限制的楼层不一定是最后一个楼层）的两种情况

## 复杂度

- 时间复杂度:$O(m\ log m)$，`m`是受限制的楼的个数，时间复杂度主要受排序的限制
- 空间复杂度:$O(m)$

## Code

```C++
class Solution {
public:
    int maxBuilding(int n, vector<vector<int>>& restrictions) {
        int len=restrictions.size();
        if(len==0)return n-1;
        sort(restrictions.begin(),restrictions.end(),[&](const vector<int>& a,const vector<int>& b)->bool{
            return a[0]<b[0];
        });
        vector<int> list(len);
        list[0]=min(restrictions[0][0]-1,restrictions[0][1]);
        for(int i=1;i<len;++i){
            list[i]=min(restrictions[i][0]-restrictions[i-1][0]+list[i-1],restrictions[i][1]);
        }
        for(int i=len-2;i>=0;--i){
            list[i]=min(list[i],restrictions[i+1][0]-restrictions[i][0]+list[i+1]);
        }
        int ans=max(list[0]+(restrictions[0][0]-1-list[0])/2,list[len-1]+n-restrictions[len-1][0]);
        for(int i=1;i<len;++i){
            int res=max(list[i],list[i-1])+(restrictions[i][0]-restrictions[i-1][0]-abs(list[i]-list[i-1]))/2;
            ans=max(ans,res);
        }
        return ans;
    }
};
```
