# 方法一：双指针

## 解题思路

这题需要一定的成最多水的容器的基础，就是在许多边中找到最大存水量

我们在两端分别设置两个指针作为接水的容器边界，那么在这个容器之间我们能接到水的情况就是在遇到比我们两个边界中较低（horizon）的一边还要低的边,并且在这个容器中还可以存在其他的容器（由不完全同于这两边的边组成），那我们需要继续找其他的容器，这时我们面临移动那个边的问题，答案是移动较小的一边，较大的一边才能组成更大的容器

**我们求面积时是以较低的一边为高的，也就是说如果我们移动较高的一边，只能通过像距离远的位置移动才能使面积变大，如果距离变小的话，无论遇到比之前高多少的边，它的面积都只会减小，遇到低的边就更不用说了，那也就代表中间的全被淘汰了，那么假如我们使用两个指针从两端出发，他的距离就没办法变大，里面的又被排除（与里面的都被遍历一遍无异），所以如果我们想要面积变得更大，就只能通过移动较小的边才有可能增加面积，而移动过后我们又会面临同样的问题：怎么使最大面积变大（不用回退了，我们统计最大面积，刚刚的情况已经考虑完了），那我们只能接着移动较小的一边了，如此反复最大面积就求出来了**

在这个过程中我们需要不断更新容器的高度（horizon），找容器是自外向里找的，也就是说我们遇到的比(horizon低的边都可以存雨水

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int trap(vector<int>& height) {
        int ans=0;
        int n=height.size();
        int left=0;
        int right=n-1;
        int horizon=0;
        while(left<right){
            if(height[left]<height[right]){
                if(height[left]<horizon){
                    ans+=horizon-height[left];
                }
                horizon=max(horizon,height[left]);
                left++;
            }else{
                if(height[right]<horizon){
                    ans+=horizon-height[right];
                }
                horizon=max(horizon,height[right]);
                right--;
            }
        }
        return ans;
    }
};
```

# 方法二：单调栈

## 解题思路

通过单调栈构建一个单调递减的柱子高序列，这样就能不断的找出能够存水的区间，同样通过水位的办法计算出每次能存水的体积

我们单调栈要严格遵守单调递减，不存相同的元素，这样的话也能保证遇到相同的高度元素时，能够计算他的水的体积

## 实现细节

1. 每次遇到当前高度大于等于栈顶元素的高度的时候，就要计算两个柱子中间能存水的体积，因为积水不一定是从0高度开始的，所以我们弹出来的第一个元素高度作为我们积水的底部，再次弹出元素高度才是积水的另一根柱子
2. 如果只能弹出一个元素说明没办法形成积水，直接break就行
3. 如果还能有另外一个元素满足条件就说明能形成积水，通过两根柱子的较小值计算水的高度即可
4. 长度通过坐标来计算，所以栈里面存的也要是坐标

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        int ans = 0;
        stack<int> st;
        for (int i = 0; i < n; ++i) {
            while (!st.empty() && height[i] >= height[st.top()]) {
                int bottom_h=height[st.top()];
                st.pop();
                if(st.empty())break;
                int left=st.top();
                int dh=min(height[left],height[i])-bottom_h;
                ans += dh * (i - left - 1);
            }
            st.push(i);
        }
        return ans;
    }
}; 

```

# 方法三：前后缀分解

## 解题思路

在前两个方法的基础上，我们知道能否积水的关键在于，当前位置的左右两边是否存在大于a当前位置的高度的柱子，且积水量等于两边最高之柱子的最小值减去当前位置的高度，那么我们可以通过计算前后缀的最大值（因为每个位置对应的不同我们用数组存起来），来找到两边最高的两个柱子

## 实现细节

1. 可以先找后缀的不同位置的最大值，因为在找前缀的时候可以同时通过前后缀计算，每个位置的积水量并累积起来（少层循环）

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int trap(vector<int>& height) {
        int n=height.size();
        vector<int> pre_max(n);
        vector<int> suf_max(n);
        suf_max[n-1]=height[n-1];
        pre_max[0]=height[0];
        for(int i=n-2;i>=0;--i){
            suf_max[i]=max(suf_max[i+1],height[i]);
        }
        int ans=0;
        for(int i=1;i<n;++i){
            pre_max[i]=max(pre_max[i-1],height[i]);
            ans+=min(pre_max[i],suf_max[i])-height[i];
        }
        return ans;
    }
};
```
