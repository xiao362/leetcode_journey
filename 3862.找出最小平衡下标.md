# 方法一：单指针/前后缀分解

## 解题思路

按照题意模拟，我们可以分别用两个数来统计一个指针左边的和以及右边的积，倘若指针从左至右依次移动，那么我们就要先求出指针右边的积，根据数据

- `1 <= nums.length <= 105`
- `1 <= nums[i] <= 109`

就算是long long也会有可能溢出，所以我们要换个方向从右向左移动指针，这样我们就只需要事先求出前面的和就行了

> 注意：这里的和与积都不包括指针所在的位置

然后我们再依次移动指针就行了，遇到相等的就直接返回

> 和至左向右只增而不减，积就相反，这也就意味着最多有一个解

然后就是溢出问题，因为和最大到`1e14`所以积一旦大于该值直接返回-1即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int smallestBalancedIndex(vector<int>& nums) {
        int n=nums.size();
        int ans=INT_MAX;
        long long sum=0;
        for(auto i:nums)sum+=i;
        sum-=nums[n-1];
        if(sum==1)return n-1;
        long long multiple=1;
        int left=n-1;
        while(multiple<sum){
            if(multiple>(long long)1e14/nums[left])return -1;
            multiple*=(long long)nums[left];
            left--;
            sum-=nums[left];
            if(sum==multiple){
                return left;
            }
        }
        return ans==INT_MAX?-1:ans;
    }
};
```

优化

```C++
class Solution {
public:
    int smallestBalancedIndex(vector<int>& nums) {
        long long sum=reduce(nums.begin(),nums.end()-1,0LL);
        long long mul=1;
        for(int i=nums.size()-1;i>0;--i){
            if(sum==mul)return i;
            sum-=nums[i-1];
            if(mul>sum/nums[i]){
                return -1;
            }
            mul*=nums[i];
        }
        return -1;
    }
};
```

# 方法二：双指针

## 解题思路

与单指针类似，不过我们运用双指针替班停止在相遇的位置的性质，同样用两个数统计，和与积，因为前面说过和是至左向右递增的，积是自右向左递增的，所以和大于积的话右指针左移，反之左指针右移

结束时如果二者相等，就返回相遇位置，否者就返回-1

> 注意：溢出问题，因为和最大到`1e14`所以积一旦大于该值直接返回-1即可
>
> 小技巧，用相除规避一次溢出

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int smallestBalancedIndex(vector<int>& nums) {
        long long sum=reduce(nums.begin(),nums.end()-1,0LL);
        long long mul=1;
        for(int i=nums.size()-1;i>0;--i){
            if(sum==mul)return i;
            sum-=nums[i-1];
            if(mul>sum/nums[i]){
                return -1;
            }
            mul*=nums[i];
        }
        return -1;
    }
};
```
