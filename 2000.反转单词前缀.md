# 方法一：单指针

## 解题思路

我们先遍历一遍如果没有遇到ch直接返回word即可，倘若遇到ch，我们就把ch位置以及之前的区间进行反转

反转可以用双指针，且这个双指针是可以相互表示的，那就也可以用单指针，或者直接用库函数reverse反转

> 注意没有遇到ch要直接返回

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string reversePrefix(string word, char ch) {
        int n=word.size();
        for(int i=0;i<n;++i){
            if(word[i]==ch){
                n=i;
                break;
            }
        }
        if(n==word.size())return word;
        char t='\0';
        n++;
        for(int i=0;i<n/2;++i){
            t=word[i];
            word[i]=word[n-i-1];
            word[n-i-1]=t;
        }
        return word;
    }
};
```

双指针:

```C++
class Solution {
public:
    string reversePrefix(string word, char ch) {
        int n=word.size();
        for(int i=0;i<n;++i){
            if(word[i]==ch){
                n=i;
                break;
            }
        }
        if(n==word.size())return word;
        char t='\0';
        int right=n;
        for(int i=0;i<(n+1)/2;++i){
            t=word[i];
            word[i]=word[right];
            word[right]=t;
            right--;
        }
        return word;
    }
};
```
