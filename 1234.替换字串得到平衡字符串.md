# 方法一：滑动窗口

## 解题思路和过程

题目要求我们将一个只有四种字符的字符串（个数可能不相同）改成同样字符，但各个字符的个数都相同的字符串，且要求我们改变字符串只能通过一个子串来替换原字符串的子串的方式修改，也就是只能一串一串的改必须改一个区间，更确切地说只能改一次，一个区间，要求我们返回，这个子串的最小长度

为了求出这个子串我们要先明确一些事情，要想补齐所有的字符及个数，我们两个思路：

1. 找缺的，即不足的，给加上
2. 找多的给减去

但是在实际模拟更改时我们找缺的就是在找多的位置给补上，所以我们要找多的元素的位置，并画出一个区间，使区间内多的元素恰好能够补上缺的元素，我们发现我们要找的是区间，且区间的维护条件可以实现，我们就能用滑动窗口来实现

算法的核心是：先统计一遍多的元素的个数，然后找一个区间，使得区间内含有足够的多的元素的个数，然后开始缩短区间，使得恰好是多的元素（种类可能是多个，必须同时满足）的个数刚好少一些，给后面的元素留位子

> 注意别忘了在缩短空间前，先统计当前的长度（满足条件），后再缩短

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

```C++
class Solution {
public:
    int balancedString(string s) {
        int cnt['Z']{};
        for(char ch:s){
            cnt[ch]++;
        }
        int n=s.size();
        int m=n/4;
        if(cnt['Q']==m&&cnt['W']==m&&cnt['E']==m&&cnt['R']==m){
            return 0;
        }
        int ans=n;
        int left=0;
        for(int i=0;i<n;++i){
            cnt[s[i]]--;
            while(cnt['Q']<=m&&cnt['W']<=m&&cnt['E']<=m&&cnt['R']<=m){
                ans=min(ans,i-left+1);
                cnt[s[left++]]++;
            }
        }
        return ans;
    }
};
```

思路比较简单的

```C++
class Solution {
public:
    int balancedString(string s) {
        vector<int> cnt(4,0);
        int n=s.size();
        int ans=n;
        for(int i=0;i<n;++i){
            switch(s[i]){
                case 'Q':
                    cnt[0]++;
                    break;
                case 'W':
                    cnt[1]++;
                    break;
                case 'E':
                    cnt[2]++;
                    break;
                default:
                    cnt[3]++;
            }
        }
        for(int i=0;i<4;++i){
            if(cnt[i]-n/4 >0)cnt[i]-=n/4;
            else cnt[i]=0;
        }
        if(cnt[0]==0&&cnt[1]==0&&cnt[2]==0&&cnt[3]==0)return 0;
        vector<int> res(4,0);
        int left=0;
        for(int i=0;i<n;++i){
            switch(s[i]){
                case 'Q':
                    res[0]++;
                    break;
                case 'W':
                    res[1]++;
                    break;
                case 'E':
                    res[2]++;
                    break;
                default:
                    res[3]++;
            }
            while(res[0]>=cnt[0]&&res[1]>=cnt[1]&&
            res[2]>=cnt[2]&&res[3]>=cnt[3]){
                switch(s[left]){
                    case 'Q':
                        res[0]--;
                        break;
                    case 'W':
                        res[1]--;
                        break;
                    case 'E':
                        res[2]--;
                        break;
                    default:
                        res[3]--;
                }
                ans=min(ans,i-left+1);
                left++;
            }
        }
        return ans;
    }
};
```
