# 方法一：滑动窗口

## 解题思路和过程

依题意需要我们求解一个定长字串内元音字母的最大值，这道题是相当经典的滑动窗口题，即我们可以设置一个窗口，这个所谓的窗口里面存储着这个定长字串，我们**只需要不断的更新这段子串中的元音字母数（res）即可**，为了方便理解，我们先用一个队列直观的模拟这个过程，其大致有两种情况需要面对：

1. **当我们这个窗口的长度没有达到k时，我们只需要向里面存数，而在存时我们可以判断进来的数是vowel（元音）与否，如果是res++，不是的话就不操作**
2. **当这个窗口存满的时候，在遇到要进去的数，不仅要存，更要取，存时同样如果是vowel，res++，类似的在取的时候，如果取出来的数是vowel，res--，因为他从这个窗口离开了**

在这个过程中不断更新最大值ans，在数组遍历完后，直接返回ans即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(k)$,k是窗口的长度

## Code

```
class Solution {
public:
    int maxVowels(string s, int k) {
        auto isvowel=[&](char ch){
            switch(ch){
                case 'a':
                case 'e':
                case 'o':
                case 'u':
                case 'i':
                    return true;
                default:
                    return false;
            }
        };
        int ans=0,res=0;
        queue<char> q;
        for(auto ch:s){
            if(q.size()==k){
                if(isvowel(q.front())){
                    res--;
                }
                q.pop();
            }
            if(isvowel(ch)){
                res++;
                ans=max(ans,res);
            }
            q.push(ch);
        }
        return ans;
    }
};
```



# 优化

让我们仔细观察这整个便利的过程，我们发现我们的存和取是不太必要的，因为我们主要依靠对“首尾”的判断进行加减，所以我们省略掉存和取的过程，直接判断,并且在最大变成k之后，直接返回k，因为已经达到最大值无法再变大，直接返回

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```
class Solution {
public:
    int maxVowels(string s, int k) {
        int n=s.size();
        int ans=0;
        int res=0;
        for(int i=0;i<n;++i){
            
            if(s[i]=='a'||s[i]=='e'||s[i]=='o'||s[i]=='u'||s[i]=='i'){
                res++;
            }
            if(i-k+1<0){
                continue;
            }
            ans=max(ans,res);
            if(ans==k)return k;
            if(s[i-k+1]=='a'||s[i-k+1]=='e'||s[i-k+1]=='o'||
            s[i-k+1]=='u'||s[i-k+1]=='i'){
                res--;
            }
            
        }
        return ans;
    }
};
```
