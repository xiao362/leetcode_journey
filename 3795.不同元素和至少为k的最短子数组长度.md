# 方法一：滑动窗口（越长越合）

## 解题思路和过程

**题目要求我们求出一个子数组（以为这是一段连续的区间），区间内的值的和要大于等于k，且每个值只能被用一回，然后求出这个区间的最短长度**，因这是连续的区间，且条件是可以量化的，那我们就可以**用滑动窗口解决**，这题的算法倒不难，**用一个指针维护区间和大于等于k即可**，至于**每个数只用一此那是在计算和时用的条件，与区间大小的维护无关**

我们left维护区间左边界，用hashmap保证计算和的时候只用一次，用**while以`sum>=k`为条件不断改变left的值**，用**`right-left`计算长度**，并**用ans更新最小值**

> 别忘了ans没结果时返回-1,还右更新ans时要保证`sum>+k`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int minLength(vector<int>& nums, int k) {
        int n=nums.size();
        int ans=INT_MAX;
        unordered_map<int,int> mp;
        int left=0;
        int sum=0;
        for(int i=0;i<n;++i){
            mp[nums[i]]++;
            if(mp[nums[i]]==1)sum+=nums[i];
            while(sum>=k){
                if(sum>=k)ans=min(ans,i-left+1);
                mp[nums[left]]--;
                if(mp[nums[left]]==0)sum-=nums[left];
                left++;
            }
        }
        return ans==INT_MAX?-1:ans;
    }
};   
```
