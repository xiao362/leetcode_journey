# 方法一

## 解题思路和过程

因为要使用交换完成任务，那就会牵扯到数的后移，时间复杂度可能会来到$O(n^2)$,所以就用最常见的空间换时间，开一个2n长的数组，然后不断在ans后面添加nums[ i ],nums[ i+n ],这样就完成了要求 

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```c++
class Solution {
public:
    vector<int> shuffle(vector<int>& nums, int n) {
        vector<int> ans;
        for(int i=0;i<n;++i){
            ans.push_back(nums[i]);
            ans.push_back(nums[i+n]);
        }
        return ans;
    }
};
```
