#方法一：暴力

##解题方法和过程

依照题目，我们要求的是执行过n次后距离远点的最远距离，故我们只需要统计三个字符的个数，并**取L和R差值的绝对值再加上'_'的个数即可**

##复杂度

- 时间复杂度:O(n)
- 空间复杂度:O(1)

#Code

```
class Solution {
public:
    int furthestDistanceFromOrigin(string moves) {
        int ans=0;
        int x=0;
        int t=0;
        for(char ch:moves){
            if(ch=='L'){
                x--;
            }else if(ch=='R'){
                x++;
            }else{
                t++;
            }
        }
        ans=abs(x)+t;
        return ans;
    }
};
```
