# 方法一：滑动窗口

## 解题思路

可以把**子字符串看作是一个区间**，分别对窗口内的0，1的个数进行统计，只要**当区间不满足k约束的条件就缩短区间**，使得区间一直满足条件：如果一个 **二进制字符串** 满足以下任一条件，则认为该字符串满足 **k 约束**：

- 字符串中 `0` 的数量最多为 `k`。
- 字符串中 `1` 的数量最多为 `k`。

> 注意一个字符串满足k约束，那他的子串也全都满足k约束

## 实现细节

1. ans统计最终的子字符串的个数，**每次增加`i-left+1`个，对应前面注意**
2. 用left配合while维护区间左边界，**条件是不满足k约束也就是`cnt[0]>k&&cnt[1]>k`**

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int countKConstraintSubstrings(string s, int k) {
        int n=s.size();
        int ans=0;
        int left=0;
        int cnt[2]{};
        for(int i=0;i<n;++i){
            cnt[s[i]-'0']++;
            while(cnt[0]>k&&cnt[1]>k){
                cnt[s[left++]-'0']--;
            }
            ans+=i-left+1;
        }
        return ans;
    }
};
```
