# 方法一：双指针/贪心

## 解题思路

为保证尽量少的能量剩余，只要我们的`power`足够换成分就换成分，为了使能量的效率最高，我们应该**用最少的能量换最多的得分，并用最少的分换最多的能量**，因为**每次操作都涉及当前的最大以及最小**，故我们可以**先将原数组排序**，并**设置两边的指针控制目前最小以及最大的值**

## 实现细节

1. 左右指针相遇时也要判断一次
2. 等分不够，能量也不够的情况要直接返回，不然会死循环

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int bagOfTokensScore(vector<int>& tokens, int power) {
        sort(tokens.begin(),tokens.end());
        int ans=0;
        int score=0;
        int n=tokens.size();
        int left=0;
        int right=n-1;
        while(left<=right){
            if(power>=tokens[left]){
                score++;
                ans=max(ans,score);
                power-=tokens[left];
                left++;
                continue;
            }
            if(score){
                power+=tokens[right];
                right--;
                score--;
                continue;
            }
            return ans;
        }
        return ans;
    }
};
```
