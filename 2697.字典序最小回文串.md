# 方法一：双指针/单指针+贪心

## 解题思路

我们用左右两个指针同时缩小位置，在遇到不想同的字母时，把两个字母变成相同的字典序小的字母

因为这两个指针的位置有关系，可以相互计算所以也可以用单指针

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string makeSmallestPalindrome(string s) {
        int n=s.size();
        for(int i=0;i<n/2;++i){
            if(s[i]<s[n-i-1])s[n-1-i]=s[i];
            else s[i]=s[n-1-i];
        }
        return s;
    }
};
```
