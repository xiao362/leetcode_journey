# 方法一：滑动窗口

## 解题思路和过程

根据题目我们需要求取，数组arr里长度为k且平均值大于等于threshold的子数组的个数，那么**我们可以开一个“窗口”，用sum记录窗口内，个数之和，并用其求平均值，如果平均值大于等于threshold，那ans++**，然后在进窗口并且**计算使用过后，将窗口最前端的数“出窗口”减去他的值**，就这样遍历完整个数组后，返回ans即可

 ## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int numOfSubarrays(vector<int>& arr, int k, int threshold) {
        int n=arr.size();
        int ans=0;
        int sum=0;
        for(int i=0;i<n;++i){
            sum+=arr[i];
            if(i>=k-1){
                if(sum/k>=threshold){
                    ans++;
                }
                sum-=arr[i-k+1];
            }
        }
        return ans;
    }
};
```
