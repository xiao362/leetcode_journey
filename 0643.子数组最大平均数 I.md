# 方法一：滑动窗口

## 解题思路和过程

依据题目要求，要我们求平均数最大值，且长度为k的连续子数组，这也是一个容易想出来的定长滑动窗口，这样我们设ans为最终的返回答案，设sum不断更新窗口内数的和，**为了不漏掉最后一个数组值，我们再每一次遍历中，让可以进的数当场进，计算过后立即出，始终留一个位置给下一个数**，还有一点也要注意，**ans的更新，也就是平均值的求取，只能发生在窗口满的时候**，最后**不要将ans初始化为0，不负数的情况就没办法更新**

 ## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    double findMaxAverage(vector<int>& nums, int k) {
        double ans=INT_MIN;
        double sum=0;
        int n=nums.size();
        for(int i=0;i<n;++i){
            sum+=nums[i];
            
            if(i>=k-1){
                ans=max(ans,sum/k);
                sum-=nums[i-k+1];
            }
        }
        return ans;
    }
};
```
