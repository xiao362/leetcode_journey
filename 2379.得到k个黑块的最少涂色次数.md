# 方法一：滑动窗口

## 解题思路和过程

题目要求我们求解，**在一个定长区间内获得全'B'数组的最小操作数，即是求该区间的’W‘的个数**，这种定区间统计个数的问题，我们可以发现就是**定长滑动窗口问题**，故我们可以设置一个”窗口“，即是表示这个定长区间，**设置cnt统计区间内白色方块的个数**，遍历数组当区间的长度**达到要求长度时，开始更新最少涂色次数ans**，将其初始化为k，因为他不可能比k大，不要忘了在统计过后，让定长区间的头个元素”**出区间“，类似的当他是白色方块时就让cnt--**

 ## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int minimumRecolors(string blocks, int k) {
        int n=blocks.size();
        int ans=k;
        int cnt=0;
        for(int i=0;i<n;++i){
            if(blocks[i]=='W')cnt++;
            if(i>=k-1){
                ans=min(ans,cnt);
                if(blocks[i-k+1]=='W')cnt--;
            }
        }
        return ans;
    }
};
```
