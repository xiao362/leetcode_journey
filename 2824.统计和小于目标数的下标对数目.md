# 方法一：双指针

## 解题思路

因为统计时与大小有关系，而原数组是无序的，所以我们如果要统计两个值的和小于target的数据对的个数，最简单的办法就是用两层循环把每一种情况都枚举出来，实现比较简单，而我们想要优化，就是将数组排序，排序后分别设置在数组的两端，在二者和大于target的时候，移动右指针直到和小于target这时因为非递减排列左指针和两指针区间任何一个数相加都小于target，数量直接加`right-target`个，然后移动左指针，每移动一步就重复一遍上述操作

当两指针相遇时结束即可

## 复杂度

- 时间复杂度:$O(n \log n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int countPairs(vector<int>& nums, int target) {
        ranges::sort(nums);
        //sort(nums.begin(),nums.end());也行
        int n=nums.size();
        int left=0;
        int right=n-1;
        int ans=0;
        while(left<right){
            if(nums[left]+nums[right]<target){
                ans+=right-left;
                left++;
            }else{
                right--;
            }
        }
        return ans;
    }
};
```
