# 方法一：滑动窗口（越长越合）

## 解题思路和过程

题目要求我们找出一个最短的字符串，且要求字符串中的`1`的个数恰好的等于k，并且要求倘如最小长度的字符串有多个的，返回字典序最小的那一个。

由上面可知字符串是连续的，且对字符串的要求是可以量化，那我们就可以使用滑动窗口来维护区间，用双指针（主要是左指针left维护区间的大小，保持区间内`1`的个数刚好小于k，即k-1，使区间一直留一个1的位置给下一个数进），然乎用len不断维护长度的最小值，如果遇到比len小的，或者与len相等且字典序比原来的答案字符串ans小的，就更换点原来ans，最后返回ans即可

以上使代码实现的核心算法，关于上面字典序的问题我提一嘴，是可以直接用>,<来直接比较出来的，不要忘记用cnt记录区间内的`1`的个数

> `std::move` 是 C++11 引入的一个函数，定义在 `<utility>` 头文件中。它的作用是**将左值转换为右值引用**，从而允许编译器选择移动构造函数或移动赋值运算符，而不是拷贝构造函数/拷贝赋值运算符。

## 复杂度

- 时间复杂度:$O(n^2)$，n是字符串的长度，因为过程中可能要比较某一段子串，而且返回的也是字符串
- 空间复杂度:$O(n)$,或者$O(1)$

## Code

```C++
class Solution {
public:
    string shortestBeautifulSubstring(string s, int k) {
        string ans="";
        int len=INT_MAX;
        int cnt=0;
        int left=0;
        for(int i=0;i<s.size();++i){
            if(s[i]=='1')cnt++;
            while(cnt==k){
                if(s[left]=='1')cnt--;
                if((i-left+1)<len){
                    len=i-left+1;
                    ans=s.substr(left,len);
                }else if((i-left+1)==len){
                    ans=ans<s.substr(left,len)? ans:s.substr(left,len);
                }
                ++left;
            }
        }
        return ans;
    }
};
```

# 方法二：暴力枚举

`std::ranges::count(s, '1')` 是 C++20 范围库中的一个函数，用于**统计某个范围内等于指定值的元素个数**。

## 解题思路和过程

如果整个串的`1`的个数大于k这一定有解，反之则无解，直接返回即可

倘若有解，我们可以从最小长度，依次更换长度（递增），直到最先在某一个长度找到最优解，就可以返回（后面的长度更大了没必要找了），并且同长度时我们只需比较字典序，实现时可以用`ranges::count(s, '1')` ，来统计字符串中`1`的个数

## 复杂度

- 时间复杂度:$O(n^3)$，不同长度一层，不同子串起点位置一层，以及对应字符串还有一层
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    string shortestBeautifulSubstring(string s, int k) {
        if(ranges::count(s,'1')<k){
            return "";
        }
        int sz=k;
        string ans="";
        while(sz){
            for(int i=sz;i<=s.size();i++){
                string t=s.substr(i-sz,sz);
                if(ranges::count(t,'1')==k&&(t<ans||ans=="")){
                    ans=t;
                }
            }
            if(!ans.empty()){
                return ans;
            }
            sz++;
        }
        return ans;
    }
};
```
