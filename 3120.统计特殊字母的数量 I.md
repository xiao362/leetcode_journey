# 方法一：bool

## 解题思路

因为出现的字符只有大小写字母，我们可以直接通过字符的ascll码为坐标，直接标注每个字符是否出现过，然后通过遍历二十六个字母（小写）看是否同时出现大小写字母，并通过ans来记录特殊字符的个数

## 实现细节

1. 数组的大小一定到开够，可以开到'Z'或者128

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int numberOfSpecialChars(string word) {
        vector<bool> cnt(128,0);
        int ans=0;
        for(char ch:word){
            cnt[ch]=1;
        }
        for(int i='a';i<='z';++i){
            if(cnt[i]&&cnt[i-32])ans++;
        }
        return ans;
    }
};
```

# 方法二：hashmap（hashset）

## 解题思路

如果你用的是hashmap，你可以在每次遇到一个字符时，判断与之对应的字符是否存在，并通过是否是第一次出现来判断是否要ans++

## 实现细节

1. 如果你开始时就将mp[ch]++,那么你可以通过mp[ch]==1,来判断是否是第一次出现
2. ans++，必须满足有对应字符，和第一次出现两个条
3. 如果你用hashset，那就要像上一个方法再开一个循环

## Code

```C++
class Solution {
public:
    int numberOfSpecialChars(string word) {
        unordered_map<int,int> mp;
        int ans=0;
        for(char ch:word){
            if(ch>='a'&&ch<='z'){
                if(mp[ch-32]&&mp[ch]==0){
                    ans++;
                }
            }else{
                if(mp[ch+32]&&mp[ch]==0){
                    ans++;
                }
            }
            mp[ch]++;
        } 
        return ans;
    }
};
```

# 方法三：二进制

“掩码”这个词听起来很神秘，但它的本质很简单：**用一个数字把你关心的那几位“摘”出来，把不关心的位挡在外面**

## 解题思路

在ascll码的大小字母是由一定规律的基础上，我们可以用二进制来解决，首先大写字母的第六位字符一定是0，小写的一定是1，这方便我们区分数据（存储数据），通过字母与31取交来获得位置编号，并通过将1左移（交集对应的数）位与存储数据的值取并，构成独一无二的掩码，通过popcount统计大小写字母的交集个数

## 代码实现

1. `ch>>5&1`区分大小写字母
2. `1<<(ch&31)`形成独一无二的掩码
3. `popcount(1u*(mp[0]&mp[1]))`看交集有多少个重复字母

## Code

```C++
class Solution {
public:
    int numberOfSpecialChars(string word) {
        vector<int> mp(2,0);
        for(char ch:word){
            mp[ch>>5&1]|=1<<(ch&31);
        }
        return popcount(1u*(mp[0]&mp[1]));
    }
};
```
