# 方法一：双指针

## 解题思路

要想找到最接近x的k个元素，因为原素组是有序的，我们用双指针从数组左右两端出发，通过比较两指针位置值与x的差值的绝对值的大小，来缩小区间（具体思路就是差值大的一边缩小期间范围），直到区间长度为k

整数 `a` 比整数 `b` 更接近 `x` 需要满足：

- `|a - x| < |b - x|` 或者
- `|a - x| == |b - x|` 且 `a < b`

也就是说当二者的差值绝对值相等时我们要保留数值小的那个，也就是左边的那个

## 实现细节

1. 如果差值的绝对值，左边小于等于右边，那右指针就向左移动，反之左指针向右移动

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

优化版本

```C++
class Solution {
public:
    vector<int> findClosestElements(vector<int>& arr, int k, int x) {
        int n=arr.size();
        int left=0;
        int right=n-1;
        while(right-left+1!=k){
            int a=abs(arr[left]-x);
            int b=abs(arr[right]-x);
            if(a>b)left++;
            else right--;
        }
        return vector<int>(arr.begin()+left,arr.begin()+right+1);
    }
};  
```

常规方法

```C++
class Solution {
public:
    vector<int> findClosestElements(vector<int>& arr, int k, int x) {
        int n=arr.size();
        int left=0;
        int right=n-1;
        while(right-left+1!=k){
            int a=abs(arr[left]-x);
            int b=abs(arr[right]-x);
            if(a>b)left++;
            else right--;
        }
        vector<int> ans(k,0);
        for(int i=0;left+i<=right;++i){
            ans[i]=arr[left+i];
        }
        return ans;
    }
};
```

# 方法二：双指针+二分

## 解题思路

该方法本质上与方法一相同，只是这次我们的指针是从中间往外扩

我们通过二分（lower_bound）想找到等于或者大于x的第一个值，设他为右边界（不一定是我们需要的结果），这时区间分为左右两部分，左边小于x，右边大于等于x，`left=right-1`是左区间的最右边元素（也不一定满足条件），所以这个区间目前没有任何元素，是个开区间，然后循环k次，找够k个满足条件的元素，有下面几种情况：

1. 二者其中之一到达边界，这时只能让另一边往外扩
2. 没到达边界时，如果左边与x差值的绝对值大于右边差值的绝对值，此时就往右边扩展，反之就往左边扩展（小于等于左边扩展，上述条件）

## 实现细节

>1. 因为我们区间的左右两边都是开的，所以返回时，如果以vector赋值的形式返回ans就需要左边界的值加一(有边界不用，赋值函数本身就是开的)

## 复杂度

- 时间复杂度:$O(\log n +k)$，k是要返回的数组长度
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    vector<int> findClosestElements(vector<int>& arr, int k, int x) {
        int right=lower_bound(arr.begin(),arr.end(),x)-arr.begin();
        int left=right-1;
        while(k--){
            if(left<0){
                right++;
            }else if(right>=arr.size()){
                left--;
            }else if(x-arr[left]>arr[right]-x){
                right++;
            }else{
                left--;
            }
        }
        return vector<int>(arr.begin()+left+1,arr.begin()+right);
    }
};
```
