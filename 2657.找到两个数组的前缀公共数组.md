# 方法一：hashset+前缀和

## 解题思路和过程

因为A，B两个数组是**一个长度为 `n` 的数组包含 `1` 到 `n` 的元素恰好一次**的长度为n的排列，只是二者的顺序可能会有不同，让我们返回一个下标 i ，表示A，B数组在 i 之前相同元素的个数的数组，因为同一个元素的数组元素不相同，我们可以开一个hashset用来查询前面是否出现过某个元素，倘若出现过我们可以将ans[ i ]++(长度为A.size()，初始化为0，用来充当返回数组，也可以直接用AhuoB数组)，没有的话，就加里面一个，然后对ans求一个前缀和即可

> 求前缀和要从i=1开始

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$，不算返回值

## Code

```C++
class Solution {
public:
    vector<int> findThePrefixCommonArray(vector<int>& A, vector<int>& B) {
        int n=A.size();
        unordered_set<int> set;
        vector<int> ans(n,0);
        for(int i=0;i<n;++i){
            if(set.count(A[i]))ans[i]++;
            else set.insert(A[i]);
            if(set.count(B[i]))ans[i]++;
            else set.insert(B[i]);
            if(i>0)ans[i]+=ans[i-1];
        }
        return ans;
    }
};
```

# 方法二：二进制

利用二进制的位运算加速

用到一个**函数popcount，统计一个整数的二进制表示中1的个数**

## 解题思路和过程

注意到数组A的长度最大是50，那我们可以通过二进制来解题，我们遍历数组A，B再分别用a,b来分别存每个位置的状态（与前面的hashset作用相同）

> 注意要用uint64_t来声明a，b，不然int一般是32位（4个字节）铁定不够用

然后还用到1ULL，其中ULL表示unsigned long long ,也是确保有64位，每次让a|=1ULL<<A[i],B[i]，位表示i的位置前面已经存过了，别忘了取并

然后通过popcount（a&b）看两个里边有多少相同个数即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$,不算返回数组

## Code

```C++
class Solution {
public:
    vector<int> findThePrefixCommonArray(vector<int>& A, vector<int>& B) {
        uint64_t a=0,b=0;
        vector<int> ans(A.size());
        for(int i=0;i<A.size();++i){
            a|=1ULL<<A[i];
            b|=1ULL<<B[i];
            ans[i]=popcount(a & b);
        }
        return ans;
    }
};
```
