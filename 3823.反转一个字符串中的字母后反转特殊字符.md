# 方法一：双指针

## 解题思路

因为我们在交换时只能是同种类型（是否是小写字母）交换，所以我们可以用双指针通过while来随时更新左右边界同类型的边界坐标，交换过后缩小边界即可，使左右指针一直在可以交换的位置

## 实现细节

1. 可以一次循环来完成任务，但是需要多加两个参数

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string reverseByType(string s) {
        int n=s.size();
        int l=0;
        int r=n-1;
        while(l<r){
            while(!islower(s[l])&&l<r)l++;
            while(!islower(s[r])&&l<r)r--;
            swap(s[l],s[r]);
            l++;
            r--;
        }
        l=0;
        r=n-1;
        while(l<r){
            while(islower(s[l])&&l<r)l++;
            while(islower(s[r])&&l<r)r--;
            swap(s[l],s[r]);
            l++;
            r--;
        }
        return s;
    }
};
```
