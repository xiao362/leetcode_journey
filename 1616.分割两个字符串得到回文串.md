# 方法一：双指针

## 解题思路

我们需要判断两个字符串在同一位置分割后在拼接后形成的字符串是否是回文的，且有两种情况就是分别以a,b的前缀为拼接字符串的前缀，要我们判断这两种情况，我们可以写一个solve函数，这样就不用我们写两遍逻辑相同的的函数了，返回结果时，只需要交换a，b的位置，并将两个结果取并就行

下面我们以a为前缀为例写具体的算法

在判断回文子串时我们常用的就是双指针，设置在两端，判断每个字母是否相同知道两指针相遇，但是因为这个子串时拼接的，所以我们把`left`设置在`a`的左边，`right`设置到`b`的右边，然后同时移动两端指针，直到遇到不相同字符为止，这时我们要想满足条件（让做右指针相遇也就代表是回文子串），唯一的条件就是，`a`或者`b`，中间子串本身时回文的，这时我们才有可能在拼接后，字符串还是回文的，同上面一样为避免重复代码，我们在写一个check函数，判断a或者b在left与right之间是否是回文的，这两个条件也是取并集

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    bool check(string a,int left,int right){
        while(left<right&&a[left]==a[right]){
            left++;
            right--;
        }
        return left>=right;
    }

    bool solve(string a, string b) {
        int n = a.size();

        int left = 0;
        int right = n - 1;

        while (a[left] == b[right]&&left<right) {
            left++;
            right--;
        }
        return check(a,left,right)||check(b,left,right);
    }

    bool checkPalindromeFormation(string a, string b) {
        return solve(a,b)||solve(b,a);
    }
};
```
