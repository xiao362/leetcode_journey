# 方法一：状态机

## 解题思路

某个字母 `c` 的小写形式和大写形式，并且 **每个** 小写形式的 `c` 都出现在第一个大写形式的 `c` 之前，则称字母 `c` 是一个 **特殊字母** 

我们要统计特殊字母的个数时我们可能会遇到下面几种情况（以字母a为例）：

1. 一开始先遇到小写字母a，这时可能还会遇到：
   - **遇到大写字母A**，这时就遇到了一个特殊字母，这又可以分两种情况
     - 后面**没有与a相关的字母，或者是遇到大写字母A**，这时特殊字母的状态延续 **（状态：2）**
     - 后面遇到a，字母a**被彻底淘汰****（状态：-1）**
   - 这时如果**后面遇到的是a或者没有与a相关的字母**，此时状态延续**（状态：1）**
2. 一开始**直接遇到大写字母A**，字母a直接**被淘汰****（状态：-1）**

如果我们将**初始状态设为 0** ，那么我们可以**有四种状态**：

1. **初始状态（ 0 ），没遇到与之相关的字母**
2. **等待状态（ 1 ），遇到小写字母，等待大写字母**
3. **特殊字母状态（ 2 ），特殊字母个数+1**
4. **淘汰状态（ -1 ），其不可能再形成特殊字母**

我们可以通过状态的改变来应对我们要面对的不同字母，并更新特殊字母数

## 实现细节

1. 可以用数组`i**nt state[27]={};`来存储状态**
2. **ans可以与状态同步更新**，这样可以节省再次遍历的时间
3. 利用**`p=ch&31`将字母都转化为数字**，方便状态的查看

> **二进制里大小写字母，只有第六位（右到左）0和1的差别**

## 复杂度

- 时间复杂度:$O(n+|\Sigma|)$,$|\Sigma|=26$是字母集合的大小

- 空间复杂度:$O(|\Sigma|)$

> 注意**创建大小为 O(∣Σ∣) 的数组需要 O(∣Σ∣) 时间。**

## Code

```C++
class Solution {
public:
    int numberOfSpecialChars(string word) {
        int ans=0;
        int state[27]={};
        for(char ch:word){
            int p=ch&31;
            if(ch&32){
                if(state[p]==0){
                    state[p]=1;
                }else if(state[p]==2){
                    state[p]=-1;
                    ans--;
                }
            }else{
                if(state[p]==1){
                    state[p]=2;
                    ans++;
                }else if(state[p]==0){
                    state[p]=-1;
                }
            }
        }
        return ans;
    }
};
```

## 二进制写法

流程大致相同，存数据的方式变了，并且**最终要求`lowwer&upper`来确定是否满足条件，所以upper不用再细分化**

**invalid是被淘汰的字母**

> 注意：**popcount函数里要是无符号数**

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int numberOfSpecialChars(string word) {
        int lowwer=0,upper=0,invalid=0;
        for(char ch:word){
            int p=1u<<(ch&31);
            if(ch&32){
                lowwer|=p;
                if(upper&p){
                    invalid|=p;
                }
            }else{
                upper|=p;
            }
        }
        return popcount(1u*(lowwer&upper&~invalid));
    }
};
```
