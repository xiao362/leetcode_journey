# 方法一：建树查询

## 解题思路和过程

题目要求我们求出两个数组arr1以及arr2的最长公共前缀，最直接的办法，暴力将每个数依次比较一边，时间复杂度来到$O(n^2)$,但比较时还要依次比较每个数的前缀（一个一个比），思路无比简单，实现也相当容易，但复杂度已经比$O(n^2)$还大了，毫无疑问会爆时间，那有其他的办法吗，观察数据**我们发现在用暴力实现的过程中有一些数据是没有必要重复比较的**，就像100中已经含有1和10了那么他们两个就没必要比较了

如果只是**模拟查找相同前缀的过程时有：比较1他可以是10，100，110，121，1000等，就这点，我们就能想到树形结构**，这就能i尽量的**避免重复前缀的查询**了，知道如何查询然后就可以用代码实现了

首先就是建树，因为每位最大可能只有10个数，我们直接在里面树里面建一个数组（初始化为空，推荐用nullptr更安全），结构题写好后，遍历arr1建树，可以用string库里的to_string（）转化为字符串后用坐标取数字的前缀，然后就是查询（用`num[ i ]`是否位nullptr来判断是否含有该前缀），用cnt记录长度，并用ans不断更新最大长度

## 复杂度

- 时间复杂度:$O(n+m)$,或者$O(L1+L2)$
- 空间复杂度:$O(n)$,或者$O(L1)$

## Code

```C++
class Solution {
public:
    struct prerequisite{
        prerequisite* num[10];
        prerequisite(){
            for(int i=0;i<10;++i){
                num[i]=nullptr;
            }
        }
    };
    
    int longestCommonPrefix(vector<int>& arr1, vector<int>& arr2) {
        int ans=0;
        prerequisite* pre=new prerequisite();
        prerequisite* next=nullptr;
        for(auto i:arr1){
            string s=to_string(i);
            int sz=s.size();
            next=pre;
            for(int j=0;j<sz;++j){
                if(next->num[s[j]-'0']==nullptr){
                    next->num[s[j]-'0']=new prerequisite();
                }
                next=next->num[s[j]-'0'];
            }
        }
        int cnt=0;
        for(auto i:arr2){
            string s=to_string(i);
            int sz=s.size();
            next=pre;
            cnt=0;
            for(int j=0;j<sz;++j){
                if(next->num[s[j]-'0']!=nullptr){
                    ans=max(ans,++cnt);
                    next=next->num[s[j]-'0'];
                }else{
                    break;
                }
            }
        }
        return ans;
    }
};
```

# 方法二：hashset存前缀



## 解题思路和过程

思路与上面大致相同，我们换一种存储的思路（查询的方法），同样是为了避免重复查询和存储，我们**用无重复的特点，用hashset来存每一个数的每一个前缀**，同样用另一个数组的每一个数的每一个前缀来查，并取得最大值

> 注意：`string.size()` 的返回类型是 **`size_t`**。
>
> `size_t` 是一个**无符号整数类型**（通常定义在 `<cstddef>` 中），具体大小取决于平台（32位系统通常为 `unsigned int`，64位系统通常为 `unsigned long long`），用于表示对象的大小或数组的长度。使用 `size_t` 可以确保跨平台兼容性，并且避免负值问题。

## 复杂度

- 时间复杂度:$O(n+m)$，或着$O(L1+L2)$
- 空间复杂度:$O(n)$,或者$O(L1)$

## Code

```C++
class Solution {
public:
    int longestCommonPrefix(vector<int>& arr1, vector<int>& arr2) {
        unordered_set<string> set;
        int ans=0;
        for(auto i:arr1){
            string s=to_string(i);
            string str="";
            for(char ch:s){
                str+=ch;
                set.emplace(str);
            }
        }
        for(auto i:arr2){
            string s=to_string(i);
            string str="";
            for(char ch:s){
                str+=ch;
                if(set.count(str)){
                    ans=max(ans,(int)str.size());
                }else{
                    break;
                }
            }
        }
        return ans;
    }
};
```

有几个函数值得说一下：



在 C++ 中，`std::set`（以及 `std::unordered_set`）的 `contains` 成员函数返回类型是 **`bool`**。

- `true`：集合中包含该元素。
- `false`：集合中不包含该元素。
- 

*语法:*

```
  basic_string substr( size_type index, size_type num = npos ); 
```

substr()返回本字符串的一个子串，从index开始，长num个字符。如果没有指定，将是默认值  **string::npos**。这样，substr()函数将简单的返回从index开始的剩余的字符串。



> **上面的方法都用的是string类型，比较的，因为比较直观，另一种方法是直接用int类型来写，不过是反着来的，每次先存完整的数，然后每次除以10求出前面的前缀，一直重复这个过程**

```C++
class Solution {
public:
    int longestCommonPrefix(vector<int>& arr1, vector<int>& arr2) {
        unordered_set<string> set;
        int ans=0;
        for(auto str:arr1){
            string s=to_string(str);
            for(int i=0;i<=s.size();++i){
                set.insert(s.substr(0,i));
            }
        }
        for(auto str:arr2){
            string s=to_string(str);
            for(int i=1;i<=s.size();++i){
                if(set.contains(s.substr(0,i))){
                    ans=max(ans,i);
                }else{
                    break;
                }
            }
        }
        return ans;
    }
};
```
