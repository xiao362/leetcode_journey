# 方法一：双指针

## 解题思路

题目要求我们，把每一行进行逆置，我们只需要对每一行进行首尾互换即可，同时他还要求我们互换后对每个数取反，我们可以在互换时直接对数值取反

> 但是同时操作时要注意，因为我们普遍是互换到n/2就结束，所以在遇到计数时，中间会有个数值没有取反，故我们要对这个数单独取反

## 复杂度

- 时间复杂度:$O(n^2)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    vector<vector<int>> flipAndInvertImage(vector<vector<int>>& image) {
        int n=image.size();
        int p=0;
        for(int i=0;i<n;++i){
            for(int j=0;j<n/2;++j){
                p=!image[i][j];
                image[i][j]=!image[i][n-j-1];
                image[i][n-j-1]=p;
            }
            if(n%2)image[i][n/2]=!image[i][n/2];
        }
        return image;
    }
};
```
