# 方法一：数学计算

## 解题思路

通过每分钟，或者每小时分别对应的时针和分针偏转的角度，分别计算出二者的位置，然后通过计算差值的绝对值，并与180比较，大于就用360-ans后返回，否则就直接返回

## 复杂度

- 时间复杂度:$O(1)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    double angleClock(int hour, int minutes) {
        double h=30.0*hour+0.5*minutes;
        double m=6.0*minutes;
        double ans=fabs(h-m);
        return ans<=180?ans:360.0-ans;
    }
};
```
