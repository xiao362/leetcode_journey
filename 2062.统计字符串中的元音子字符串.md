# 方法一：滑动窗口

## 解题思路

题目要求我们找出元音子字符串的个数，既然研究对象是子字符串，也就可以看作是一个连续的区间，一个可以滑动的窗口，那么我们要做的就是开一个**hashmap来统计窗口各个元素的的个数**，当五种元音字母都有了的时候我们就可以计数，证明已经满足条件，这时无论再添加任何元音字母，窗口都是满足条件的

倘若我们让窗口处于恰好不满足条件的状态，即`[left,right]`是不满足条件的，但`[left-1,right]`，`[left-1,right]`，`[left-3,right]`......`[0,right]`都是满足条件的，个数也就是`left-0`个

> 注意当我们**遇到非元音字母的时候，**这时**前面元音字母形成的窗口已经完全统计完**了，且**窗口已经不能再接着使用**了，这时我们就要**清空hashmap**，更重要的是当我们再次遇到满足条件的窗口时，因为再次**统计时窗口的起点不在`0`的位置**，所以我们就用**start来记录起点的位置**，所以上面**统计个数的公式应该是`left-start`**

## 实现细节

1. `int start=0`来统计起点
2. 用while维护区间边界，使得区间处于恰好不满足条件的状态
3. 遇到非原因字母要清空mp，（可以用clear()）
4. 统计个数时用`ans+=(left-start)`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(|\Sigma|)$,$|\Sigma|$是元音字母的种数+1，直接写$O(1)$也可以

## Code

```C++
class Solution {
public:
    int countVowelSubstrings(string word) {
        int n=word.size();
        int left=0;
        int ans=0;
        int start=0;
        unordered_map<int,int> mp;
        for(int i=0;i<n;++i){
            mp[word[i]]++;
            if(word[i]!='a'&&word[i]!='e'&&word[i]!='i'&&word[i]!='o'&&word[i]!='u'){
                start=i+1;
                while(mp[word[i]]){
                    mp[word[left++]]--;
                } 
            }
            
            while(mp['a']&&mp['e']&&mp['i']&&mp['o']&&mp['u']){
                mp[word[left++]]--;
            }
            ans+=(left-start);
        }
        return ans;
    }
};
```
