# 方法一

## 解题思路和过程

因为得分只与左右两个串的0或1的个数有关，那我们，可以在开始前获得s中1的个数，用其表示左子串为空时的得分，然后再遍历s串，因为在计分时左右两个子串不能为空，所以我们不能用foreach遍历，而使用常规的遍历，在最后一个字符前停止。再用ans不断统计最大得分，左边得分为score1,右边为score2=num2（即一开始统计的1的个数，左串为空，右串肯定把分得满了），然后遇到0score++，遇到1，score--，不断更新最大得分即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```c++
class Solution {
public:
    int maxScore(string s) {
        int n=s.size();
        
        int num2=0;
        for(auto it:s){
            if(it=='0'){}
            else num2++;
        }
        int score1=0;
        int score2=num2;
        int ans=0;
        
        for(int i=0;i<n-1;++i){
            if(s[i]=='0')score1++;  
            else score2--;
            ans=max(ans,score1+score2);
        }

        return ans;
    }
};
```
