# 方法一：滑动窗口

## 解题思路和过程

题目要求我们在字符串s上找到一个最短的子串，使得这个最短子串ans含有t串的所有元素（包括种类和个数），倘若没有这样的字串可以直接返回“”（空串），思路还是比较简单的，我们可以按照题目的要求**，开一个窗口**，使得窗口内的元素满足要求即可，由**left维护区间的左边界**

代码实现过程中，可能会遇到的一些问题，首先就是条件的量化，我们可以**开两个数组，分别统计两个子串中出现元素个数**

> 因为**出现的字符都是英文字母**，最大的‘z’的ascll值只有122，我们是可以直接开一个123长的数组（128也可以）来统计分别出现的次数，别怕多余出很多无用的空间，**这点空间节省了大量的时间**

如用**数组require来存字符串t中元素出现次数**，**用数组mp（用hashmap来统计也行）来统计窗口内个元素出现的次数**，统计工作完成后接下来就是ans的更新

首先是更新的条件，我们要判断mp中的所有元素的个数都大于require中对应元素的个数，更准确地说是，**require中所有元素的个数都要小于mp中对应元素的个数**，我们可以直接使52个英文字母都用require以及mp对应比较一下

> 这里直接用52个英文字母比较可以省去一些复杂的在构造，倘若我们想要更准确，只统计require中出现过的元素，免不了在创建数据来统计，况且对于较大的数据，基本所有元素的出现了个遍，而较小的数据这点时间差根本看不出来

但是上面的思路我们每次都要同各种元素，就像我愿本这次23种元素都满足条件了，但就是还差一种，在下一次统计时不免要把前面23种在统计一边，所以我们在原来的基础上，再将判断条件，进一步改变，改成**满足条件元素的种数**，一种满**足了就对应加一**，直到与需要的条件相等，**checked设为满足条件种数，required设为需要满足条件的种数**，最后也是相当关键的一步，**返回值问题**，**如果我们在循环内直接更新返回值（string），就相当于增加了一层循环**，毕竟这个复制过程是相当耗时间的，为此**我们不每次更新返回值string，而是更新他的起点**，在我们**知道ans的起点以及长度**时，最后**只需要复制一遍就能解决问题**

> **别忘了用cnt记录长度才能更新，毕竟是要找最短的子串**

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string minWindow(string s, string t) {
        int m=s.size();
        int n=t.size();
        if(m<n)return "";
        int ans=0;
        vector<int> require('z'+1,0);
        for(int i=0;i<n;++i){
            require[t[i]]++;
        }
        int left=0;
        int cnt=INT_MAX;
        vector<int> mp('z'+1,0);
        int checked=0;
        int required=0;
        for(int i='A';i<='z';++i)if(require[i]>0)required++;
        for(int i=0;i<m;++i){
            mp[s[i]]++;
            if(mp[s[i]]==require[s[i]])checked++;
            while(checked==required){
                mp[s[left]]--;
                if(mp[s[left]]+1==require[s[left]])checked--;
                if(cnt>i-left+1){
                    ans=left;
                    cnt=i-left+1;
                }
                left++;
            }
        }
        if(cnt==INT_MAX)return "";
        return s.substr(ans,cnt);
    }
};
```

```C++
class Solution {
public:
    string minWindow(string s, string t) {
        int m=s.size();
        int n=t.size();
        if(m<n)return "";
        int ans=0;
        vector<int> require('z'+1,0);
        for(int i=0;i<n;++i){
            require[t[i]]++;
        }
        int left=0;
        int cnt=INT_MAX;
        unordered_map<int,int> mp;
        auto check=[&](){
            for(int i='A';i<='z';++i){
                if(mp[i]<require[i])return false;
            }
            return true;
        };
        for(int i=0;i<m;++i){
            mp[s[i]]++;
            while(check()){
                mp[s[left]]--;
                if(cnt>i-left+1){
                    ans=left;
                    cnt=i-left+1;
                }
                left++;
            }
        }
        if(cnt==INT_MAX)return "";
        return s.substr(ans,cnt);
    }
};
```
