# 方法一：双指针

## 解题思路

依据题目我们要想统计，与点对中至少一个点相连的边的个数，我们可以统计每一个点具有的边的个数（度数degree），这样将度数相加后再减去重复的边就能求出我们需要的边的个数，又因为题目要求我们每个查询的答案，也就是：

第 `j` 个查询的答案是满足如下条件的点对 `(a, b)` 的数目：

- `a < b`
- `cnt` 是与 `a` **或者** `b` 相连的边的数目，且 `cnt` **严格大于** `queries[j]` 。

这里`cnt`表示与点对中至少一个点相连的边的个数，也就是我们上面求出的个数（相加后），因为我们只涉及个数的问题，为方便求解（暴力算法，也就是枚举的时间复杂度是$O(n^2)$，太高了，还要占用空间存储，所以我们干脆先不加，统计得到的deg排序后，利用有序性结合双指针来计算符合条件的有序（`a<b`）点对个数），双指针相加后得到的个数，还要筛选掉那些因为有重复边，计算错误的点对个数

对上面出现的一些概念以及思路进行总结：

> 首先就是题目最后给的容易忽略的条件：多重边，起点和终点相同，但是却记为不同的边，`[1,2],[2,1],[1,2]`记为不同的边，

一条边贡献两个度，但是如果这两个点之间存在一(`n`)条边，那么度数和就要减一(`n`)，也就是与点对中至少一个点相连的边的个数减一（`n`），这里因为多重边的性质可能会弄混

所以在上面统计度数的同时，我们要统计不同边的个数，也就是说`[1,2],[2,1],[1,2]`记为同一条边个数为`3`，为方便统计以及映照前面`a<b`的条件，我们统计边的时候，把小的放前面，也引出二进制的计边方法，将`32`位数前`16`位记录`a`，后`16`位记为`b`，然后通过hashmap计数，方便好用

不能事先对度进行处理重复边，只有当统计两点度数和时才能去掉这两点的相同边

## 实现细节

1. 因为我们统计度数之和必须放在去重复边之前（计算满足条件的度数和的时候用到双指针，要把deg排序），而在去重复边的时候要用到原来的deg值，所以我们要复制一个deg数组排序后，用于双指针

2. 去重的公式是`deg[k>>16]+deg[k&0xffff]-c`，`c`是重复边的个数，但是我们最终的目的是统计数对的个数，所以我们只需要在那些没去重是满足条件而去重后不满足条件的时候将`ans[i]--`即可

   > 注意双指针操作是对所有可能的度数之和进行操作，也就表他考虑了所有可能，而去重边的操作同样是以所有边进行操作的，也是全体范围

## 复杂度

- 时间复杂度:$O(n \log n+q(n+m))$，`q`是查询数组长度，`m`是`edges`的长度
- 空间复杂度:$O(n+m)$

## Code

```C++
class Solution {
public:
    vector<int> countPairs(int n, vector<vector<int>>& edges, vector<int>& queries) {
        vector<int> deg(n+1);
        unordered_map<int,int> cnt_e;
        for(auto x:edges){
            deg[x[0]]++;
            deg[x[1]]++;
            if(x[0]<x[1])cnt_e[x[0]<<16|x[1]]++;
            else cnt_e[x[1]<<16|x[0]]++;
        }
        
        int len=queries.size();
        vector<int> sort_deg=deg;
        sort(sort_deg.begin(),sort_deg.end());
        
        vector<int> ans(len,0);
        for(int i=0;i<len;++i){
            int left=1;
            int right=n;
            while(left<right){
                if(sort_deg[left]+sort_deg[right]<=queries[i]){
                    left++;
                }else{
                    ans[i]+=right-left;
                    right--;
                }
            }
            for(auto [k,c]:cnt_e){
                int s=deg[k>>16]+deg[k&0xffff];
                if(s>queries[i]&&s-c<=queries[i]){
                    ans[i]--;
                }
            }
        }
        return ans;
    }
};
```
