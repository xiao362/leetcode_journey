# 方法一：二分

## 解题思路和过程

因为要找到山脉的顶峰索引，即是在一个数组被分为两段顺序相反，但都保持相对有序的数组的最大值的坐标，且该最大值的位置敲好位于俩段数组的交界处，而且**题目要求我们使用时间复杂度为$O(\log(n))$的算法，因为这个时间复杂常见于二分算法，且这个数组在一定程度上是属于相对有序的，故我们可以使用二分**，为此我们要找到合适的check条件以方便能够不断缩小我们探查的位置，我们可以以mid为中心截取出一个长度为3小片段，用来**探测当前mid所处位置的顺序有3种情况：**

1. **mid正好位于顶峰**，那么此时有 arr[mid]>arr[mid-1]&&arr[mid]>arr[mid+1] ,**这时就可以直接返回mid坐标**
2. **mid位于左半坡**，即arr[mid-1]<arr[mid]&&arr[mid]<arr[mid+1]，那么我们可以**使 l =mid+1,压缩空间**
3. 反之，**mid位于右半坡**，即arr[mid-1]>arr[mid]&&arr[mid]>arr[mid+1]，即**令 r =mid-1**

注意不要忘了，特殊情况：***当mid来到左右边界时，mid-1或+1可能会越界，要对这种情况单独处理，来到边界也就意味着前面遇到了左半坡或者右半坡的，因为一开始是来不到边界位置的，只有缩小范围的时候才能来到，同时又因为必须要有递增或者递减的区间，这也意味着，与边界mid相连的就是山峰***

## 复杂度

- 时间复杂度:$O(\log(n))$
- 空间复杂度:$O(1)$

## Code

```c++
class Solution {
public:
    int peakIndexInMountainArray(vector<int>& arr) {
        int n=arr.size();
        int l=0;
        int r=n-1;
        while(l<=r){
            int mid=(l+r)>>1;
            if(mid-1<0)return 1;
            else if(mid+1==n)return n-2;
            if(arr[mid]>arr[mid-1]&&arr[mid]>arr[mid+1]){
                return mid;
            }else if(arr[mid-1]<arr[mid]&&arr[mid]<arr[mid+1]){
                l=mid+1;
            }else if(arr[mid-1]>arr[mid]&&arr[mid]>arr[mid+1]){
                r=mid-1;
            }
        }
        return l;
    }
};
```
