# 方法一：暴力（排序）

## 解题思路

我们直接将原数组排序，排序后按照要求把第一个元素改为1（大于等于1），然后根据条件元素只能缩小，那么我们只要遇到当前元素大于前一个元素的情况，就把当前元素改为前一个元素加一即可

修改后的最大值也就是最后一个元素

**核心思想就是元素大小只能缩而不能增**

> 注定返回值只能小于等于n(元素个数)

## 复杂度

- 时间复杂度:$O(n \log n)$,主要是排序消耗
- 空间复杂度:$O(1)$,不算题目给的数组

## Code

```C++
class Solution {
public:
    int maximumElementAfterDecrementingAndRearranging(vector<int>& arr) {
        int n=arr.size();
        ranges::sort(arr);
        arr[0]=1;
        for(int i=1;i<n;++i){
            if(arr[i]>arr[i-1])arr[i]=arr[i-1]+1;
        }
        return arr[n-1];
    }
};
```

# 方法二：计数排序

## 解题思路

观察上面暴力枚举过程中最大值的变化，我们亦可以换种思路，将ans初始值设为n减去上面修正后的数组中重复元素的个数减一同样是我们的答案（相当于保留一个），那就引出我们下面的算法

我们统计计数排序，先统计每个元素的个数（大于n的记为n），每个元素只能是比当前坐标（从1开始）还小的数，那么我们可以通过

$ans = min(ans + cnt[i], i)$ 筛选掉重复的数据（ans记为前面遇到的数（个数不为0），`i`每次加一，也就意味着ans每次至多+1）

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$
