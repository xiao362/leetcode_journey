# 方法一：滑动窗口（越长越合）

如果没有滑动窗口的基础，可以先看后两个方法

## 解题思路和过程

题目要求我们找出满足其总和大于等于 `target` 的长度最小的 **子数组** ，其中**子数组** 是数组中连续的 **非空** 元素序列。

这也就意味着我们要找一个最短区间，使得区间和大于target即可，那么我们同样可以使用双指针来维护区间长度，**使得区间刚刚小于target**，这样可以**给下一个元素留位置**，我们**并不需要保持区间和大于target**（这样**不好维护**，毕竟我们要找恰好能使区间和大于target的临界值），我们**只需要统计临界时的状态**就行，即**不断用ans更新满足要求的最小值**

代码实现，核心就是**用while以sum>=target为条件维护左边界**

## 复杂度

- 时间复杂度:$O(n)$,一个元素最多访问两遍
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int n=nums.size();
        int ans=INT_MAX;
        int left=0;
        int sum=0;
        for(int i=0;i<n;++i){
            sum+=nums[i];
            while(sum>=target){
                ans=min(ans,i-left+1);
                sum-=nums[left++];
            }
        }
        return  ans==INT_MAX? 0:ans;
    }
};
```

# 方法二：暴力

## 解题思路和过程

这是比较好想的办法，我们以每个元素都为起点依次往后遍历，直到和sum>=target停止，用ans来变换最小值，最后倘若ans的值没有变就说明不存在符合条件的子数组

## 复杂度

- 时间复杂度:$O(n^2)$
- 空间复杂度:$O(1)$

## Code

```
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int n=nums.size();
        int ans=INT_MAX;
        for(int i=0;i<n;++i){
            int sum=0;
            for(int j=i;j<n;++j){
                sum+=nums[j];
                if(sum>=target){
                    ans=min(ans,j-i+1);
                    break;
                }
            }
        }
        return ans==INT_MAX? 0:ans;
    }
};
```

# 方法三：前缀和+二分

是对方法二的优化

## 解题思路和过程

因为我们要求的是区间的和，并以区间和作为判断条件，在求区间和时我们常用前缀和来计算，并且因为前缀和是递增的，故我们可以搭配二分，来**找出第一个时区间和大于target的右边界值**，这种check好判断的（**找第一个大于某个值的位置，可以直接用lowerbound**，但我习惯手搓了），同样不要忘记更新ans最小值

## 复杂度

- 时间复杂度:$O(nlogn)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int n=nums.size();
        int ans=INT_MAX;
        vector<int> diff(n+2,0);
        for(int i=0;i<n;++i){
            diff[i+1]=nums[i]+diff[i];
        }
        for(int i=0;i<=n;++i){
            int left=i+1;
            int right=n;
            while(left<=right){
                int mid=(left+right)>>1;
                if(diff[mid]-diff[i]<target){
                    left=mid+1;
                }else{
                    right=mid-1;
                }
            }
            if((diff[left]-diff[i])>=target)ans=min(ans,left-i);
        }
        return ans==INT_MAX?0:ans;
    }
};
```
