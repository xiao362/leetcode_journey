# 方法一：

## 解题思路和过程

直接开一个长度为2的数组，分别代入公式计算结果返回即可

## 复杂度 

- 时间复杂度:O($$1$$)
- 空间复杂度:O($$1$$)

## Code

```
class Solution {
public:
    vector<double> convertTemperature(double celsius) {
        vector<double> ans(2);
        ans[0]=celsius+273.15;
        ans[1]=celsius*1.80+32.00;
        return ans;
    }
};
```
