# 方法一：两次扫描，传播约束

## 解题思路

我们要构造一个序列，该序列受到几个条件的限制：

- `a[0]` 为 0。
- 序列中的所有元素都是 **非负整数** 。
- 对于每个下标 `i` (`0 <= i <= n - 2`)，满足 `abs(a[i] - a[i + 1]) <= diff[i]`。
- 对于每个 `restrictions[i] = [idx, maxVal]`，序列中位置 `idx` 的值不得超过 `maxVal`（即 `a[idx] <= maxVal`）。

我们的目标是在满足上述所有条件的情况下，构造一个合法的序列并 **最大化** 该序列中的 **最大** 值。

对于上面目标我们可以贪心的使每一次的序列的值都使他与前者（后者，后者可以通过反向遍历来保证，我们先考虑前者）的差值恰好等于对应的diff，在贪心的思路下我们只需要满足三个条件：

1. 我们的值必须小于等于对应的districtions
2. 与前者的差值要小于等于对应的diff
3. 与后者的差值要小于等于对应得diff

那么我们可以初始化一个长为n得子序列list（初始化为INT_MAX，这样的话方便我们取小于等于），通过三次遍历，使得这个序列分别满足上面得三个条件

## 实现细节

1. `diff`的长度为`n-1`，注意不要越界访问
2. `list[0]=0`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int findMaxVal(int n, vector<vector<int>>& restrictions, vector<int>& diff) {
        vector<int> list(n,INT_MAX);
        for(auto i:restrictions){
            list[i[0]]=i[1];
        }
        list[0]=0;
        for(int i=1;i<n;++i){
            list[i]=min(list[i],list[i-1]+diff[i-1]);
        }
        int ans=list[n-1];
        for(int i=n-2;i>=0;--i){
            list[i]=min(list[i],list[i+1]+diff[i]);
            ans=max(ans,list[i]);
        }
        return ans;
    }
};
```
