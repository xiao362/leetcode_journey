# 方法一：滑动窗口

## 解题思路和过程

按照题目要求，我们需要**求一个固定长（长度为k）的数组的最大和**，且要求，**数组中的元素各不相同**，这是一个相当经典的**定长滑动窗口思路**，我们可以设置一个所谓的“窗口”，即表示这个定长区间，用**ans不断更新区间和的最大值（因为数据较大可以用longlong来作为数据类型**），同时为了保证数组中的元素各不相同，我们可以**用hashmap来存储数据（查询较快**），然后设置**cnt来记录这个窗口里不同元素的个数**，只有在cnt==k时才能更新ans的值，可以在最后将**已经“满”的窗口的第一个元素给出窗口，这样就给下一个要进的元素腾好位置了**

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$,最坏的情况（k==n）且元素全不相同

## Code

```C++
class Solution {
public:
    long long maximumSubarraySum(vector<int>& nums, int k) {
        int n=nums.size();
        long long ans=0;
        unordered_map<int,int> mp;
        int cnt=0;
        long long sum;
        for(int i=0;i<n;++i){
            sum+=nums[i];
            mp[nums[i]]++;
            if(mp[nums[i]]==1)cnt++;
            if(i<k-1)continue;
            if(cnt==k)ans=max(ans,sum);
            sum-=nums[i-k+1];
            mp[nums[i-k+1]]--;
            if(mp[nums[i-k+1]]==0)cnt--;
        }
        return ans;
    }
};
```
