# 方法一：暴力

## 解题思路和过程

直接用log10统计x的长度n，分奇偶两种情况，用数reserve记反转后的后一半数，x记前一半数，比较两者是否相等即可，**注意n为奇数的话x要多除一次10**，还有可以在开始前把特殊情况排除掉，将x=0和x<0直接处理了

## 复杂度

- 时间复杂度:O(n)
- 空间复杂度:O(1)

## Code

```C++
class Solution {
public:
    bool isPalindrome(int x) {
        if(x<0)return false;
        if(x==0)return true;
        int reverse=0;
        int n=log10(x)+1;
        for(int i=0;i<n/2;++i){
            reverse*=10;
            reverse+=x%10;
            x/=10;
        }
        if(n%2==1)x/=10;
        if(x==reverse)return true;
        return false;
    }
};
```



# 方法二：转化为字符串

这个或许才是暴力，毕竟这个比较好想

## 解题思路和过程

利用string里面带的to_string 直接将x转化为string格式，然后用n记长度，开一个循环，看首尾是否相同就行，这个倒是没有啥需要注意的特殊情况

## Code

```C++
class Solution {
public:
    bool isPalindrome(int x) {
        string s=to_string(x);
        int n=s.size()-1;
        for(int i=0;i<n/2+1;++i){
            if(s[n-i]!=s[i])return false;
        }
        return true;
    }
};
```
