#方法一：数字反转

> 也可以用string的to_string转化成字符串在解题，但是可能要借助long long

##解题思路和过程

题目相当简单，思路也非常的清晰，但代码却不容易实现，因为我们要限制的条件颇为复杂，思路就是将数字反转看他有没有超出范围即可，但题目要求只能用int，所以在反转过程中可能会越界，为此我们只能在未完成反转之前解决掉他，即在第八位已经加上，而第九位还没加上时给他限制条件，将不合适的排除掉

> 负数的取余与正数无异（会保留-号）

> 当ans>INT_MAX/10说明其八位加上任何数都会越界
> ans==INT_MAX && digit>7 这个考虑前八位相等的情况下第九位的数digit是否合适


##复杂度

- 时间复杂度:O(n)
- 空间复杂度:O(1)

##Code

```
class Solution {
public:
    int reverse(int x) {
        int ans=0;     
        while(x){
            int digit=x%10;
            x/=10;
            if(ans==0&&digit==0) continue;
            if(ans>INT_MAX/10||ans==INT_MAX&&digit>7) return 0;
            if(ans<INT_MIN/10||ans==INT_MIN&&digit<-8) return 0;
            ans*=10;
            ans+=digit;
        }
        return ans;
    }
};
```
