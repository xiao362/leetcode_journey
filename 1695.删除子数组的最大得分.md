# 方法一：滑动窗口

这里更适合用hashset来做，因为不涉及个数问题，区间内元素不重复

## 解题思路和过程

通过题意我们要删除一个子数组，这个子数组含有 **若干不同元素** ，且还必须**是连续的**，并要求我们返回这个子数组和的最大值，这也是滑动窗口可以解决的经典问题，我们用双指针维护区间是区间内的元素使得每个元素只有一个，并用res/sum来记录区间元素的和，并且用ans不断更新区间的最大值

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int maximumUniqueSubarray(vector<int>& nums) {
        vector<bool> exist(1e5+1,0);
        int n=nums.size();
        int ans=0;
        int sum=0;
        int left=0;
        for(int i=0;i<n;++i){
            while(exist[nums[i  ]]){
                exist[nums[left]]=0;
                sum-=nums[left++];
            }
            sum+=nums[i];
            exist[nums[i]]=1;
            ans=max(ans,sum);
        }
        return ans;
    }
};
```

# 方法二：bool记忆

因为这道题的数据不大，所以倘若用bool来记忆的话也可以，速度也能快一些

## 解题思路和过程

思路与上面的过程差不多，就是记忆方法改成了bool，倘若你想尽量少些bool的开销可以用下面的方法取得nums中的最大值：

`ranges::max(nums)` 是 **C++20 范围库** 中用来求最大值的函数。

- 它直接接受一个**范围（range）**作为参数，比如 `vector`、`array` 等容器。
- 返回的是范围内所有元素中的最大值（以**值**形式返回）。

如果想省事，直接把数组开到数据的最大值就行

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int maximumUniqueSubarray(vector<int>& nums) {
        int n=nums.size();
        int ans=0;
        int res=0;
        unordered_map<int,int> mp;
        int left=0;
        for(int i=0;i<n;++i){
            mp[nums[i]]++;
            res+=nums[i];
            while(mp[nums[i]]>1){
                res-=nums[left];
                mp[nums[left++]]--;
            }
            ans=max(ans,res);
        }
        return ans;
    }
};
```
