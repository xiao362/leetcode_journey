# 方法一：双指针滚动法

## 解题思路和过程

要想找出两个数组的最小公共值，在两个数组是非降序排列的前提下，我们呢可以设置两个指针同时滚动数组上的元素，将nums1作为做主动指针，在这个过程中我们可能会面对三种情况：

1. 当我们第一次遇到相同的元素，直接返回即可
2. 遇到的`nums2[j]<nums1[i]`，那我们就可以直接滚动`j`的值直到相同或者数组结束亦或者`nums2[j]>nums1[i]`
3. 遇到`nums2[j]>nums1[i]`时，这就需我们变更`i`的值了，条件基本与上面的类似

不保证出界，时刻使num2的值小于num1的值即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int getCommon(vector<int>& nums1, vector<int>& nums2) {
        int n=nums1.size();
        int m=nums2.size();
        int i=0;
        int j=0;
        while(j!=m){
            if(nums2[j]<nums1[i]){
                j++;
            }else if(nums1[i]==nums2[j]){
                return nums1[i];
            }else{
                if(i==n-1)return -1;
                else i++;
            }
        }
        return -1;
    }
};
```

# 方法二：暴力

时间复杂度遇上一个相同，但访问次数最大普遍比其多，但相差不大

## 解题思路和过程

其实用双指针并不是最直接易想的办法，最直接的办法可以是逐个比较，可以设置一个双层循环之一比肩每一个数，当然这个时间复杂度是$O(n^2)$,换个思路可以用hashmap存nums1的数据，然后遍历第二个数组如果其中有hashmap直接返回对应的nums即可，否则就返回-1即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code
