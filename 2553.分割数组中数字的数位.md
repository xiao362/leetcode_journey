# 方法一：暴力

## 解题思路和过程

最直接，不用啥技巧的方法，那就是获取，数的长度然后依次将每个数的第一位放到answer中，最后返回即可

## 复杂度

- 时间复杂度：O($$n$$)
- 空间复杂度：O(n)

## Code

```
class Solution {
public:
    vector<int> separateDigits(vector<int>& nums) {
        vector<int> answer;
        for(auto num:nums){
            int i=num;
            int n=log10(i)+1;
            for(int j=n-1;j>=0;--j){
                answer.push_back(i/pow(10,j));
                i%=(int)pow(10,j);
            }
        }
        return answer;
    }
};
```

这个方法的缺点，浮点数误差可能会导致结果错误（概率较小），效率低下

# 方法二：reverse

## 解题思路和过程

利用c++标准库里面的reverse函数，对每一个数开一个数组，依次存每个数的最后一位，这也是方便我们使用常用的%10/10的操作，不用再调用pow，还要变数，这里说一点知识，C++标准库操作基本遵循：***前开后闭***的原则，所以在调用reverse时搭配begin以及end还是相当好用的

## 复杂度

- 时间复杂度:O(n)
- 空间复杂度:O(n)

## Code

```
class Solution {
public:
    vector<int> separateDigits(vector<int>& nums) {
        vector<int> answer;
        for(int i:nums){
            vector<int> digits;
            while(i){
                digits.push_back(i%10);
                i/=10;
            }
            reverse(digits.begin(),digits.end());
            answer.insert(answer.end(),digits.begin(),digits.end());
        }
        return answer;
    }
};
```

# 方法三：to_string

## 解题思路和过程

这个倒是和回文数的判断差不多，可以直接使用to_string 将数直接转化为string类型，然后以此输入即可，但是要***注意直接输出的类型是char类型要转话成int类型，别忘了-'0'***

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## Code

```
class Solution {
public:
    vector<int> separateDigits(vector<int>& nums) {
        vector<int> ans;
        for(int num:nums){
            string s=to_string(num);
            for(int ch:s){
                ans.push_back((int)ch-'0');
            }

        }
        return ans;
    }
};
```
