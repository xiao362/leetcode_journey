


#方法一：模拟

##解题思路和过程

题目已经将算法给我们了：我们要对他给的算法依次用代码实现，我们可以对他给我们的算法进行分析，方便我们更好的实现代码：

1. 丢弃无用的前导空格（" "）
> 对于前导的空格我们直接用while将此吃掉即可，至于中间出现的空格我们一会儿在分析

2. 查下一个字符（假设还未到字符末尾）为 '-' 还是 '+'。如果两者都不存在，则假定结果为正
> 我们可以定义变量sign来保存结果的符号，sign的值的大小为1或者-1

3. 通过跳过前置零来读取该整数，**直到遇到非数字字符或到达字符串的结尾**。如果没有读取数字，则结果为0。
> 我们定义ans来存储答案，如果ans==0，证明还没遇见过其他数字，并且当前字符ch（s[i]）==0的话就证明现在遇见的就是前置0，就可以跳过去。
**注意：无论是否遇见数字只要遇见字母就返回ans，当已经遇见过数字后或者'-''+'一次后只要遇见，任何非数字字符(包括前面说的空格，及正负号)都要返回ans**

4. 如果整数数超过 32 位有符号整数范围 [−231,  231 − 1] ，需要截断这个整数，使其保持在这个范围内。具体来说，小于 −231 的整数应该被舍入为 −231 ，大于 231 − 1 的整数应该被舍入为 231 − 1 。
> 遇到超出int范围的值直接**返回对应的边界值**
注意：**sign是在最后才返回，在判断时他一直是正数，注意判断超限的条件**

##复杂度

- 时间复杂度:O(n)
- 空间复杂度:O(1)

#Code

```
class Solution {
public:
    int myAtoi(string s) {
        int i=0;
        int n=s.size();

        while(i<n&&s[i]==' '){
            i++;
        }

        int sign=1;
        if(i<n&&(s[i]=='-'||s[i]=='+')){
            sign = s[i]=='-' ? -1:1;
            i++;
        }

        int ans=0;
        while(i<n&&isdigit(s[i])){
            if(ans==0&&s[i]=='0'){
                i++;
                continue;
            }
            if(sign==1&&(ans>INT_MAX/10||(ans==INT_MAX/10&&s[i]>'7'))){
                return INT_MAX;
            } 
            if((sign==-1)&&(ans>INT_MAX/10||(ans==INT_MAX/10&&s[i]>='8'))){
                return INT_MIN;
            }
            ans*=10;
            ans+=s[i]-'0';
            ++i;
        }

        return ans*sign;
    }
};
```
