#方法一：暴力(不补0)

##过程及思路

**分为2种情况考虑**：
1.两个链表长度相同时
2.两个链表长度不同时

- 相同时节点的值为对应两个节点的值相加并加前两个节点进的值
- 长度不同时，先把相同的节点处理了，至于不同的只需要将进的值再加上该节点的值即可
- 注意结果把首元节点跳过去
  
##复杂度

时间复杂度:O(n)
空间复杂度:O(n)

##Code
```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode* list=new ListNode;
        ListNode* cur=new ListNode;
        list=cur;
        int t=0;
        while(l1&&l2){
            int sum=l1->val+l2->val+t;

            t=sum/10;
            cur->next=new ListNode(sum%10);
            cur=cur->next;
            l1=l1->next;
            l2=l2->next;
        }

        ListNode* it=l1?l1:l2;
        while(it){
            int sum=it->val+t;

            t=sum/10;
            cur->next=new ListNode(sum%10);
            cur=cur->next;
            it=it->next;
        }
        if(t){
            cur->next=new ListNode(t);
        }

        return list->next;
    }
};
```

#方法二：补0

##过程及思路

该方法比另一个更好像和实现，不用考虑他们的长度是否相等，只需要在走到不相等的地方时，将已经为空的节点补为0即可

- 同样别忘了跳过首元节点
##复杂度

时间复杂度：O(n)/O(max(n,m))
空间复杂度：O(n)

##Code
```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode* list=new ListNode;
        ListNode* cur=list;
        int t=0;
        while(l1||l2){
                
            int a = l1== nullptr ? 0:l1->val;
            int b = l2== NULL ? 0:l2->val;
            int sum=a+b+t;
            t=sum/10;
            cur->next=new ListNode(sum%10);
            cur=cur->next;

            if(l1){
                l1=l1->next;
            } 
            if(l2){
                l2=l2->next;
            }
        }

        if(t){
            cur->next=new ListNode(t);
        }

        return list->next;
    }
};
```
