# 方法一：前后缀极值分段法（基于贪心）

通过题目上给的条件，我们可以直接看出的条件就是，其所谓的可以交换的条件，就是构成逆序对的时候，如果只是考虑到这里，就直接解题的话，如果我们直接使用暴力解题（用两层循环）的话，我们在解题过程中会发现我们会偏离题意，因为我们漏掉了，他是可以跳跃无数次的，也就是我们的暴力每次实现的只是部分解，为了得到正解我们需要再加一层循环，甚至更多，这样的话时间复杂度就已经来到了惊人的O(n^3)甚至更大，不用想，这肯定是会超时的；

那么我们继续观察，**如果一个数跳到了另一个数身上，他是可以再跳回来的，及这个过程是可逆的，也就是如果a可以跳b，b可以跳c，那么a是可以跳c的，这是具有传递性的**，那么这时我们应该可以想到并查集这个方法，把所有可以相互跳的分到一起，然后再在这一部分找最大值即可（可以将最大值存到根那里，也可以加一个hashmap，减少重复值）代码如下：

```
class Solution {
public:
    vector<int> maxValue(vector<int>& nums) {
        vector<int> ans;
        int sz=nums.size();
        for(auto it:nums){
            ans.push_back(it);
        }
        vector<int> f(sz);
        for(int i=0;i<sz;++i){
            f[i]=i;
        }
        auto find = [&](auto&& self,int x){
            if(f[x]==x)return x;
            else return f[x]=self(self,f[x]);
        };
        auto merge = [&](int a,int b){
            int ra=find(find,a);
            int rb=find(find,b);
            f[ra]=rb;
        };
        unordered_map<int ,int > mp;
        for(int i=0;i<sz;++i){
            for(int j=i+1;j<sz;++j){
                
                    if(nums[j]<nums[i]){
                        merge(j,i);
                        ans[i]=max(ans[i],ans[j]);
                        ans[j]=ans[i];
                    }
                
            }
        }

        for(int i=0;i<sz;++i){
            int ra=find(find,i);
            mp[ra]=max(mp[ra],nums[i]);
        }
        for(int i=0;i<sz;++i){
            int ra=find(find,i);
            ans[i]=mp[ra];
        }
        

        return ans;
    }
};
```

这里我们可以发现，因为每次只能一个数对应一组数的判断，时间复杂度是O(n^2)，提交上去答案倒是对了，时间却是超了，故我们还要在优化

## 解题思路和过程

在以上分析的基础上，以及对数据的观察上，我们可以进一步分析，**假如a可以跳跃到c即(a>c)，那么b就可以跳跃到a和c**：
1. 如果b<a，那么他可以跳跃到a毫无疑问
2. 如果b>a，那么必然b>c，即它可以跳跃到c

因为跳跃是相互的且是可以多次或重复的，那就说明abc是连通的，那么我们就可以得出结论：**如果a和n是连通的，那么他们中间的数必然是连通的**，那么我们只需要找不同连通块的边界即可，通过观察我们可以发现，如果要他们不连通，**需要前面数最大值小后面数的最小值**，这就是我们划分不同连通块的方法

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## Code

```
class Solution {
public:
    vector<int> maxValue(vector<int>& nums) {
        int sz=nums.size();
        vector<int> ans(sz,0);
        int maxval=nums[0];
        vector<int> minval(sz);
        minval[sz-1]=nums[sz-1];
        for(int i=sz-2;i>=0;--i){
            minval[i]=min(minval[i+1],nums[i]);
        }
        for(int i=1;i<sz;++i){
            if(maxval<=minval[i]){
                ans[i-1]=maxval;
                maxval=minval[i];
            }
            if(maxval<nums[i])maxval=nums[i];
        }
        for(int i=sz-1;i>=0;--i){
            if(ans[i]==0)ans[i]=maxval;
            else maxval=ans[i];
        }
        return ans;
    }
};
```



# 方法二：结论+单调栈

## 解题思路和过程

通过前面的分析我们已经得出一个基本的结论：

- *在一个可以相互跳跃的连通块内，他在nums数组上的是连续的*

因为使用并查集的方法维护来连通块的大小的时间复杂度太大，利用上面他在nums数组上连续的性质，我们可以***换一种结构来代替并查集的方法，因为他连续我们可以直接用一个结构体(其中定义left和right，来维护连通块的边界)，再加上value来表示这个连通块的最大值，***不仅是因为输出最后的结果需要我们用到value值，在更新维护连通块的边界时也需要用到value，具体如下：

- **如果a和n是连通的，那么他们中间的数必然是连通的**，那么我们只需要找不同连通块的边界即可，通过观察我们可以发现，如果要他们不连通，**需要前面数最大值小后面数的最小值**，这就是我们划分不同连通块的方法

接下来就是代码实现的具体算法：

用上面的结论对具体数据来分析，假如前面已经有一个连通块，当我们遇到nums[i]时，如果nums[i]<前面连通块的value时，那他是可以进入这个连通块的，只需更新右坐标即可，如果nums[i]>前面连通块的value时，我们就需要再开一个临时连通块，为什么是临时的呢，因为如果下一个nums的值小于前面连通块时，这个临时的连通块也会被合并在上一个连通块中，观察这整个流程我们都只对最后一个连通块进行处理，类似栈，故我们使用栈来存储连通块的信息，这里有***单调栈***的思想

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## Code

```
class Solution {
public:
    struct range{
        int value;  
        int left;
        int right;
    };

    vector<int> maxValue(vector<int>& nums) {
        int sz=nums.size();
        vector<int> ans(sz,0);

        vector<range> stack; //直接用栈也行 

        for(int i=0;i<sz;++i){
            range cur={nums[i],i,i};

            while(!stack.empty()&&stack.back().value>nums[i]){
                range top=stack.back();
                stack.pop_back();
                cur.value=max(cur.value,top.value);
                cur.left=top.left;
            }
            stack.push_back(cur);
        }

        for(auto idx:stack){
            for(int j=idx.left;j<=idx.right;++j){
                ans[j]=idx.value;
            }
        }

        return ans;
    }
};
```
