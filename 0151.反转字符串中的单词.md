# 方法一：双指针

## 解题思路

题目要求我们把单词的循序给反转了，而不反转单词中字母的顺序，所以我们可以用二次反转的办法，先将整个字符串反转，在对每一个单词反转，就这样我们就能得到答案了，其中要注意我们会遇到前导空格，后导空格以及多个来连续空格，为了避开他们的影响我们可以再开一个字符串，用来存储答案

## 实现细节

1. 我们用while控制左边界一直是单词的首字母，右边界一直是单词后的紧接的位置
2. 防止前导0，我们先更新左边界在开始循环
3. 开始后我们更新右边界前先更新右边界使`right=left`，然后逆序存储，存储前根据ans是否为空来判断是否要加“ ”（空格）
4. 更新左边界`left=right`，后用while维护

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$，不算返回值

## Code

```C++
class Solution {
public:
    string reverseWords(string s) {
        reverse(s.begin(),s.end());
        int n=s.size();
        int left=0;
        int right=0;
        string ans="";
        while(s[left]==' ')left++;
        while(left<n){
            right=left;
            while(s[right]!=' '&&right<n)right++;
            if(ans!="")ans+=' ';
            for(int i=right-1;i>=left;--i){
                ans+=s[i];
            }
            left=right;
            while(s[left]==' ')left++;
        }
        return ans;
    }
};
```

# 方法二：双端队列

这个方法有用到move来提升效率（可以不用）

`std::move(word)` 的作用是**将 `word` 转换为右值引用**，从而触发移动语义。在 `d.push_front(move(word))` 中，它让容器选择**移动构造**而非拷贝构造来添加元素，可以高效地转移 `word` 内部的资源（如字符串的字符数据），避免不必要的深拷贝。执行后，`word` 通常变为空字符串或不确定状态，不能再安全使用。这是 C++11 引入的重要优化手段。

## 解题思路

我们以单词为单位把每一个单词依次放进去，先进后出（可以用栈），我们用双端队列

然后用ans存储新的答案，前导和后导0我们在开始时直接给去了（用while），与前面相同我们用双指针控制两端

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$,双端队列的开销

## Code

```C++
class Solution {
public:
    string reverseWords(string s) {
        int left=0;
        int right=s.size()-1;
        while(left<=right&&s[left]==' ')left++;
        while(left<=right&&s[right]==' ')right--;
        deque<string> dq;
        string word="";
        while(left<=right){
            char ch=s[left];
            if(word.size()&&ch==' '){
                dq.push_front(move(word));
                word="";
            }else if(ch!=' '){
                word+=ch;
            }
            ++left;
        }
        dq.push_front(move(word));
        string ans="";
        while(!dq.empty()){
            if(ans!="")ans+=' ';
            ans+=dq.front();
            dq.pop_front();
        }
        return ans;
    }
};
```
