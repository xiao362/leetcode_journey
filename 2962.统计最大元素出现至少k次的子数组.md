# 方法一：滑动窗口

## 解题思路

题目要求我们统计有多少满足 「 `nums` 中的 **最大** 元素」至少出现 `k` 次的子数组，并返回满足这一条件的子数组的数目

因为数组个数较多，所以传统的暴力算法肯定会爆时间，且题目要求我们是子数组得个数，即代表他是连续的，那我们就可以用滑动窗口来解决，我们让窗口处于恰好不满足条件的状态，即`[left,right]`是不满足条件的，但`[left-1,right]`，`[left-1,right]`，`[left-3,right]`......`[0,right]`都是满足条件的，个数也就是left个

## 实现细节

1. 首先我们可以先用`ranges:max(nums)`得到数组得最大数
2. 用`int maxcnt=0`来统计最大值得个数
3. 用while以`maxcnt>=k`来不断维护区间的左边界
4. 用`ans+=left`来计数

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    long long countSubarrays(vector<int>& nums, int k) {
        long long ans=0;
        int n=nums.size();
        int left=0;
        int maxval=ranges::max(nums);
        int maxcnt=0;
        for(int i=0;i<n;++i){
            if(nums[i]==maxval)maxcnt++;
            while(maxcnt>=k){
                if(nums[left++]==maxval)maxcnt--;
            }
            ans+=left;
        }
        return ans;
    }
};
```
