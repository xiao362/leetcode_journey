# 方法一：暴力

## 解题思路和过程

同上面2的幂类似，只需要反复地看他是不是3的倍数即可，然后不断地除以3，最后如果结果等于1，那就返回true,否则就返回false，同样别忘了，负数和0直接返回false

## 复杂度

- 时间复杂度:$O(\log_3n)$
- 空间复杂度:$O(1)$

## Code

```
class Solution {
public:
    bool isPowerOfThree(int n) {
        if(n<=0)return false;
        while(n>1){
            if(n%3!=0)return false;
            n/=3;
        }
        if(n==1)return true;
        return false;
    }
};
```
