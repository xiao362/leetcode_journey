# 方法一：双指针

## 解题思路

我们在进行反转之前，要先对第一个单词的元音字母的个数进行统计，因为

- `s` 仅由小写的英文字母和空格组成。
- `s` 中的单词由 **单个空格** 隔开。
- `s` **不**包含前导或尾随空格。

所以我们就直接用while统计第一个单词中元音字母的个数，并跳过反转第一个单词，接下来我们就让left作为交换区间的左边界（值一直是单词的第一个字母），right维护右边界（值是单词后边的位置），用right遍历一个单词的时候统计该单词中元音字母的个数，如果与第一个相等就逆置这个单词

## 实现细节

1. 用右边界为条件防止越界
2. 逆置时，如果用库函数reverse，右边界是开的，恰好就是right的位置

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string reverseWords(string s) {
        int n = s.size();
        int k = 0;
        int left = 0;
        int right = 0;
        while (right < n && s[right] != ' ') {
            if (s[right] == 'a'|| s[right] == 'e'|| s[right] == 'i'||
                s[right] == 'o'|| s[right] == 'u')
                k++;
            right++;
        }
        int cnt = 0;
        while (right < n) {
            cnt = 0;
            left = right + 1;
            if (s[right] == ' ')
                right++;
            while (right < n && s[right] != ' ') {
                if (s[right] == 'a' || s[right] == 'e' || s[right] == 'i' ||
                    s[right] == 'o' || s[right] == 'u')
                    cnt++;
                right++;
            }
            if (cnt == k)
                reverse(s.begin() + left, s.begin() + right);
        }
        return s;
    }
};
```

我们来逐段拆解这份代码，看看它用到了哪些现代 C++ 特性，以及每一部分具体在做什么。

```cpp
class Solution {
    // 1. 元音字母表
    static constexpr string VOWELS = "aeiou";

    // 2. 统计元音个数的函数模板
    template<ranges::input_range R>
    int count_vowel(const R& s) {
        int vowel = 0;
        for (char c : s) {
            if (VOWELS.contains(c)) {  // C++23 的 string::contains
                vowel++;
            }
        }
        return vowel;
    }

public:
    string reverseWords(string s) {
        int cnt0 = -1;  // 基准元音数

        // 3. 用 views::split 按空格切分，遍历每个单词
        for (auto t : s | views::split(' ')) {
            // 4. 统计当前单词的元音数
            int cnt = count_vowel(t);

            // 5. 决定要不要反转
            if (cnt0 < 0) {
                cnt0 = cnt;                  // 记住第一个单词的元音数
            } else if (cnt == cnt0) {
                ranges::reverse(t);          // 和第一个元音数相同 → 反转
            }
        }
        return s;
    }
};
```

---

### 1. 元音字母表
```cpp
static constexpr string VOWELS = "aeiou";
```
- `static`：属于整个类，所有 `Solution` 对象共享同一份。
- `constexpr`：编译期常量，编译时就能确定值。
- 作用：一个集中的元音集合，供后面的 `contains` 查询。

### 2. 统计元音个数的函数模板
```cpp
template<ranges::input_range R>
int count_vowel(const R& s) {
    int vowel = 0;
    for (char c : s) {
        if (VOWELS.contains(c)) {  // 如果 c 是元音
            vowel++;
        }
    }
    return vowel;
}
```
- `template<ranges::input_range R>`：这是一个**概念约束**，表示 `R` 必须是一个“输入范围”（可遍历的容器或视图）。
- `const R& s`：接受任意满足约束的类型，比如 `string`、`vector<char>`，以及 `views::split` 返回的子范围。
- `VOWELS.contains(c)`：C++23 新增的成员函数，直接判断字符是否在字符串中。等价于以前的 `VOWELS.find(c) != string::npos`，但更简洁可读。
- 功能：返回范围 `s` 中元音字母的个数。

### 3. `views::split` 与管道操作
```cpp
for (auto t : s | views::split(' ')) {
```
这是整段代码最核心的现代 C++ 写法，包含两个关键概念：

- **`views::split(' ')`**：一个**范围适配器**，返回一个**视图**。它不创建新字符串，只是定义了一种“按空格切分”的规则。
- **`|`（管道操作符）**：将左边的数据源 `s` 和右边的适配器 `split` 串接起来，形成一个“惰性流水线”。数据在遍历时才真正被处理，没有额外的内存分配。

循环中的 `t` 是什么？它不是一个 `std::string`，而是一个**视图**（子范围），只包含原字符串中某个单词的起始和结束位置，不拷贝字符。但你可以像遍历容器一样遍历 `t`，所以 `count_vowel(t)` 可以直接统计这个单词内的元音。

### 4. 统计当前单词的元音数
```cpp
int cnt = count_vowel(t);
```
把视图 `t` 直接传给函数模板 `count_vowel`，它遍历这个单词的每个字符，返回元音个数。

### 5. 决定是否反转
```cpp
if (cnt0 < 0) {
    cnt0 = cnt;                  // 第一个单词，记下基准元音数
} else if (cnt == cnt0) {
    ranges::reverse(t);          // 和基准相同 → 反转这个单词
}
```
- 第一个单词永远不反转，只是记下它的元音个数作为基准。
- 后续单词如果元音数与基准相同，就调用 `ranges::reverse(t)` 反转它的字符顺序。
