# 方法一：双指针

```C++
#include <ctype.h> 
int isalnum( int ch );
```

功能：如果参数是数字或字母字符，函数返回非零值，否则返回零值。

## 解题思路

用左右两个指针维护左右边界，用while以`isalsum`为条件，让边界值一直处于字母或者数字的状态，如果时大写字母就转成小写，然后进行比较左右边界的值即可，如果不同直接返回false，否者就继续缩小左右边界的距离，直到二者相遇，返回true（证明满足条件没遇到不相等的字符）

## 实现细节

1. 缩小边界，以及比较时都不能少了条件`left<right`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    bool isPalindrome(string s) {
        int n = s.size();
        int left = 0;
        int right = n - 1;
        while (left < right) {
            while (!isalnum(s[left]) && left < right)
                ++left;
            while (!isalnum(s[right]) && left < right)
                --right;
            if (left >= right)
                return true;
            if (isupper(s[left]))
                s[left] += 32;
            if (isupper(s[right]))
                s[right] += 32;
            if (s[left] != s[right])
                return false;
            ++left;
            --right;
        }
        return true;
    }
};
```
