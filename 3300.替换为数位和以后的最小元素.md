# 方法一：暴力逐位累加

## 解题思路

题目要求我们求出各个位数之和，并找出这些和的最小值，我们只需通过取余操作，把各个位数求出来，并用数来取和，并不断更新最小值即可

## 复杂度

- 时间复杂度:$O(L)$，L是整个数组每个数的和
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int minElement(vector<int>& nums) {
        int n=nums.size();
        int minval=INT_MAX;
        for(int i=0;i<n;++i){
            int ans=0;
            while(nums[i]){
                ans+=nums[i]%10;
                nums[i]/=10;
            }
            minval=min(minval,ans);
        }
        return minval;
    }
};
```
