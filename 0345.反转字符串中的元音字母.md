# 方法一：双指针

contains是C++23引入的库函数

## 解题思路

用左右指针控制可以交换值的位置，即用while保证两端的值都是元音字母，减缓过后同时缩小两端距离，重复上面操作

## 实现细节

1. 可以用contains来判断是否是元音字母
2. 更新两端位置，以及判断是否还能交换的条件都要包含`left<right`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string reverseVowels(string s) {
        int n=s.size();
        int left=0;
        int right=n-1;
        string vowel="aeiouAEIOU";
        while(left<right){
            while(!vowel.contains(s[left])&&left<right)++left;
            while(!vowel.contains(s[right])&&left<right)--right;
            swap(s[left],s[right]);
            left++;
            right--;
        }
        return s;
    }
};
```
