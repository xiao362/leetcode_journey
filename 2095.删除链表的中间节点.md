# 方法一：快慢指针

## 解题思路

题目要求我们删除链表的中间的节点，要删除链表的中间节点毫无疑问我们要会找到链表的中间节点以及他前面的节点，删除操作：我们只需要把前面节点的next改成中间节点后面节点的位置即可，而找中间节点的位置我们可以先遍历一遍链表统计好数组长度后，再次遍历，找到中间节点以及前面节点的位置，然后进行删除操作即可

或者我们用快慢指针搭配pre找中间节点，即慢指针每次走一步，快指针，每次走两步这样就能保证满指针一直在头指针和快指针的中间了，等快指针到终点结束时慢指针的位置就是中间节点的位置，pre更新他前面的位置

## 实现细节

1. 在只有一个节点时，单独处理直接返回`nullptr`
2. 删除操作是`pre->next=slow->next`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    ListNode* deleteMiddle(ListNode* head) {
        if(head->next==nullptr)return nullptr;
        ListNode* pre=head;
        ListNode* slow=head;
        ListNode* fast=head;
        while(fast&&fast->next){
            pre=slow;
            slow=slow->next;
            fast=fast->next->next;
        }
        pre->next=slow->next;
        return head;
    }
};
```

暴力：统计长度

```C++
class Solution {
public:
    ListNode* deleteMiddle(ListNode* head) {
        if(head->next==nullptr)return nullptr;
        int n=0;
        ListNode* middle=head;
        while(middle!=nullptr){
            n++;
            middle=middle->next;
        }
        int cnt=0;
        middle=head;
        ListNode* pre=head;
        while(cnt!=n/2){
            pre=middle;
            middle=middle->next;
            cnt++;
        }
        pre->next=middle->next;
        return head;

    }
};
```
