# 方法一：双指针

其实还是三指针，只是有两个指针同时操作，减少了一层时间复杂度

## 解题思路

题目要求我们统计三数之和恰好等于target的情况，正常来说我们要用三个指针来确认所有的情况，这样的话时间复杂度是$O(n^3)$，会超时，所以我们改换思路，先固定一个值，然后用双指针同时操作另外两个值，这样会减少一层时间复杂度，就能完成任务了

为避免重复，如果固定值是正序遍历的话，双指针就只能在他的右边，不然会有重复坐标

## 实现细节

1. 因为我们统计的是坐标对，所以相同大小的不同坐标的数我们也要考虑到，这是我们面对两种情况(前提是sum==target)：
   - `arr[left]=arr[right]`，这是任意两个数都满足条件，所以我们等差（为1）数列的求和公式来求增加的个数
   - 不相等的情况，左边相等的数（个数cnt_1）以及右边相等的数(个数cnt_2)分别从两端各抽一个数也能组成一对，增加个数为`cnt_1*cnt_2`
2. `sum>target`是右指针左移，反之左指针右移

## 复杂度

- 时间复杂度:$O(n^2)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int threeSumMulti(vector<int>& arr, int target) {
        sort(arr.begin(),arr.end());
        int n=arr.size();
        int ans=0;
        for(int i=0;i<n;++i){
            int x=target-arr[i];
            int left=i+1;
            int right=n-1;
            while(left<right){
                int sum=arr[left]+arr[right];
                if(sum==x){
                    if(arr[left]==arr[right]){
                        int cnt=right-left+1;
                        ans+=cnt*(cnt-1)/2;
                        ans%=(int)1e9+7;
                        break;
                    }
                    int cnt_1=1;
                    int cnt_2=1;
                    left++;
                    right--;
                    while(arr[left]==arr[left-1]&&left<=right){
                        cnt_1++;
                        left++;
                    }
                    while(arr[right]==arr[right+1]&&left<=right){
                        cnt_2++;
                        right--;
                    }
                    ans+=cnt_1*cnt_2;
                    ans%=(int)1e9+7;
                }else if(sum>x){
                    right--;
                }else{
                    left++;
                }
            }
        }
        return ans;
    }
};
```
