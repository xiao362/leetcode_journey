# 方法一：枚举

## 解题思路

找所谓的最高海拔，前缀和的最大值，我们逐一求前缀和，并不断更新最大值即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```c++
class Solution {
public:
    int largestAltitude(vector<int>& gain) {
        int n=gain.size();
        int ans=0;
        int height=0;
        for(auto i:gain){
            height+=i;
            ans=max(ans,height);
        }
        return ans;
    }
};
```
