#方法一：暴力

##解题思路和过程

依照题目我们做容易想到的方法就是，**用三重循环将每个原数组，每个目标数组，跑一遍，将对应的一组词的每个字母都比较一下，统计他们不相同的字母数num，当不同的字母数大于3时，结束该层循环，假若一组词跑完之后，num<=2，就说明该组词符合要求，即可把其添加到ans数组中去**，思路非常简单，且代码容易实现，在此不再赘述

> 注意：为了尽量节省时间以及结果的正确性，不要忘了在num>2时结束最内层循环，以及在找到我们所需要的结果时，结束第二层循环

##复杂度

- 时间复杂度:O(n*m*l)
- 空间复杂度:O(n*l)

##Code

```
class Solution {
public:
    vector<string> twoEditWords(vector<string>& queries, vector<string>& dictionary) {
        vector<string> ans;
        int n=queries.size();
        int n1=dictionary.size();
        int m=queries[0].length();
        for(int i=0;i<n;++i){
            for(int j=0;j<n1;++j){
                int num=0;
                for(int k=0;k<m;++k){
                    if(queries[i][k]!=dictionary[j][k]){
                        num++;
                        if(num>2)break;
                    }
                }
                if(num<=2){
                    ans.push_back(queries[i]);
                    break;
                }
            }
        }

        return ans;
    }
};
```


#方法二：字典树(trie)+bfs

> 不得不提，因为该题的数据较小，所以这个方法，时间还不及暴力方法，但当数据较大时，该方法会较快，并且这个方法还适用于长度不同的字典查询


##解题思路及过程

在暴力枚举的方法中我们可以发现，我们的有些查询时可以省略的，比如[cat,car]，其中ca这两个字母我们发现，这两次查询完全是多余的吗，那么我们有什么方法可以**跳过这两次多余的查询**吗，换种说法，**即是在查询过ca两个字母我们同时匹配到car以及cat**呢，依据这种想法我们可以以图的方式表示出来
```
        root (空)
       /        \
      c          d
     /            \
    a              o
   /  \              \
  t    r              g
 (终点)  (终点)     (终点)
```
这里我们就可以看出这种结构完全是棵树嘛，故引入**字典树(trie)**的概念，**简单的理解就是把字典变成一颗树，一个字母为一个节点**，到此数据的储存结构已经明晰（树），同时又因为**树的遍历往往离不开递归**，那么我们的查询方法，也基本显露：即用**bfs遍历查询，并判断每个词是否符合条件**

> 因为字母只有26个，且不重复，故我们可以直接设置26个子节点，如果输入的字典中不是对应的字母，即将其设置为nullptr，同时不要忘记，设置一个词的终止条件(isEnd)，或者得到词的长度，用index是否等于length来判断是否终止(这题特殊单词的长度相同)

##复杂度

- 时间复杂度:O((k⋅n+q⋅n^2*25^2)
- 空间复杂度:O(kn)


##Code

```
struct trieNode{
    trieNode* children[26];//用数表示字母
    bool isEnd;//判断结束
    trieNode(){
        for(int i=0;i<26;++i){
            children[i]=nullptr;
        }
        isEnd=0;
    }
};

class Solution {
public:
    void insert(trieNode* root,string& word){
        trieNode *node=root;
        for(auto i:word){
            int n=i-'a';
            if(!node->children[n]) node->children[n]=new trieNode();
            node=node->children[n];
        }
        node->isEnd=1;
    }

    bool dfs(trieNode* node,string& word,int pos,int diff){
        if(diff>2)return 0;//剪枝
        if(pos==word.size())return node->isEnd;
        int idx=word[pos]-'a';
        
        if(node->children[idx]){
            if(dfs(node->children[idx],word,pos+1,diff)) return 1;
        }
        for(int i=0;i<26;++i){
            if(idx!=i&&node->children[i]){
                if(dfs(node->children[i],word,pos+1,diff+1)) return 1;
            }
        }

        return 0;
    }

    vector<string> twoEditWords(vector<string>& queries, vector<string>& dictionary) {
        vector<string> ans;
        trieNode* root=new trieNode();
        for(auto i:dictionary){
            insert(root,i);
        }
        for(auto i :queries){
            if(dfs(root,i,0,0)){
                ans.push_back(i);
            }
        }
        return ans;
    }
};
```
