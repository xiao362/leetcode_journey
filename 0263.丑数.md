# 方法一

## 解题思路和过程

同2，3的幂，相似这里用while循环只要是2，3，5的倍数就然他分别处以2，3，5。如果他只有这三个质因数，那他最后就会变成1，所以只需要返回n==1的结果即可

## 复杂度

- 时间复杂度:$O(logn)$
- 空间复杂度:$O(1)$

## Code

```c++
class Solution {
public:
    bool isUgly(int n) {
        if(n<=0)return false;
        while(n%2==0){
            n/=2;
        }
        while(n%3==0){
            n/=3;
        }
        while(n%5==0){
            n/=5;
        }
        return n==1;
    }
};
```
