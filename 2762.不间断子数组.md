# 方法一：滑动窗口+单调队列

单调队列会用到双向队列deque

```
deque<int> q; // 存储下标，方便判断过期
for (int i = 0; i < n; ++i) {
    // 1. 移除过期元素（窗口左边界之外的元素）
    while (!q.empty() && q.front() <= i - k) q.pop_front();
    // 2. 维护单调性：从队尾踢掉所有比当前元素小的
while (!q.empty() && nums[q.back()] < nums[i]) q.pop_back();

// 3. 当前元素入队
q.push_back(i);

// 4. 队头就是当前窗口最大值
if (i >= k - 1) ans.push_back(nums[q.front()]);
}
```

## 解题思路

要想使 `i <= i1, i2 <= j` 的下标对，都有 `0 <= |nums[i1] - nums[i2]| <= 2` ，	我们就需要不断维护某一区间的最大以及最小值，通过二者的差值来判断我们是否要压缩区间长度

> 注意：我们维护时，并不单单考虑一个方向（右边），我们还要考虑左边，并且元素可以重复，所以在维护区间极值的时候，计数就必不可少，毕竟是一段区间的极值

维护某一区间的极值我们可以用单调队列的办法（两个）

**单调队列的核心思想只有八个字：剔除无用，维持单调。**

它就像一个“汰弱留强”的排队系统：新来的人更年轻、更强，那就把前面老弱的人直接赶走，这样队头永远是当前最强者。

另外关于计数，倘若一个数组满足条件，那他的所有子数组都会满足条件

## 实现细节

1. 用两个单调队列分别维护区间最大以及最小值
2. 为了方便区间更新，队列里面可以存坐标
3. ans的更新时每次新增加的子数组，其个数为`i-left+1`;

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    long long continuousSubarrays(vector<int>& nums) {
        long long ans=0;
        int n=nums.size();
        deque<int> dq_min;
        deque<int> dq_max;
        int left=0;
        for(int i=0;i<n;++i){
            while(!dq_min.empty()&&nums[i]<=nums[dq_min.back()]){
                dq_min.pop_back();
            }
            dq_min.push_back(i);
            while(!dq_max.empty()&&nums[i]>=nums[dq_max.back()]){
                dq_max.pop_back();
            }
            dq_max.push_back(i);
            while(nums[dq_max.front()]-nums[dq_min.front()]>2){
                left++;
                while(dq_max.front()<left){
                    dq_max.pop_front();
                }
                while(dq_min.front()<left){
                    dq_min.pop_front();
                }
            }
            ans+=i-left+1;
        }
        return ans;
    }
};
```

# 方法二：滑动窗口+map/multiset

## 解题思路

就像前面所说 我们既要能存储窗口内元素的个数，还要能将元素排序，那我们就可以直接用map来存储数据，map本身排序的性质可以让我们直接得到区间的最小或最大值，并且还可以通过键值对的值来存相应元素的个数

确定存储数据的容器后，我们依旧按照上面最大值与最小值的差与2的大小关系来维护区间的左边界

## 实现细节

1. `map<int,int> mp`来存储区间的元素
2. 用while搭配left来维护左边界
3. `mp.rbegin()，mp.begin()`返回的是迭代器，要用-first来访问键
4. 每次区间改变，ans增加新增的子数组个数`i-left+1`
5. 要用erase删除，不然（r）begin（）还返回他

## Code

```c++
class Solution {
public:
    long long continuousSubarrays(vector<int>& nums) {
        int n=nums.size();
        map<int,int> mp;
        long long ans=0;
        int left=0;
        for(int i=0;i<n;++i){
            mp[nums[i]]++;
            while(mp.rbegin()->first-mp.begin()->first>2){
                mp[nums[left]]--;
                if(mp[nums[left]]==0)mp.erase(nums[left]);
                left++;
            }
            ans+=i-left+1;
        }
        return ans;
    }
};
```

下面是multiset的实现，思路大致相同，细节也差别不大

> 注意：按值删除元素要用find找迭代器，不然他给相同值的元素全删了

```C++
class Solution {
public:
    long long continuousSubarrays(vector<int>& nums) {
        long long ans=0;
        int n=nums.size();
        int left=0;
        multiset<int> set;
        for(int i=0;i<n;++i){
            set.emplace(nums[i]);
            while(*set.rbegin()-*set.begin()>2){
                set.erase(set.find(nums[left]));
                left++;
            }
            ans+=i-left+1;
        }
        return ans;
    }
};
```
