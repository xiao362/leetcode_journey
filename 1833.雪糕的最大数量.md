# 方法一：快排

## 解题思路

为了使购买尽可能多的雪糕，我们优先购买单价便宜的雪糕，所以我们可以直接通过快排排好序之后，从较小单价开始购买直到买完或者钱不够在停止

## 复杂度

- 时间复杂度:$O(n\log n)$，主要是排序消耗
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int maxIceCream(vector<int>& costs, int coins) {
        sort(costs.begin(),costs.end());
        int ans=0;
        for(int i:costs){
            if(coins-i<0)return ans;
            coins-=i; 
            ans++;
        }
        return ans;
    }
};
```

# 方法二：计数排序

这个方法适用最大值较小，且有较多重复数据

## 解题思路

先统计出单价的最大值，然后找出每个单价的个数,用一个数组记录（空间换时间），然后购买就行了，结束条件相同

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int maxIceCream(vector<int>& costs, int coins) {
        int maxval=ranges::max(costs);
        vector<int> count(maxval+1,0);
        for(int i=0;i<costs.size();++i){
            count[costs[i]]++;
        }
        int ans=0;
        for(int i=1;i<=maxval&&coins>0;++i){
            int num=min(coins/i,count[i]);
            ans+=num;
            coins-=num*i;
        }
        return ans;
    }
};
```
