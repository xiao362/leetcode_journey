# 方法一：

## 解题思路和过程

按照他的要求，模拟一边这个过程即可，开一个n-1次的循环，ans初始化为start，然后不断让ans^(start+2*i);

## 复杂度 

- 时间复杂度:O($$1$$)
- 空间复杂度:O($$1$$)

## Code

```
class Solution {
public:
    int xorOperation(int n, int start) {
        int ans =start;
        for(int i=1;i<n;++i){
            if(i!=0)ans=ans^(start+2*i);
        }
        return ans;
    }
};
```
