# 方法一：暴力

直接用reverse函数，直接反转就解决

## 解题思路

手动反转，用一个字符ch在交换时使用，然后从0到n/2交换即可，即交换`s[i]`与`s[n-i-0]`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    void reverseString(vector<char>& s) {
        char ch='\0';
        int n=s.size();
        for(int i=0;i<n>>1;++i){
            ch=s[i];
            s[i]=s[n-1-i];
            s[n-1-i]=ch;
        }
    }
};
```

上面的交换其实与双指针思路相同，只是一个指针的位置可以通过另一个指针的位置来反映罢了

下面是双指针的代码

```C++
class Solution {
public:
    void reverseString(vector<char>& s) {
        int n=s.size();
        int left=0;
        int right=n-1;
        char ch='\0';
        while(left<right){
            ch=s[left];
            s[left]=s[right];
            s[right]=ch;
            left++;
            right--;
        }
    }
};
```
