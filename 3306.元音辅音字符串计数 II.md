# 方法一：滑动窗口

## 解题思路

题目要求我们统计一个字符串内，每个元音字母（`'a'`、`'e'`、`'i'`、`'o'`、`'u'`）**至少** 出现一次，并且 **恰好** 包含 `k` 个辅音字母的子字符串的总数。

因为研究对象是字符串，且条件容易量化我们用滑动窗口来解决这个问题，但需要注意，常规的滑动窗口：

1. 比较适合解决大于等于的问题（条件是多条件并列的情况）
2. 比较适合解决恰好等于的问题（条件是一种的）

这里我们我们既要要求元音字母的个数又要求辅音字母的个数，所以我们要换种思路来解决这个问题，我们可以**通过第一个办法**，来求出**辅音字母大于等于k的个数以及大于等于k+1的个数**，利用**二者的差来求解恰好包含k个辅音字母**的子字符串数目

想要减少代码量就把求辅音字母大于等于k的个数（别忘了满足包含全部元音字母的条件）的功能写成函数

## 实现细节

1. 因为只会出现英文小写字母故可以用数组来统计元音字母出现的个数（在区间内），这一我用的是哈希表
2. 因为要同时满足包含五个元音字母，所以我们把这五个条件写成checked，搭配数组或者哈希表来使用，`checked==5`时满足条件
3. 统计大于等于k个辅音字母的个数的改变条件是`ans+=left`

> 注意：我们已经把问题转化为求大于等于k个辅音字母了，所以while的条件就是`num>=k&&checked==5`，num表示辅音字母的个数

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(|\Sigma|)$,$|\Sigma|=26$是小写英文字母的个数

## Code

> 换一种写法：可以用数组来存储区间内各个元音字母的个数
>
> 或者：不写另外写函数，那样的话通过一次遍历就能解出答案，时间可能会快些
>
> 上面是另外两种写法的简介，在这个基础上改动不大就不再赘述

```C++
class Solution {
public:
    long long solve(string word, int k){
        int n=word.size();
        int num=0;
        int checked=0;
        unordered_map<int,int> mp;
        int left=0;
        long long ans=0;
        for(int i=0;i<n;++i){
            if(word[i]=='a'||word[i]=='e'||word[i]=='i'||word[i]=='o'||word[i]=='u'){
                mp[word[i]]++;
                if(mp[word[i]]==1)checked++;
            }else{
                num++;
            }
            while(num>=k&&checked==5){
                if(word[left]=='a'||word[left]=='e'||word[left]=='i'||word[left]=='o'||word[left]=='u'){
                    mp[word[left]]--;
                    if(mp[word[left]]==0)checked--;
                }else{
                    num--;
                }
                left++;
                
            }
            ans+=left;
        }
        return ans;
    }

    long long countOfSubstrings(string word, int k) {
        return solve(word,k)-solve(word,k+1);
    }
};
```
