# 方法一：滑动窗口

## 解题思路和过程

观察数据以及题目给出的要，我们可以知道我们的目的是找到这个区间，区间内最多只有k个T或者k个F，这里可以有两种写法，**一种是分开统计（用两个区间分别更新区间内的TF的数量，不让数量超过k，右边界相同**），**另一种是把两个合在一起实现，就是当区间内的TF值同时大于k时变更指针**

两种方法各有优缺:

- 第一个方法好想，不容易实现（较第二个），容易搞混数据的含义
- 第二个方法不好想，但容易实现对标经典的不定长滑动窗口

先说第一个方法的实现，因为第二算法的核心已经提出，实现并不难

- 用**tleft维护区间内T的个数，不让他超过k个（对应把F变成T**），t记录T的个数，用**while以t>k为条件answerKey[ tleft++ ]=='T'的时候t--；**

- 反之，**用fleft维护区间内F的个数，不让他超过k个**（对应把T变成F），f记录F的个数，用**while以f>k为条件answerKey[ fleft++ ]=='F'的时候T--；**

然后用ans维护最大值即可

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int maxConsecutiveAnswers(string answerKey, int k) {
        int ans=0;
        int tleft=0,t=0;
        int fleft=0,f=0;
        for(int i=0;i<answerKey.size();++i){
            if(answerKey[i]=='T')t++;
            else f++;
            while(t>k){
                if(answerKey[tleft++]=='T')t--;
            }
            while(f>k){
                if(answerKey[fleft++]=='F')f--;
            }
            ans=max(ans,i-tleft+1);
            ans=max(ans,i-fleft+1);
        }
        return ans;
    }
};
```

第二种写法：

**T的ascall对应84，F对应70，二者除以2后的奇偶性不同，说明他，他们二进制的次低位不同（0，1），可以用来计数（用cnt数组时）**

上面计数的方法用两个量分别计数就行，上面是花哨写法

这个算法的关键是：**当区间内的T，F值同时大于k时变更指针**

实现如下：

```C++
class Solution {
public:
    int maxConsecutiveAnswers(string answerKey, int k) {
        int ans=0;
        int left=0;
        int cnt[2]={};
        for(int i=0;i<answerKey.size();++i){
            cnt[answerKey[i]>>1 & 1]++;
            while(cnt[0]>k&&cnt[1]>k){
                cnt[answerKey[left++]>>1 & 1]--;
            }
            ans=max(ans,i-left+1);
        }
        return ans;
    }
};
```
