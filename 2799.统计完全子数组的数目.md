# 方法一：滑动窗口

## 解题思路

题目要求我们统计**不同** 元素的数目等于整个数组不同元素的数目的子数组的个数，由于我们研究的对象是子数组，且要求我们的条件可以量化，那我们就可以用滑动窗口来解决

我们让**窗口处于恰好不满足条件的状态**，即`[left,right]`是不满足条件的，但`[left-1,right]`，`[left-1,right]`，`[left-3,right]`......`[0,right]`都是满足条件的，**个数也就是left个**

## 实现细节

1. 在滑动窗口之前我们需要**先统计数组的元素种数**
   - 手动遍历一遍数组，统计出元素种数
   - 或者，使用`unordered_set st(nums.begin(),nums.end())`直接插入数组元素，然后用size()得到种数
2. 在**维护区间左边界**时用while搭配`cnt（窗口内元素种数）==st.size()`为条件
3. **统计个数**用`ans+=left`

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int countCompleteSubarrays(vector<int>& nums) {
        unordered_set st(nums.begin(),nums.end());
        int n=nums.size();
        int num=st.size();
        int ans=0;
        int left=0;
        unordered_map<int,int> mp;
        int cnt=0;
        for(int i=0;i<n;++i){
            mp[nums[i]]++;
            if(mp[nums[i]]==1)cnt++;
            while(cnt==num){
                mp[nums[left]]--;
                if(mp[nums[left]]==0)cnt--;
                left++;
            }
            ans+=left;
        }
        return ans;
    }
};
```
