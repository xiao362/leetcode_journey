# 方法一：暴力

## 解题思路

题目要求我们求出一个数组左边元素和与右边元素和的差值的绝对值，我们先进行整个数组求和，这样就方便我们通过左便元素的和来计算右边元素的和，然后第二次遍历时不断滚动左边元素的和即可，这样就可计算返回对应的差值绝对值了

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    vector<int> leftRightDifference(vector<int>& nums) {
        int sum=0;
        for(int i=0;i<nums.size();++i){
            sum+=nums[i];
        }
        int leftSum=0;
        vector<int> ans(nums.size(),0);
        for(int i=0;i<nums.size();++i){
            int rightSum=sum-nums[i]-leftSum;
            ans[i]=abs(leftSum-rightSum);
            leftSum+=nums[i];
        }
        return ans;
    }
};
```

```C++
class Solution {
public:
    vector<int> leftRightDifference(vector<int>& nums) {
        int sum=reduce(nums.begin(),nums.end());
        int leftSum=0;
        for(auto& i:nums){
            leftSum+=i;
            i=abs(2*leftSum-sum-i);
        }
        return nums;
    }
};
```
