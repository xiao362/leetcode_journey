# 方法一：滑动窗口

## 解题思路

题目要求我们，一个数组的元素和与数组长度的乘积小于k的子数组的个数，因为子数组保持相对的连续，我们可以开一个窗口（区间）是用双指针来维护窗口内的分数始终小于k

另外在统计数组个数是，我们要明白倘若一个数组，满足了上述要求，那么他的所有子数组一定满足要求，另外因为倘如我们每次在某一区间结束时在变更长度，虽然可以通过数学公式，来一次计算出区间内所有子数组的个数，但是结束时的条件略显麻烦，故我们可以通过增加每次区间改变时子数组增加的来计算

## 实现细节

1. 数据尽量使用 long long，因为涉及和与积，int容易爆
2. 每次数组改变时，数组长度增加`ans+=i-left+1`

## Code

```C++
class Solution {
public:
    long long countSubarrays(vector<int>& nums, long long k) {
        long long ans=0;
        int n=nums.size();
        long long sum=0;
        long long multiply=1;
        int left=0;
        for(int i=0;i<n;++i){
            sum+=nums[i];
            multiply=sum*(i-left+1);
            while(multiply>=k){
                sum-=nums[left++];
                multiply=sum*(i-left+1);
            }
            ans+=i-left+1;
        }
        return ans;
    }
};
```
