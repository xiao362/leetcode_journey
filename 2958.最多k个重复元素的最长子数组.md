# 方法一：滑动窗口

## 解题思路和过程

由给出的定义：

- 如果一个数组中所有元素的频率都 **小于等于** `k` ，那么我们称这个数组是 **好** 数组。
- **子数组** 指的是一个数组中一段连续非空的元素序列。

题目让我们求**最长好子数组**的最大长度，那就意味着我们要找到一个在nums上连续的区间，区间内元素的个数都小于等于k，这是经典的不定长滑动窗口问题，我们用可以双指针来为区间的长度，使区间内的元素满足要求即可，具体实现如下：

我们可以用hashmap来统计元素出现的次数，只要`mp[nums[i]]>k`我们就不断缩短区间（left++），知道满足要求则停止，不断用ans来更新最大值

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int maxSubarrayLength(vector<int>& nums, int k) {
        int n=nums.size();
        int ans=0;
        int left=0;
        unordered_map<int,int> mp;
        for(int i=0;i<n;++i){
            mp[nums[i]]++;
            while(mp[nums[i]]>k){
                mp[nums[left++]]--;
            }
            ans=max(ans,i-left+1);
        }
        return ans; 
    }
};
```
