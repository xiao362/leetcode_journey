# 方法一：双指针

## 解题思路

直接对原字符串进行操作，我们只需要对反转0到k之间的字符即可，有因为一个指针是可以用另一个指针的坐标得出所以也可以用单指针，或者直接用reverse函数

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string reversePrefix(string& s, int k) {
        char ch='\0';
        int right=k-1;
        for(int i=0;i<k/2;++i){
            ch=s[i];
            s[i]=s[right];
            s[right]=ch;
            --right;
        }
        return s;
    }
};
```

单指针

```C++
class Solution {
public:
    string reversePrefix(string& s, int k) {
        char ch='\0';
        
        for(int i=0;i<k/2;++i){
            ch=s[i];
            s[i]=s[k-1-i];
            s[k-1-i]=ch;
        }
        return s;
    }
};
```

库函数

```C++
class Solution {
public:
    string reversePrefix(string& s, int k) {
        reverse(s.begin(),s.begin()+k);
        return s;
    }
};
```
