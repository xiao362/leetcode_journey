# 方法一：暴力

## 解题思路

开一个与原数组长度相同的ans数组作为返回数组，题目要求我们在保持基本顺序的前提下，把数组内元素分成三部分：

1. 左边部分小于pivot
2. 中间等于pivot
3. 右边小于pivot

我们只需要用三个循环依次筛选出这三部分并放在ans的后面即可，既能保证顺序又能筛选出三部分

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    vector<int> pivotArray(vector<int>& nums, int pivot) {
        int n=nums.size();
        vector<int> ans;
        for(int i=0;i<n;++i){
            if(nums[i]<pivot){
                ans.push_back(nums[i]);
            }
        }
        for(int i=0;i<n;++i){
            if(nums[i]==pivot){
                ans.push_back(nums[i]);
            }
        }
        for(int i=0;i<n;++i){
            if(nums[i]>pivot){
                ans.push_back(nums[i]);
            }
        }
        return ans;
    }
};
```

另一种写法，就是开三个数组，分别存三个部分，依次循环就可以结束，结束后将三个数组拼接到一起即可

用vector的insert来插入整个数组

```C++
class Solution {
public:
    vector<int> pivotArray(vector<int>& nums, int pivot) {
        int n=nums.size();
        vector<int> a,b,c;
        for(int i=0;i<n;++i){
            if(nums[i]<pivot){
                a.push_back(nums[i]);
            }else if(nums[i]==pivot){
                b.push_back(nums[i]);
            }else{
                c.push_back(nums[i]);
            }
        }
        a.insert(a.end(),b.begin(),b.end());
        a.insert(a.end(),c.begin(),c.end());
        return a;
    }
}
```

# 方法二：双指针

## 解题思路

数组长度是一定的，且返回值得中间一部分都是与pivot相等得值，我们可以在初始化时直接把pivot作为初始值，只筛选大于和小于的部分即可，我们可以用双指针分别控制两端，有小于得就放左边，大于的就放右边，中间的已经初始化好了，不用管

> 在这个工程中，因为右边界是从右往左收缩的所以大于pivot的部分是相对逆序的，我们还需要在用reverse改变一下顺序

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$(n)$

## Code

```C++
class Solution {
public:
    vector<int> pivotArray(vector<int>& nums, int pivot) {
        int n=nums.size();
        int left=0;
        int right=n-1;
        vector<int> ans(n,pivot);
        for(int i=0;i<n;++i){
            if(nums[i]<pivot){
                ans[left++]=nums[i];
            }else if(nums[i]>pivot){
                ans[right--]=nums[i];
            }
        }
        reverse(ans.begin()+right+1,ans.end());
        return ans;
    }
};
```
