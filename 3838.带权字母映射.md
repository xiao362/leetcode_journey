# 方法一：暴力

## 解题思路

思路并不难可以直接按照他给的思路模拟实现过程，以其中一个字符串为例，我们对这个字符串中的每一个元素所对应的权值求和，得到权值总和sum，与26求余后，反向映射`a`到`z`即可，所得到的字母就是该字符串对应的字符，遍历完所有字符串就是答案

## 实现细节

1. 注意映射坐标的时候用的是`sum=25-sum+'a'`不要用26，坐标对应从0到25

## 复杂度

- 时间复杂度:$O(n)$，严谨的说n是所有字符串的字符个数
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string mapWordWeights(vector<string>& words, vector<int>& weights) {
        int n=words.size();
        string ans="";
        for(int i=0;i<n;++i){
            int sum=0;
            for(int it:words[i]){
                sum+=weights[it-'a'];
            }
            sum%=26;
            sum=25-sum;
            ans+='a'+sum;
        }
        return ans;
    }
};
```
