# 方法一：双指针

## 解题思路

我们要反转每一个单词，所以我们只需要用双指针来找要更新的区间就行了，而找区间边界我们只需要用while，不断维护有边界即可，如果遇到空格是就代表前面的字母结束了，我们可以把前面的反转了，反转后左边界就变成如今右边界的位置

## 实现细节

1. 如果用双指针手动交换，要注意在区间的中点结束不要重复交换

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string reverseWords(string s) {
        int left=0;
        int right=0;
        int n=s.size();
        while(left<n){
            while(s[right]!=' '&&right<n)right++;
            for(int i=0;i<(right-left)/2;++i){
                swap(s[left+i],s[right-i-1]);
            }
            left=right+1;
            right++;
        }
        return s;
    }
};
```

reverse写法

```C++
class Solution {
public:
    string reverseWords(string s) {
        int left=0;
        int right=0;
        int n=s.size();
        while(left<n){
            while(s[right]!=' '&&right<n)right++;
            reverse(s.begin()+left,s.begin()+right);
            left=right+1;
            right++;
        }
        return s;
    }
};
```
