#方法一：暴力

##解题思路及算法

要求一个数为对应的target值，最简单也是最容易想到的方法也就是暴力枚举，用两层循环跑一下
##复杂度

-时间复杂度O(n^2)
-空间复杂度O(n)
##Code

```
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        int n=nums.size();
        for(int i=0;i<n;++i){
            for(int j=i+1;j<n;++j){
                if(nums[i]+nums[j]==target){
                    return {i,j};
                }
            }
        }
        return {};
    }
    
};
```

#方法二：哈希表

##解题思路及算法

利用HashMap的不重复，查找快的特点，可快速查找另一个值是否存在，进而判断这个数，是否能找到另一个数，使得其和为target

##复杂度

-时间复杂度O(n)
-空间复杂度O(n)

##Code
```
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        int n=nums.size();
        unordered_map<int,int> mp;
        for(int i=0;i<n;++i){
            int want=target-nums[i];
            if(mp.count(want)){
                return {i,mp[want]};
            }
            mp.emplace(nums[i],i);
        }
        return {};     
    }
    
};
```
