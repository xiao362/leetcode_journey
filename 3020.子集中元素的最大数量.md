# 方法一：遍历dfs

## 解题思路

要想形成一个满足条件的子集，我们可以通过找某个值的平方数是否存在以及个数是否大于等于2来判断，如果大于等于2，对应的cnt就加2，否者直接返回当前的cnt就行

> 注意：因为最少形成一个，所以cnt初始值就设为1，查找判断的过程可以通过将元素排序后利用二分的方法来实现，然后不断的以每个位置为起点查询，找到最大值即可

## 实现细节

1. 要对`1`单独处理，因为他的平方数等于他本身
2. 返回值的个数要为奇数，所以再返回值为偶数时-1后在返回
3. 求平方时为防止溢出要用 `long long`

## 复杂度

- 时间复杂度:$O(n \log n)$，排序和查找的消耗
- 空间复杂度:$O(1)$

## Code

> 思路与上面一致，这里顺序上略有差别，先判断当前位置的数存在两个及以上，再保证平方数存在后再将cnt（函数返回值）+2

```C++
class Solution {
public:
    int maximumLength(vector<int>& nums) {
        ranges::sort(nums);
        int n=nums.size();
        int ans=1;
        auto dfs=[&](auto& dfs, int idx,int cnt){
            if(idx>=n-1||nums[idx+1]!=nums[idx])return cnt;
            if(nums[idx]==1){
                int right=lower_bound(nums.begin(),nums.end(),2)-nums.begin();
                return (right-idx)%2==0 ? right-idx-1:right-idx;
            }
            int right=lower_bound(nums.begin(),nums.end(),(long long)nums[idx]*nums[idx])-nums.begin();
            if(right==n)return cnt;
            if(nums[idx]*nums[idx]==nums[right])return dfs(dfs,right,cnt+2);
            return cnt;
        };
        for(int i=0;i<n;++i){
            ans=max(ans,dfs(dfs,i,1));
        }
        return ans;
    }
};
```

# 方法二：暴力枚举

## 解题思路

与上面的判断思路大致相同，但是这里我们用hashmap来存储数据的个数，机构化绑定的办法遍历hashmap的每个值

与上面不同的是，初始化为0，遇到等于平方值（一开始是本身），且个数大于等于2的话个数就加2，直到遇到不包含平方值或者个数不足2为止（这时一定是偶数），最后是否存在平方值来加1（存在）减1（）不存在

## 实现细节

1. 对值为1单独处理，且处理后要在hashmap里删除1，不然会死循环
2. 不断更新最大值

## 复杂度

- 时间复杂度:$O(n \log n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int maximumLength(vector<int>& nums) {
        unordered_map<long long,int> mp;
        for(auto i:nums){
            mp[i]++;
        }
        int ans=(mp[1]-1)|1;
        mp.erase(1);
        for(auto& [i,_]:mp){
            int res=0;
            long long x=i;
            while(mp.contains(x)&&mp[x]>=2){
                res+=2;
                x*=x;
            }
            ans=max(ans,res+ (mp.contains(x)? 1:-1));
        }
        return ans;
    }
};
```
