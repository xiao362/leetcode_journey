# 方法一：

## 解题思路和过程

直接用foreach循环，遍历s中的每一个字母，对于是大写字母的直接+=32，变成小写字母，关于32的差值可以查ascall码表

## 复杂度 

- 时间复杂度:O($$n$$)
- 空间复杂度:O($$1$$)

## Code

```
class Solution {
public:
    string toLowerCase(string s) {
        for(char& ch:s){
            if(ch>='A'&&ch<='Z')ch+=32;
        }
        return s;
    }
};
```
