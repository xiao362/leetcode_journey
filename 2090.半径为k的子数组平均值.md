# 方法一：滑动窗口

## 解题思路和过程

通过题意以及给出的数据我们可以知道其让我们求得依旧是一段定长区间内的平均值，不过这个长并没有直接给出，还需要间接换算一下，即**len=k*2+1**，在有了区间长以后，我们可以设置一个所谓的“窗口”（表示定长区间），（毕竟这毫无疑问是是一道**滑动窗口**的题），给“窗口”设置性质sum，表示窗口内各位数的和，然后设置a**ve数组存放我们的结果**，因为要求在**长度没达到的位置的平均值记为-1，故我们直接初始化为-1即可，然后遍历数组时将达到长度的区间的平均值存里面即可**

注意：要按照定长滑动窗口的思想看这个区间，不要把区间的中点看的太重要，还有一点就是**数据的范围，因为数据可能来到$10^5$,所以要用long long 来存sum，不然可能会爆数据**

 ## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$,n是返回值时用的数组

## Code

```C++
class Solution {
public:
    vector<int> getAverages(vector<int>& nums, int k) {
        int n=nums.size();
        vector<int> ave(n,-1);
        long long sum=0;
        int l=2*k+1;
        for(int i=0;i<n;++i){
            sum+=nums[i];
            if(i<l-1)continue;
            ave[i-k]=sum/l;
            sum-=nums[i-l+1];
        }
        return ave;
    }
};
```
