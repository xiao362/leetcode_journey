# 方法一：双指针

## 解题思路

要找中位数和目标值，我们肯定要先对数组排序，排序后通过坐标找到中位数，然后依据条件

判定 `arr[i]` 的值比 `arr[j]` 的值更强：

-  `|arr[i] - m| > |arr[j] - m|`
-  `|arr[i] - m| == |arr[j] - m|`，且 `arr[i] > arr[j]`

也就是说，差值的绝对值较大的就是我们要取得数，在排过序的基础上，要想差值绝对值最大，那这些一定出现在距离中位数较远的地方也就是数组的两端，我们在两端设两个指针，每次把遇到的最强的值存到一个新数组里，并缩小相应的区间即可

> 上面的操作只需要重复k次即可

## 复杂度

- 时间复杂度:$O(n \log n)$，主要是排序的消耗
- 空间复杂度:$O(1)$，不算返回值

## Code

```C++
class Solution {
public:
    vector<int> getStrongest(vector<int>& arr, int k) {
        int n=arr.size();
        sort(arr.begin(),arr.end());
        int middle=arr[(n-1)/2];
        int left=0;
        int right=n-1;
        vector<int> ans;
        while(k--){
            int l=abs(arr[left]-middle);
            int r=abs(arr[right]-middle);
            if(l<=r){
                ans.push_back(arr[right]);
                right--;
            }else{
                ans.push_back(arr[left]);
                left++;
            }
        }
        return ans;
    }
};
```
