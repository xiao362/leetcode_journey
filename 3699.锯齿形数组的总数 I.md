# 方法一：dp+前缀和

## 解题思路

要满足锯齿状数组，我们需要满足下面的条件：

- 每个元素的取值范围为 `[l, r]`。
- 任意 **两个** 相邻的元素都不相等。
- 任意 **三个** 连续的元素不能构成一个 **严格递增** 或 **严格递减** 的序列。

再试着模拟形成锯齿状数组的过程中我们可以发现，无论`n`多大，`l!=r`时我们都至少可以形成两个锯齿状的数组，并且这两个数组是"相反的",也就是一个最终是以递减结尾（开始）另一个则是以递增结尾（开始）

我们先对以递减结尾进行研究，假如第`n`个是以`k`（`k>l&&k<r`，等于的情况类似）结尾，那么`n-1`位置对应的数的可能就只有`l,l+1,l+1...k`，这几种情况的所有可能的和就是当第`n`个是以`k`结尾的所有可能，这时问题就变成了，以递增结尾当`n-1`位置为`l,l+1,l+2...k`时所有可能出现的情况，把这看成子问题，直到走到了起点，这样我们就求解出了一种情况

以递增结尾与这个过程在思路上基本相同，但过程恰好相反，把上面的每一步看为子问题就能用动态规划来解决

所以我们可以用两个数组，分别以递增或递减情况下不同位置对应的可能出现的总数，也就是状态方程：

- `f[i][j]`表示在第`i`个位置为`j`，并且是以递增结尾时可能出现的情况种数

- `f[i][j]`表示在第`i`个位置为`j`，并且是以递减结尾时可能出现的情况种数

因为我们研究时用到的因子是数与数之间的大小关系，而不是数的大小本身，所以我们可以把`l...r`用`0...r-l`来替代

> `j`从`0`到`r-l`,`i`表示目前锯齿数组长度（`i=1...n`），所有的`f0[1][j],f1[1][j]初始化为1`,此时没有两个元素就没有所谓的递增或递减，每种情况对应位的元素是确定的所以为`1`

而当以递增结尾时`f0[i][j]=f1[i-1][0]+f1[i-1][1]+...+f1[i-1][j]`（前缀）

以递减结尾时`f1[i][j]=f0[i-1][j+1]+f0[i-1][j+2]+...+f0[i-1][r]`(后缀)

通过这两个我们可以发现这里可以用前缀和（后缀和）来优化，避免这里再开一层数组来求和

这里我们注意到，`f0,f1`更新时用的都是上一层数组的量所以我们可以将二维数组优化为一维数组（去掉`i`），这一步能优化空间复杂度，但时间复杂度并没有改变

让后我们只需要在上面的过程结束后，对`f0,f1`的所有可能求和即是最终答案

> 注意：计算过程中，所有可能会溢出的计算都要取模运算

## 实现细节

1. 为形成递增或者递减，`i`从2开始
2. 先用上一层的`f0,f1`(没更新时)更新上一层对应的前缀和，才能利用更新后的前缀和更新本层的`f0,f1`
3. 更新`f1`递减结尾用`s0`的后缀和

## 复杂度

- 时间复杂度:$O(nk)$，`k=r-l`
- 空间复杂度:$O(k)$

## Code

```C++
class Solution {
public:
    int zigZagArrays(int n, int l, int r) {
        const int MOD=1e9+7;
        int k=r-l+1;
        vector<int> f0(k,1);//递增
        vector<int> f1(k,1);//递减
        vector<long long> s0(k+1,0);
        vector<long long> s1(k+1,0);
        for(int i=2;i<=n;++i){
            for(int j=0;j<k;++j){
                s0[j+1]=s0[j]+f0[j];
                s1[j+1]=s1[j]+f1[j];
            }
            for(int j=0;j<k;++j){
                f0[j]=s1[j]%MOD;
                f1[j]=(s0[k]-s0[j+1])%MOD;
            }
        }
        long long res0=reduce(f0.begin(),f0.end(),0LL)%MOD;
        long long res1=reduce(f1.begin(),f1.end(),0LL)%MOD;
        return (res0+res1)%MOD;
    }
};
```

## 优化：原地计算前后缀

我们发现计算前后缀和时只与上一层相反的（变化规律的前缀和（后缀和）有关，如`f0(本层)用上一层的f1`），我们可以通过pre来计算前缀和（后缀和），这样就进一步优化空间复杂度

> 注意：
>
> 1. 要用一个变量先存储起上一层的`f`，再更新本层`f`后再更新前缀和（后缀和）（用前面存储的量）
> 2. 交换`f1,f2`，这样就能保证f0(本层)用上一层的f1`

## 复杂度

- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int zigZagArrays(int n, int l, int r) {
        const int MOD = 1e9 + 7;
        int k = r - l + 1;
        vector<int> f0(k, 1); // 递增
        vector<int> f1(k, 1); // 递减
        for (int i = 2; i <= n; ++i) {
            long long pre = 0;
            for (int j = 0; j < k; ++j) {
                int v = f1[j];
                f1[j] = pre % MOD;
                pre += v;
            }
            long long suf = 0;
            for (int j = k - 1; j >= 0; --j) {
                int v = f0[j];
                f0[j] = suf % MOD;
                suf += v;
            }
            swap(f0, f1);
        }
        long long res0 = reduce(f0.begin(), f0.end(), 0LL) % MOD;
        long long res1 = reduce(f1.begin(), f1.end(), 0LL) % MOD;
        return (res0 + res1) % MOD;
    }
};
```

## 优化：对称性

根据对称性我们可以只求一种情况的所有可能*2即可

```C++
class Solution {
public:
    int zigZagArrays(int n, int l, int r) {
        const int MOD = 1e9 + 7;
        int k = r - l + 1;
        vector<int> f(k, 1); // 递增
        for (int i = 1; i < n; ++i) {
            if (i % 2) {
                long long pre = 0;
                for (int j = 0; j < k; ++j) {
                    int v = f[j];
                    f[j] = pre % MOD;
                    pre += v;
                }
            } else {
                long long suf = 0;
                for (int j = k - 1; j >= 0; --j) {
                    int v = f[j];
                    f[j] = suf % MOD;
                    suf += v;
                }
            }
        }
        long long res0 = reduce(f.begin(), f.end(), 0LL) * 2 % MOD;

        return res0;
    }
};
```

## 优化：对称性（代码量）

我们通过逆转`f`的方向只求前缀，不用区分递增或者递减

```C++
class Solution {
public:
    int zigZagArrays(int n, int l, int r) {
        const int MOD = 1e9 + 7;
        int k = r - l + 1;
        vector<int> f(k, 1); // 递增
        for (int i = 1; i < n; ++i) {
            long long pre = 0;
            for (int j = 0; j < k; ++j) {
                int v = f[j];
                f[j] = pre % MOD;
                pre += v;
            }
            reverse(f.begin(),f.end());
        }
        long long res0 = reduce(f.begin(), f.end(), 0LL) * 2 % MOD;

        return res0;
    }
};
```
