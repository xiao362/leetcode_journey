# 方法一：双指针

## 解题思路

因为这道题我们同时面对三个数，倘若我们用暴力做法，直接开三个指针，把每一种情况都比枚举出来时间复杂度来到$O(n^3)$，毫无疑问这太大了，观察数组是无序的，在没有规律可言的情况情况下（题目越短越是麻烦）我们除了枚举也不会有其他办法了，为避免这种情况我们的第一步肯定是排序

**为了避免同时操作和三个指针的复杂情况我们采取固定一个数**，而**同时操作另外两个数的办法**，在原数组排序后的基础上，用循环把每个数依次作为固定值，这时问题就转化为了：**两数之积最接近`固定值的平方`的问题**，这个问题我们可以**用双指针来解决**，即用数组两端的双指针分别作为我们要找的另外两个值，这时会有下面的情况：

如果二者积偏大，**这时我们就要让右指针左移，直到我们发现二者的和小于`固定值的平方`了，这时我们发现，right右边的值是肯定不会成为我们需要的值，因为他们加最小值都大于`固定值的平方`，然后接着如果我们想要等于`固定值的平方`，我们就要使二者的和变大，也就是左指针右移（右指针右边的已经淘汰），直到二者和再次大于`固定值的平方`，此时类似的left左边的也已经被淘汰，因为他们加上数组可能值的最大值后还是小于`固定值的平方`**，以此类推，我们**一定会遇到让二者积等于`固定值的平方`的情况（如果存在）**

但是我们需要对二者积相等时作坐标的变化情况单独处理，因为对于大小相同数我们也是要统计在内的（我们统计的是坐标对），这时我们就又要分两种情况：

1. `nums2[left]==nums2[right]`,此时区间内任意两对坐标都能符合要求（因为我们已经排序了），此时用等差数列的求和公式来统计个数

2. 二者不同是，我们要分别统计他们相同元素的个数，然后让二者相乘就是满足条件的坐标对的个数

   > 要注意：为了防止漏解，统计个数（用while）时，要让左右指针可以向等（例如，向统计左边的个数，left会走到第一个与原来不同的元素，如果这个位置恰好是right要统计的位置，他就走不到这里了）

然后通过不断改变固定值就可以所有情况下满足条件的答案了

## 实现细节

1. 因为是计算设计到乘积，所以要用long long，不然会溢出	
2. 为了避免写两份代码，可以把上面的功能写成函数

## 复杂度

- 时间复杂度:$O(n^2)$，严谨的来说是$O(n^2)+O(n \log n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int solve(vector<int>& nums1, vector<int>& nums2){
        int n=nums2.size();
        int ans=0;
        for(int i=0;i<nums1.size();++i){
            int left=0;
            int right=n-1;
            long long multiply=(long long)nums1[i]*nums1[i];
            while(left<right){
                if(multiply==(long long)nums2[left]*nums2[right]){
                    if(nums2[left]==nums2[right]){
                        int cnt=right-left+1;
                        ans+=cnt*(cnt-1)/2;
                        break;
                    }
                    int cnt_1=1;
                    int cnt_2=1;
                    left++;
                    right--;
                    while(nums2[left]==nums2[left-1] && left<=right){
                        cnt_1++;
                        left++;
                    }
                    while(nums2[right]==nums2[right+1] && left<=right){
                        cnt_2++;
                        right--;
                    }
                    ans+= (cnt_1 * cnt_2) ; 
                    
                }else if(multiply<(long long)nums2[left]*nums2[right]){
                    right--;
                }else{
                    left++;
                }
            }
        }
        return ans;
    }
        
    int numTriplets(vector<int>& nums1, vector<int>& nums2) {
        sort(nums1.begin(),nums1.end());
        sort(nums2.begin(),nums2.end());
        return solve(nums1,nums2)+solve(nums2,nums1);
    } 
};
```
