#方法一：中心拓展法

##解题思路和过程

依照题目我们找出一个字符串的最长回文子串，如果用暴力枚举的话，我们会发现我们需要开三层循环，时间复杂度也是达到了惊人的O（n^3),所以暴力方法暂不考虑
观察回文子串的结构如"aa","aba","abba","a"等我们可以发现回文子串是关于中点对称的，即只要满足中点对称，就是回文的字符串，这是我们只需要找到中点，然后判断两端的字母是否相等即可，关于中点分为以下两种情况：

1. 回文子串的长度的是**偶数，中点位于两个数中间**
2. 其**长度为奇数，中点是一个具体的数**，且无论这个数是多少（**故一个字母也是回文的**），只要其两端的字母是对称相等的，这个字符串就是回文的

所以，我们可以让中点在原字符串上跑(不要忽略两数中间地中点)，然后依次增加长度，判读增加在两端的两个字母是否相等即可，如果相等的话就，将长度len+=2即可

##复杂度

- 时间复杂度:O(n^2)
- 空间复杂度:O(1)

##Code

```
class Solution {
public:
    string longestPalindrome(string s) {
        int n=s.size();
        string ans;
        int maxval=0;
        for(int i=0;i<n;++i){
            int p=1;
            int j=1;
            while(i-j>=0&&i+j<n&&s[i-j]==s[i+j]){
                p+=2;
                ++j;
            }
            if(p>maxval){
                ans.assign(s,i-j+1,p);
                maxval=p;
            }
            p=1;
            j=1;
            if((i+1<n&&s[i]==s[i+1])){
                p++;
                while(i-j>=0&&i+j+1<n&&s[i-j]==s[i+1+j]){
                    p+=2;
                    ++j;
                }
            }
            if(p>maxval){
                ans.assign(s,i-j+1,p);
                maxval=p;
            }

        }
        return ans;
    }
};
```

#方法二：动态规划dp

> 时间上比中心拓展法满，且占的空间较大

##解题思路和过程

在回文子串的观察中我们可以发现：**一个回文子串去点首尾端点后依旧是一个回文子串**，**同种性质的不同切入点是问题多解的核心**
依照这个性质：**我们假设数组dp[i][j](bool)表示字符串从i到j是否是回文的**
由此我们便可以推出状态方程(当s[i]==s[j])时: ** dp[i][j]=dp[i+1][j-1]**;
当s[i]!=s[j]，自然**dp[i][j]=false**

> 注意：动态规划的核心是依照前面已经推出状态不断变化的，就像这道题，如果我们直接开两个循环控制i和j的话，我们的结果是不完全正确的：如dp[i][j]=dp[i+1][j-1];
> 正常的话我们是没i+1这个状态的（正序来推），即这个状态是不能用的
> 为了避免这种错误我们一般有两种想法可以考虑：(不一定都可用)

1. 改换思路，间接求解，让i+1这个状态能提前出来，比如引入长度len，只让一个index在数组上走，用长度len来间接求解另一个索引，随着一个len的一次遍历，都能填充一次dp
2. 倒序，将其中一个遍历的方向改变，进而让i+1，提前出现

**- 动态规划就好比递归，要有适当的终止条件**，对于该题他的终止条件即是i-j<=2时,即len<=2时，因为**当长度小于等于3时，子串只要满足两端相等即可证明其是回文的**，无法在小了
剩下就只有输出问题了，只需要设置maxval，当回文子串大于maxval时修改ans即可

> str.substr(0,1);的作用是去除一段字符串
> str.assign(s,i,len)的作用是取出一段字符串并覆盖在另一串字符串上面

##复杂度

- 时间复杂度:O(n^2)
- 空间复杂度:O(n^2)

##Code

```
class Solution {
public:
    string longestPalindrome(string s) {
        int n=s.size();
        if(n==0)return "";
        vector<vector<bool>> dp(n,vector<bool>(n,0));      
        string ans=s.substr(0,1);

        for(int i=0;i<n;++i){
            dp[i][i]=1;
        }
        int maxval=0;
    for(int len=1;len<n;++len){
            for(int i=0;i<n;++i){
                int r=i+len;
                if(r>=n) break;

                if(s[i]!=s[r]){
                    dp[i][r]=0;
                }else{
                    if(len<=2){
                        dp[i][r]=1;
                    }else{
                        dp[i][r]=dp[i+1][r-1];
                    }
                }

                if(dp[i][r]&&len+1>maxval){
                    ans.assign(s,i,len+1);
                    maxval=len+1;
                }                
            }
        }

        return ans;
    }
};
```

#方法三：manacher(马拉车)算法

##解题思路和过程

通过对回文子串观察，我们可以继续寻找一些回文子串的性质：如"cabdbac"，我们可以发现
- 在回文子串长度为奇数时，是**关于中点对称相等的**，也就是说，**在这个回文子串内，如果子串的左半部分中含有回文子串，那么对应的右半部分也应该有同样的回文子串**

为了更好地解释这条性质，我们**引入数组p[i]，表示在以i为中点的回文子串的半径(回文子串中点到端点的距离，不包括中点)**，那么上面对应的性质便可以转化为(同样以"cabdbac"为例)：

- 定义center为回文子串的中心，即d对应的坐标
- i为d右侧回文字串的中心(默认i在右边mirror在左边，更新i时要用到mirror)
- **mirror为i关于center的对称点(mirror=2*center-i)**
- right为回文子串的右边界坐标

> 则上面的性质即可转化为**p[i]==p[mirror]**;

因为以上讨论的都是mirror的回文半径对应的字母在回文子串内，而倘若超出回文子串的范围，mirror能保证而i却不能保证，因为我们才探测到right(右边界)，故p[i]的准确公式为**p[i]=min(p[mirror],right-i)**;

这下我们的p[i]就可以继承前面的已经计算过的回文半径了，但是以上的讨论都是基于回文子串的长度为奇数时展开的，倘若其为偶数的话，我们就又要分请情况讨论：故在这里引入一种方法，**将每个元素之间都差入一个'#',首尾用'^'，'$'，来站岗，防止越界，这样的话半径p[i]，就能更好地与长度兼容了，同时也解决了回文子串的个数为偶数的问题（因为，'#'本身的相同，并不影响回文的判断，且一个字母两旁对应的#，使得半径的值恰好等于回文子串长度）**

##复杂度

- 时间复杂度:O(n)
- 空间复杂度:O(n)

##Code

```
class Solution {
public:
    string longestPalindrome(string s) {
        string text="^#";
        for(char ch:s){
            text+=ch;
            text+='#';
        }
        text+='$';
		//原数据处理
        int center=0,right=0;
        int n=text.size();
        vector<int> p(n,0);
        int maxval=0;
        string ans="";
        for(int i=1;i<n-1;++i){
            int mirror=2*center-i;
            if(i<right){
                p[i]=min(right-i,p[mirror]);
            }
			//找对称点更新p[i]，继承

            while(text[i+p[i]+1]==text[i-p[i]-1]){
                p[i]++;
            }
			//将p[i]变成其能形成的最长子串长度，扩展
            if(i+p[i]>right){
                right=i+p[i];
                center=i;
            }
			//更新边界

            if(p[i]>maxval){
                ans.assign(s,(i-p[i])/2,p[i]);
                maxval=p[i];
            }
			//寻找最大值
        }
        return ans;
        
    }
};
```
