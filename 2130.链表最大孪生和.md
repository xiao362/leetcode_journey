# 方法一：暴力（数组存储）

## 解题思路

因为要想求链表的孪生和，我们需要兼顾链表的首尾值，但是我们得到的是的单向指针，所以我们可以开一个数组，用数组按顺序存储每个节点的值，这样就可以直接通过双指针或者单指针，来就直接求每一对孪生和并求出最大值了

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    int pairSum(ListNode* head) {
        int n=0;
        vector<int> list;
        while(head){
            list.push_back(head->val);
            head=head->next;
            n++;
        }
        int ans=INT_MIN;
        for(int i=0;i<n/2;++i){
            ans=max(ans,list[i]+list[n-i-1]);
        }
        return ans;
    }
};
```

# 方法二：递归

## 解题思路

我们利用**递归中归的性质也就是回退的性质**，先让**指针遍历（递）到链表的终点**，然后再**回退时，我们用另一个指针从头节点出发**，这样就**求得首尾值的和**了（孪生和）

## 实现细节

1. 递归时用right来控制递归到终点位置时，要以`right->next`不为空为（递）条件，不然尾指针就有偏差了
2. 这个方法适用于长度为偶数的情况，奇数要进行修改
3. **用lambda写递归函数时，要显示的定义返回值类型，不然会产生循环依赖（报错）**

> 原因如下：
>
> - 编译器读到第 3 行 `dfs(dfs, right->next)` 时，需要知道 `dfs` 的**返回类型**（即使没有用返回值，编译器也要检查整体语义）。
> - 但 `dfs` 的返回类型要到函数体结束、分析完所有 `return` 语句后才能确定。
> - 于是出现了**循环依赖**：要解析递归调用，需要知道返回类型；要知道返回类型，又必须解析完整个函数体。

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    int pairSum(ListNode* head) {
        int ans=0;
        ListNode* left=head;
        auto dfs=[&](auto& dfs,ListNode* right)->void{
            if(right->next){
                dfs(dfs,right->next);
            }
            ans=max(ans,left->val+right->val);
            left=left->next;
        };
        dfs(dfs,head);
        return ans;
    }
};
```

# 方法三：快慢指针+反转链表

## 解题思路

我们的目标是求首尾和，我们可以将原来链表的后半段反转，这样得到的链表就与前一半一一对应，可以直接相加了，但是再反转来链表时，我们需要先找到原来链表的中点

找链表的中点我们可以用快慢指针

## 实现细节

1. 快慢指针找中点的原理是始终让慢指针位于起点和快指针的中点，这个方法不论链表的长度是奇或偶
2. 得到反转指针后只需要与前半部分指针同步更新求和即可
3. **更新时要用反转后的指针为条件更新**，因为前半部分指针的末端还连着反转后的指针

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    ListNode* middleNode(ListNode* head){
        ListNode* slow=head;
        ListNode* fast=head;
        while(fast && fast->next){
            slow=slow->next;
            fast=fast->next->next;
        }
        return slow;
    }

    ListNode* reverseList(ListNode* head){
        ListNode* cur=head;
        ListNode* pre=nullptr;
        while(cur){
            ListNode* nxt=cur->next;
            cur->next=pre;
            pre=cur;
            cur=nxt;
        }
        return pre;
    }

    int pairSum(ListNode* head) {
        ListNode* mid=middleNode(head);
        ListNode* list=reverseList(mid);
        int ans=INT_MIN;
        while(list){
            ans=max(ans,head->val+list->val);
            head=head->next;
            list=list->next;
        }
        return ans;
    }
};
```
