# 方法一：bfs+记忆化

## 解题思路和过程

依据题意，我们需要从start出发，走遍所有可以走过的点，如果这些点中有值为0的点就可以返回true，否者就返回false，也就是在这个**连通块中找是否存在值为零的元素**，对于每一点我们都有两个方向可以走，**那么这种搜索问题我们常用的方法就有bfs，以及dfs**，因为呢这个数据不算小，所以**dfs递归深度可能会过**深，当然也可以手搓栈空间来来解决这个问题，但是我还是用bfs吧（要考虑的毕竟少点），**同时为了减少重复数据的反复访问（会爆时间），我们可以加一个记忆化数组vis，看是否访问过，最后就是要注意不要越界访问数组即可**

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(n)$

## Code

```C++
class Solution {
public:
    bool canReach(vector<int>& arr, int start) {
        if(arr[start]==0)return true;
        int n=arr.size();
        queue<int> q;
        vector<bool> vis(n,0);
        q.push(start);
        vis[start]=1;
        while(!q.empty()){
            int x=q.front();
            q.pop();
            if(arr[x]==0)return true;
            if(x+arr[x]<n&&x+arr[x]>=0&&!vis[x+arr[x]]){
                q.push(x+arr[x]);
                vis[x+arr[x]]=1;
            }
            if(x-arr[x]<n&&x-arr[x]>=0&&!vis[x-arr[x]]){
                q.push(x-arr[x]);
                vis[x-arr[x]]=1;
            }
        }
        return false;
    }
};
```
