# 方法一：双指针

## 解题思路

题目要求我们再长度为n的数组上反转k个数后，接下来k个数不反转，在接下来的k个数继续反转，就这样一直交替，且

- 如果剩余字符少于 `k` 个，则将剩余字符全部反转。
- 如果剩余字符小于 `2k` 但大于或等于 `k` 个，则反转前 `k` 个字符，其余字符保持原样。

我们可以用双指针，来控制要反转的边界，然后在其中内置一个循环把该区间反转，然后改变区间边界，为保证接下来k个数不反转我们直接跳过，即边界递增值为2*k

## 代码细节

1. 我们可以用while来控制边界的更新，以`l<n`为条件，没用右边界是为了在遇到接下来数组长度不满足k时也能交换
2. 在确定右边界时，为了保证不出界我们用`l+min(r,n-1)`来定义右边界
3. 在交换过程中，为保证不重复交换，要在走到区间中点时结束

## 复杂度

- 时间复杂度:$O(n)$
- 空间复杂度:$O(1)$

## Code

```C++
class Solution {
public:
    string reverseStr(string s, int k) {
        int n=s.size();
        int l=0;
        int r=k-1;
        while(l<n){
            for(int i=0;l+i<=(l+min(r,n-1))/2;++i){
                swap(s[l+i],s[min(r,n-1)-i]);
            }
            l+=2*k;
            r+=2*k;
        }
        return s;
    }
};
```

reverse写法

> 注意reverse是左开右闭即可

```C++
class Solution {
public:
    string reverseStr(string s, int k) {
        int n=s.size();
        for(int i=0;i<n;i+=2*k){
            reverse(s.begin()+i,s.begin()+min(i+k,n));
        }
        return s;
    }
};
```
